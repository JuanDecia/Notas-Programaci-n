Formulario y Validación Ejercicio

Realiza las siguientes tareas

  1. Crea un formulario con un campo de texto para "nombre".
  2. Crea una etiqueta "span" al lado del campo de texto con letra de color roja para los errores
  3. Crea una función que valide que:
    • El nombre debe ser obligatorio.
    • Debe contener más de dos caracteres.
    • No debe tener números.
  4. Enlaza la función definida con el campo de texto e indica un error en la sección del span cuando sea necesario. Esta función debe ser llamada cada vez que el usuario presione una tecla
