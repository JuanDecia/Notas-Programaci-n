Constructores

Introducción

Anteriormente hemos estado trabajando con objetos, hemos visto como es la construcción del dato de este tipo. Ahora, teniendo en cuenta que un objeto puede cambiar por un detalle lo que nos obligaría crear otro nuevo pero con prácticamente los mismos datos. Para evitar esta práctica y que el código sea tedioso existe un método llamado constructor el cual nos da la oportunidad de crear distintos objetos sin tener que reescribir todo el bloque de código.

Para utilizar estos constructores debemos utilizar la palabra reservada de JS new la cual nos guarda un espacio en memoria cada vez que realizamos la llamada de la misma para crear un nuevo objeto.

Para realizar una buena práctica de un método constructor de objetos es recomendable realizar una function y el nombre de la función debe comenzar con mayúsculas (parecido a como se crea una clase). Dentro de los parámetros del constructor, colocaremos los atributos que tendrá el objeto. Posteriormente, en el cuerpo de la función usaremos la palabra reservada this para indicar que estamos trabajando con este mismo objeto. Es muy importante esta práctica porque al crear un nuevo objeto con new al ejecutar la función, this va a estar apuntando al objeto que se está creando.

Luego de this debemos colocar un punto (.) seguido del nombre de la propiedad que va a ocupar el espacio del objeto y posteriormente le asignamos el valor del parámetro.

function Persona (nombre, apellido, email) { this.nombre = nombre;

La declaración anterior trabaja de la siguiente manera: this.nombre esta guardando el espacio para la propiedad nombre que va a ser creada y le asignamos el valor que se va a pasar por el parámetro cuando se cree el nuevo objeto.

Sintaxis

La sintaxis de la definición anterior quedaría de la siguiente manera:

function Persona(nombre, apellido, mail) { this.nombre = nombre; this.apellido = apellido; this.mail = mail; };

Lo que acabamos de definir es una función constructor de Persona.

Llamado constructor

Lo que nos resta por hacer es el llamado del constructor. Recordemos que debemos realizarlo con la palabra reservada new y al nombre del objeto debemos colocarlo con mayúscula. Para este ejemplo usaremos padre para el objeto creado por el constructor.

let padre = new Persona('Juan', 'Decia', 'juan@gmail.com');
