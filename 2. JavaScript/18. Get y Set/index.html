Getter & Setter

Introducción

A continuación veremos ejemplos acerca de Getter y Setter, pero antes de eso aclararemos un poco el significado para mejorar el entendimiento de estas funciones.

Si prestamos atencion Getter viene de la palabra en ingles "Get" la cual significa tomar en español. Así que, esta función lo que hace es tomar un valor.
En cambio, Setter que en inglés es Set y significa colocar, nos está refiriendo que esta función lo que hace es poner un valor.

Get

Getter es una función que nos ofrece JavaScript para mejorar la sintaxis de los methods en los objetos. Debemos tener en cuenta que tanto Getter debe estar dentro de un objeto Clase.

Sintaxis

Para utilizar este concepto debemos primero declarar "get"", reemplaza la palabra clave function por lo que seguido debemos colocar el nombre del método del objeto. Luego, colocaremos los paréntesis "()" y para finalizar las llaves que es donde va el cuerpo de la función.

get nombreCompleto() { return this.nombre + this.apellido; },

Gracias a get al llamar la función para mostrar el resultado, no debemos aclarar con los paréntesis que estamos llamando a una función. La sintaxis quedaría finalizada de la siguiente manera:

console.log(arregloPersonas.nombreCompleto) // Juan Decia

Setter

Esta propiedad puede modificar los valores de nuestro objeto. Para este ejemplo, agregaremos una propiedad llamada "lang" la cual vamos a modificar posteriormente con set.

La sintaxis es muy parecida a la de "get", aunque, debemos hacer unos pequeños cambios.

Para comenzar la declaración debemos colocar set, luego, el nombre correspondiente al método que va a cambiar los valores y dentro del parametro, debemos colocar el elemento que hemos tomado. Posteriormente debemos construir el cuerpo de la función, en este caso, seleccionaremos la propiedad "idioma" con this.idioma y le asignaremos el parámetro de la función. La construcción en código quedaría de la siguiente manera:

set lang(lang) { this.idioma = lang.toUpperCase(); },

Para finalizar este ejemplo, solo nos quedaría realizar la llamada del método y este paso también es similar al de get. En manera que al llamar el method, no debemos colocar los paréntesis, ya que, esto nos retornaría un error. Recordemos que este método modifica el valor de nuestro objeto. La sintaxis sería la siguiente:

objetoPersona.lang = 'en';

Ahora lo mistramos en consola:

console.log(objetoPersona.idioma);

Sintaxis

Dentro de las llaves de esta declaración colocaremos: this.color = color, porque, le estamos pasando a setter un color y ese color se va a pasar al constructor. Lo que nos cambiaría el color que teníamos por defecto anteriormente en el constructor.
