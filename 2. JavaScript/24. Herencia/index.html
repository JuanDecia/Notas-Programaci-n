Herencia

Las herencias nos son útiles cuando tenemos que hacer un nuevo objeto con similares atributos. A continuación, veremos como se construye la sintaxis de una herencia.

Sintaxis

Para crear una herencia de una clase objeto, primero debemos declarar que es una nueva clase con la palabra reservada que ya conocemos class, seguido del nombre que hereda esta nueva clase, para este ejemplo vamos a utilizar "Nautica" (recordemos que los nombres de las clases van en "mayúsculas").
A continuación, viene la palabra reservada de JS que declara la herencia y esta es: extends, seguido del nombre de la clase que estamos heredando todos sus atributos. Hasta ahí es la primer parte de la construcción de la nueva clase y en limpio quedaría de la siguiente manera:

class Nautica extends Automovil { }

Dentro de las llaves debemos crear los atributos propios de la nueva clase. Recordemos que debemos usar para esto el method constructor. Primero debemos colocar lo que se hereda puede ser: marca y modelo. Luego, debemos colocar los nuevos atributos de la nueva clase que estos pueden ser: materialCasco y tipoMotor Esta construcción queda de la siguiente manera:

constructor(marca, modelo, materialCasco, tipoMotor,) { }

Debemos saber que nos falta agregar la palabra clave super() que se coloca dentro del method constructor y no lleva llaves ({ }). Super solo contiene lo que vamos a heredar de la clase padre, esto debe estar colocado entre sus parámetros. En nuestro caso, solo heredamos de "Automovil" marca y modelo.

super(marca, modelo)

Ahora nos resta construir los atributos internos de la nueva clase heredada que con materialCasco y tipoMotor. Recordemos que para esto debemos usar la palabra clave this.

this.materialCasco = materialCasco
this.tipoMotor = tipoMotor

En el caso que requiera construir otros methods, esto sustituyen a los de la clase heredada, para este ejemplo, construiremos el mismo method de velocidad pero que la condición sea de 80Km/h.

Creando Objeto Heredado

Ahora ya tenemos creada y construida nuestra nueva clase, en este paso debemos instanciarla para crear el objeto que necesitamos para nuestra página.

Como hemos visto recientemente debemos crear una instancia de la clase para copiar los datos, esto se realiza creando una variable y le pasamos como valor la clase, pero, recordemos que debemos usar la palabra clave new antes de aplicar la clase:

const titanic = new Nautica('metal', 'carbon', 'titanic', 'titanicModel');

Como podemos observar dentro de los parámetro de "Nautica" se colocan los atributos especificos de "Titanic". Estos atributos son los elementos que nos dan el formato del objeto.

Heredando un método

Para heredar un método desde la clase padre, solo debemos realizar su llamado sin utilizar nada más en especial.
