Estructuras Condicionales

Introducción

Las estructuras condicionales son un tipo de estructura de control de flujo que se utilizan en programación para ejecutar un bloque de código si se cumple una condición específica. Básicamente, se trata de una forma de programar una serie de instrucciones para que se ejecuten sólo si se cumple cierta condición.

Estructura If

La estructura if en JavaScript es una sentencia condicional que permite ejecutar un bloque de código si se cumple una determinada condición. Su sintaxis es la siguiente:

if ( condicion ) {
// Código a ejecutar si la condición es verdadera.
}

En la línea del if, se evalúa la condición entre paréntesis. Si esta condición es verdadera, se ejecuta el bloque de código que se encuentra dentro de las llaves ('{ }') que le siguen.

let numero = 5;

if ( numero > 0 ) {
console.log ('El número es mayor que cero');
}

Aquí vemos que si se cumple la condición de que el número sea mayor a 0, se ejecutará el console.log.

Estructura If / else

La estructura if...else en JavaScript es similar a la estructura if, pero nos permite definir un bloque de código alternativo que se ejecutará en caso de que la condición evaluada en el if sea falsa. Su sintaxis es la siguiente:

if ( condicion ) {
// Código a ejecutar si la condición es verdadera.
} else {
// Código a ejecutar si la condición es falsa.
}

En la línea del if, se evalúa la condición entre paréntesis. Si esta condición es verdadera, se ejecuta el bloque de código que se encuentra dentro de las llaves ('{ }') que le siguen. Si la condición es falsa, se ejecuta el bloque de código que se encuentra dentro de las llaves ('{ }') que siguen a la palabra else.

let numero = -2;

if ( numero > 0 ) {
console.log ('El número es mayor que cero.');
} else {
console.log ('El número es menor o igual que cero.');
}

En este ejemplo vemos que si el número es mayor a 0 se ejecutará lo que está dentro del bloque if, mientras que en el caso contrario se ejecutará lo que está dentro del bloque del else. Al ser la variable 'numero = -2' se ejecutará lo del bloque else.

Estructura if/else if/else

La estructura if...else if...else en JavaScript nos permite evaluar múltiples condiciones y ejecutar distintos bloques de código en función de cada una de ellas. Su sintaxis es la siguiente:

if ( condicion ) {
// Código a ejecutar si la condición es verdadera.
} else if ( condicion2 ) {
// Código a ejecutar si la condición es falsa y condicion2 es verdadera.
} else if ( condicion3 ) {
// Código a ejecutar si la condición es falsa y condicion2 es falta y condicion3 es verdadera.
} else {
// Código a ejecutar si todas las condiciones son falsas.
}

En esta estructura, primero se evalúa la condición1. Si esta condición es verdadera, se ejecuta el bloque de código correspondiente y se omite el resto de las condiciones. Si la condición1 es falsa, se evalúa la condición2. Si esta condición es verdadera, se ejecuta el bloque de código correspondiente y se omite el resto de las condiciones. Si la condición2 es falsa, se evalúa la condición3, y así sucesivamente. Si ninguna de las condiciones es verdadera, se ejecuta el bloque de código que se encuentra dentro de las llaves {} que siguen a la palabra else.

Estructura Switch

La estructura switch en JavaScript es una alternativa a la estructura if-else que se utiliza cuando se necesitan múltiples condiciones para evaluar una variable o expresión. Esta estructura toma una expresión como entrada y la compara con diferentes casos. Si la expresión coincide con un caso específico, se ejecuta el código asociado a ese caso.

Sintaxis Switch

switch ( expresion ) {
case valor1:
// Código a ejecutar si la expresión es igual a 'valor1'.
break;
case valor2:
// Código a ejecutar si la expresión es igual a 'valor2'.
break;
default:
// Código a ejecutar si la expresión no coincide con ninguno de los casos anteriores.
break;
}

La expresión que se evalúa puede ser cualquier tipo de dato, incluyendo números, cadenas, booleanos, entre otros. Cada caso dentro de la estructura switch debe tener un valor específico que se comparará con la expresión.


Es importante destacar que la instrucción break se utiliza para salir del switch una vez que se ha encontrado una coincidencia. Si no se utiliza la instrucción break, el switch seguirá ejecutando el código de todos los casos siguientes, incluso si no se cumplen las condiciones. La cláusula default es opcional y se ejecuta si la expresión no coincide con ninguno de los casos anteriores.

Operador Ternario

El operador ternario en JavaScript es una forma concisa de escribir una estructura condicional if-else en una sola línea de código. Este operador se utiliza comúnmente para asignar un valor a una variable en función de una condición. La sintaxis básica del operador ternario en JavaScript es la siguiente:

condicion ? valorSiTrue : valorSiFalse;

Donde "condicion" es una expresión que se evalúa como verdadera o falsa, "valorSiTrue" es el valor que se asignará si la condición es verdadera y "valorSiFalse" es el valor que se asignará si la condición es falsa.

Por ejemplo, podemos usar el operador ternario para asignar un valor a una variable "esMayor" que indica si un número "edad" es mayor o menor que 18 años:

let edad = 20;

let esMayor = edad >= 18 ? 'Mayor de edad' : 'Menor de edad';

También es posible anidar operadores ternarios para evaluar múltiples condiciones:

let esMayor = edad >= 18
? 'Mayor de edad'
edad >= 13
? 'Adolescente'
: 'Niño';

En este ejemplo, se evalúa primero si la "edad" es mayor o igual a 18 años. Si es verdadero, se asigna el valor "Mayor de edad". Si es falso, se evalúa si "edad" es mayor o igual a 13 años. Si es verdadero, se asigna el valor "Adolescente". De lo contrario, se asigna el valor "Niño".
