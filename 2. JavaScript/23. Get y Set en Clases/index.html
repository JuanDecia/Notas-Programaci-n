Get y Set en Clases

Introducción

Como anteriormente habíamos trabajado en objetos, también, habíamos visto los conceptos de get y set. Ambos son métodos para simplificar el llamado de las propiedades de los objetos. En esta ocasión veremos como trabajar este concepto en las clases, la sintaxis que comparte es similar y su funcionamiento también. Lo importante de este punto es destacar que es una buena práctica realizar este paso en las plantillas para facilitar luego los llamados de los valores cuando instanciemos en un objeto.

Para los ejemplos seguiremos trabajando con el ejemplo de Persona y sus instancias.

Sintaxis

Repasemos la sintaxis y el funcionamiento de ambos métodos. Anteriormente colocabamos el nombre de la propiedad, según correspondía. Ahora, para mejorar la práctica le agregaremos un guión bajo "_" por delante del nombre. La construcción de la clase quedaría de la siguiente manera:

Get

class Persona { constructor (nombre, apellido, edad, mail) { this._nombre = nombre; this._apellido = apellido; this._edad = edad; this._email = email; }

Esto es debido a que al usar los métodos get y set vamos a necesitar utilizar los nombres sín los guiones. A continuación, nos corresponde crear el método get:

get nombre() { return this._nombre; }

Set

Recordemos que este método es para poder cambiar el valor de un objeto y su sintaxis es muy similar a la de get, a diferencia que a set debemos pasarle la propiedad: como parámetro:

set nombre(nombre) { this._nombre = nombre; }
