Prompt

Introducción

La función prompt dentro de JavaScript nos permite almacenar un dato que ingrese un usuario dentro de una variable.

Al usuario le aparecerá una ventana de alerta en el navegador donde se le solicita que ingrese datos y esos datos estarán dentro de la variable para luego ser utilizados en el código de JavaScript

Dentro de los parámetros de prompt podemos dejar un string que será el mensaje que verá el usuario cuando le aparezca el cartel en el navegador, esto es útil para darle una pista al usuario sobre que dato debería ingresar.

Sintaxis

let variable = prompt("Texto que le aparecerá al usuario");

Lo que ingrese el usuario se guardará en variable. Tengamos en cuenta que el usuario puede colocar en esta ocasión, cualquier tipo de dato. Lo adecuado sería, que nosotros construyamos una función en la que el usuario solo pueda ingresar un solo tipo de dato. En el caso contrario, que este proceso se repita hasta que el usuario ingrese el dato correcto.
