Sentencia if y else

Introducción

Las expresiones de if y else son expresiones booleanas, es decir, puede retornar el valor de verdadero o falso. Esto lo podemos graficar como un camino para if y otro camino para else, si la condición es verdadera para if se tomará el camino del mismo ejecutando el fragmento construído en el ámbito de if. En el caso contrario, se tomará el camino de else y se ejecutará el fragmento construído ahí mismo.

Else if

En ocasiones, vamos a necesitar más de una condición para efectuar la sentencia. Es decir, no nos alcanzará con declarar solo un if y un else, en este caso tendremos que usar la sentencia else if (condición) {resultado si es posivito}. Esta sentencia nos otorga una ruta adicional a la primera condición efectuada, en el caso de que la primera sea false, llegará la oportunidad de comprobar la sentencia de si else if es true y así sucesivamente si llegaramos a tener más de una sentencia de else if. A continuación, veremos un ejemplo de su sintaxis:

if ((mes == 11 && day >= 21) || mes == 0 || mes == 1 || (mes == 2 && day < 21)) {
estacion = 'Verano';
}
else if ((mes == 2 && day >= 21) || mes == 3 || mes == 4 || (mes == 5 && day < 21)) {
estacion = 'Otoño';
}
else if ((mes == 5 && day >= 21) || mes == 6 || mes == 7 || (mes == 8 && day < 21)) {
estacion = 'Invierno';
}
else if ((mes == 8 && day >= 21) || mes == 9 || mes == 10 || (mes == 11 & day < 21)){
estacion = 'Primavera';
}

Como podemos observar en el ejemplo, tenemos una sentencia para todas las condiciones que pueden efectuarse si desde la primera no es correcta. En resumen, lo que trata de buscar este fragmento de código es que según la fecha dada nos retorne la estación del año en la que nos encontramos.
