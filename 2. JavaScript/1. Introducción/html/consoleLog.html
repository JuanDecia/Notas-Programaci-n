Introducción

A continuación, se va a observar una función llamada “sumaValor ()”. Esta función va a realizar una suma y el resultado lo podremos ver por la consola. Para ver los resultados debemos apretar el F12 y dirigirnos a la sección de console. Nos debería aparecer como resultado en la consola: "El valor de la suma es 4".

Código de la función

const sumar = () => {
let a = 1;
let suma = a + 3;
console.log(`El valor de la suma es: ${suma}.`);
}
sumar();
