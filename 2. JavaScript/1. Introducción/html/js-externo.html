Introducción

En esta página realizaremos el mismo ejercicio que en la página JS interno, pero a diferencia en esta ocasión, enlazaremos el archivo externo main.js a esta página llamada 'js-externo.html. El archivo main.js es el que tendrá el código que hará la dinámica, es decir, no va a estar dentro de las etiquetas <script>.

Enlazar Js

A continuación para esta práctica, dejaremos un ejemplo de como debemos enlazar el archivo main.js a esta página.

<script src='../scripts/main.js'></script>

Como se puede observar en el ejemplo anterior, hemos enlazado la ruta relativa del archivo a nuestra página. Ahora, debemos crear el código que utilizaremos para el funcionamiento de la misma. Para agilizar este concepto, utilizaremos el mismo ejemplo que en la página JS interno.
