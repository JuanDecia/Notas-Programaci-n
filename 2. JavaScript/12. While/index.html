Introducción

Para ejecutar ciertas partes de nuestro programa varias veces necesitamos de los bucles, una de las cláusulas para hacer esto en JS es el “while”. Su funcionamiento se basa en realizar un conjunto de instrucciones mientras una condición se cumpla:

Sintaxis

En el ejemplo vemos que primero definimos la variable “n” con un valor inicial de “0”. En el ciclo se va a leer “mientras “n” sea menor que 10” imprimimos el valor de “n”. Si ejecutamos en este momento como la variable no cambia “n” se queda en “1”, es decir, se trata de un bucle infinito.

Para que salgamos del bucle debemos incrementar “n”, por ejemplo, usando el operador de adición “++” para que se incremente la variable “n” en “1”, cada vez que pasemos por allí. De esta manera el resultado incluye una secuencia de números del 0 al 9. Se acaba en 9 porque cuando el contador llega a 10, no se cumple la condición y ocasionando que salgamos del bucle. Si deseamos que aparezca el “10” debemos poner el operador condicional “<=”.

La declaración del operador “++” es como poner: n = n + 1;

En este caso cada iteración del bucle, incrementa “n” y añade ese resultado a “x”. Por lo tanto, tendrían estos valores:

  • Después del primer pase: “n = 1” y “x = 1”.
  • Después del segundo pase: “n = 2” y “x = 3”.
  • Después del tercer pase: “n = 3” y “x = 6”.

Palabra clave: Break

Cuando estamos en un bucle while y deseamos que termine en una condición dada, debemos usar la instrucción “break”. Esto sucede porque break nos retorna un resultado false que nos termina el ciclo.
