Peticiones con Axios

Primero debemos entrar a la página de axios para poder importarlo . Luego, si estamos trabajando con el HTML básico debemos importar el “jsDeliver CDN” a nuestro HTML. Un aspecto muy importante a destacar es que primero debemos importar la etiqueta de axios y luego debemos colocar la etiqueta del archivo externo de JS.

Para utilizar axios debemos realizar una estructura muy parecia a cuando utilizamos fetch, a diferencia de esta vez, debemos cambiar un poco la sintaxis. Para comenzar debemos usar async - await y try - catch.

const getUsers = async () => { try { const peticion = await axios('https://jsonplaceholder.typicode.com/users'); const users = await peticion.data; }

Con esa instrucción estamos importando y utilizando axios. Luego, debemos crear el código necesario para mostrar los resultados que necesitemos, en nuestro caso realizamos un bucle for para mostrar los atributos de cada uno de ellos.

Notemos que lo único que cambia es que debemos utilizar axios en vez de "fetch" y luego, utilizamos data en vez de "json()".
