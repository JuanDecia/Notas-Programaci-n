Ejemplos para los operadores aritméticos

Volver a Inicio

Introducción

A continuación, se realizaran los ejemplos de cada uno de los operadores aritméticos en JS para poder aplicar y visualizar los resultados.

Operador Suma (+)

Se utiliza para sumar valores:

const edadJuan = 28;

console.log(2 + 2); // Output: 4
console.log(edadJuan + 10); // Output: 38

Operador Resta (-)

El operador de resta es útil para restar valores:

const edadJuan = 28;

console.log(2 - 2); // Output: 0
console.log(edadJuan - 10); // Output: 18

Operador Multiplicación (*)

Este operador es necesario cuando necesitamos multiplicar números:

const edadJuan = 28;

console.log(2 * 2); // Output: 4
console.log(edadJuan * 10); // Output: 280

Operador División (/)

El operador de división ('/') se utiliza para dividir números.

const edadJuan = 28;

console.log(2 / 2); // Output: 1
console.log(edadJuan / 10); // Output: 2.8

Operador Módulo

El operador 'modulo' ('%') se utiliza para obtener el resto de una división.

const edadJuan = 28;

console.log(2 % 2); // Output: 0
console.log(edadJuan % 10); // Output: 8

Operador de Incremento

El operador de incremeto se utiliza para aumentar el valor de la variable en '1'.

const edadJuan = 28;
edadJuan++;

console.log(edadJuan); // Output: 29

Es importante destacar que el operador '++' reemplaza la siguiente declaración: edadJuan = edadJuan + 1;.

Operador de Decremento

El operador de decremento ('--') se utiliza para disminuir el valor de la variable en '1'.

const edadJuan = 28;
edadJuan--;

console.log(edadJuan); // Output: 27

Es importante destacar que el operador '--' reemplaza la siguiente declaración: edadJuan = edadJuan - 1;.
