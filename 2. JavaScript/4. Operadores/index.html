Operadores

Introducción

En programación debemos conocer los distintos tipos de operadores que pueden utilizarse, ya que, a la hora de la construcción de nuestro código son sumamente importantes y debemos saber con exactitud la definición y la función de cada uno de ellos.

Operadores de Asignación

Los operadores de asignación asignan valores a las variables de JavaScript.

Operador Significado Ejemplo (Sintaxis) Es igual a
= Asignación x = 1 x tiene el valor de "1"
+= Asignación con suma: Se utiliza para sumar un valor a una variable y asignar el resultado a la misma. x += 1 x tiene el valor de "1" + 1 (x = x + 1)
-= Asignación con Resta: Se utiliza para restar el valor a una variable y asignar el resultado a la misma. x -= 1 x tiene el valor de "1" - 1 (x = x - 1)
*= Asignación con multiplicación: Se utiliza para multiplicar un valor con una variable y asignarle el resultado a la misma. x *= 1 x tiene el valor de "1" * 1 (x = x * 1)
/= Asignación con División: Se utiliza para dividir una variable por un valor y asignar el resultado a la misma variable. x /= 1 x tiene el valor de "1" / 1 (x = x / 1)
%= Asignación con Módulo: Se utiliza para obtener el módulo de la división entre una variable y un valor y asignar el resultado a la misma. x %= 1 x tiene el valor de "1" % 1 (x = x % 1)

Operadores Lógicos

Hay tres operadores lógicos en JavaScript: || (Or), && (And), ?? (Fusión de nulos). En las notas de esta sección veremos solo los primeros dos. Aunque sean llamados lógicos, pueden ser aplicados a valores de cualquier tipo, no solo booleanos. El resultado también puede ser de cualquier tipo.

Operador Significado Ejemplo
&& y (and) 1 == 1 && 2 < 1 (es falso)
|| o (or) 1 == 2 || 15 > 2 (es verdadero)

&& (And)

Se utiliza para comprobar si dos condiciones son verdaderas. El resultado de la operación AND es verdadero sólo si ambas condiciones son verdaderas.


Todas las funciones que están unidas por && se tienen que cumplir. El intérprete de JS verifica si la primera condición se cumple y luego chequea la segunda pudiendo tener nosotros más de un “&&”. Si en este proceso se consigue alguna que no se cumpla, la condición completa es falsa.

let edadLucas = 15;
let edadAbril = 11;

if ( edadLucas && edadAbril < 18 ) {
console.log('Los chicos son menores de edad');
} else {
console.log('Uno o ninguno de los chicos es mayor de edad.');
}

En el ejemplo anterior tenemos una condición '&&' que nos retorna true, porque, lucas y abril tienen un valor menor a 18 (15 y 11 respectivamente).

|| (Or)

Se utiliza para comprobar si al menos una de las dos condiciones es verdadera. El resultado de la operación OR es verdadero si alguna de las dos condiciones es verdadera.

let edadLucas = 15;
let edadAbril = 11;
let edadJuan = 28;

if ( edadLucas || edadAbril || edadJuan > 18 ) {
console.log('Una de las personas es mayor de edad.');
} else {
console.log('Ninguno es mayor de edad.');
}

! (not)

Se utiliza para invertir el valor de una expresión booleana. El resultado de la operación NOT es verdadero si la expresión booleana es falsa, y falso si la expresión booleana es verdadera.

let edadJuan = 28;

console.log( ! ( edadJuan > 0 ) ); // edadJuan es mayor a 0, el resultado es false
console.log( ! ( edadJuan < 0 ) ); // edadJuan es mayor a 0, el resultado es true
