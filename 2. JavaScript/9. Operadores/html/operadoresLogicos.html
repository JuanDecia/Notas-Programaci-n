Ejemplo and (&&)

Podemos hacer condiciones anidadas con los operadores “and” (&&). Todas las funciones que están unidas por && se tienen que cumplir. El intérprete de JS verifica si la primera condición se cumple y luego chequea la segunda pudiendo tener nosotros más de un “&&”. Si en este proceso se consigue alguna que no se cumpla ya la condición completa es falsa.

Generalmente usamos estos operadores en bucles o sentencias de if - else. Para este ejemplo, lo aplicaremos dentro de una sentecia if. Para el ejemplo, tenndremos una variable con nombre "lucas" y un valor de tipo int asignado de 15. También, tenemos una variable llamada "abril" con otro valor de tipo int de 11.

Sintaxis

if (lucas && abril < 18) {
console.log(`Los chicos son menores de edad.`);
contResultado[0].innerHTML += "Los chicos son menores de edad.";
} else {
console.log(`Los chicos son mayores de edad.`);
contResultado[0].innerHTML += "Uno o ambos de los chicos no es menor de edad.";
};

En este caso tenemos una condición and que nos retorna true, porque, lucas y abril tienen un valor menor a 18 (15 y 11 respectivamente). Por lo que la consola nos retornó:

Recordemos que para que esta sentencia pueda ser verdadera, ambas condiciones nos deben retornar en true.

Ejemplo or (||)

Otra forma de comparar es utilizando or “||”. Para que se cumpla la condición o al menos una de ellas debe ser verdadera.

Sintaxis

if (lucas || abril || juan > 18) {
console.log("Pueden ingresar, uno de ellos es mayor.");
} else {
console.log("No pueden ingresar, son todos menores.");
}

Como juan tiene un valor de 28, fue suficiente para que la condición nos retorne true y entre en la primer sentencia.
