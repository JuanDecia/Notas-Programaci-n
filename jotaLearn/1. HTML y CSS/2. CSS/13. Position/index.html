Propiedad Position

Introducción

La propiedad position nos permite manipular una posición fija en relación a su elemento contenedor en el DOM. La propiedad position puede recibir cinco valores, es decir, tenemos cinco maneras diferentes de posicionar los elementos según nuestro sitio web lo requiera.

Sintaxis

.position-element {
position: static;
position: relative;
position: fixed;
position: absolute;
position: sticky;
}

Z-index

Esta propiedad de CSS es útil cuando necesitamos superponer un contenedor sobre otro, haciendo que el contenedor con z-index más elevado siempre esté visible por encima de todos los demás elementos. Esto es posible ya que podemos manejar los ejes que posicionan los elementos. Por ejemplo, el eje horizontal es el eje x-axis; el eje vertical corresponde a y-axis y finalizando tenemos el eje z-index.


Entonces al utilizar la propiedad de z-index estamos utilizando la posición en relación de 3d del elemento. Esta propiedad recibe un valor numérico y mientras más grande sea este valor, estará el elemento más superpuesto a los demás. Es decir, el valor de z-index más alto es el que se va a mostrar por encima de los demás. El valor por defecto de esta propiedad es de 0 o auto el siguiente valor sería 1. Esto no significa que podemos también colocar valores como -1. Estos elementos estarían por detrás de los elementos posicionados con el valor de z-index por defecto, es decir, estarán ocultos.

.position-element {
z-index: 1;
}
