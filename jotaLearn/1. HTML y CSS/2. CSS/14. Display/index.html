Display

Introducción

Comenzaremos a ver la propiedad display para los elementos de CSS, esta propiedad es una de las más importantes a la hora de maquetar una página web, ya que, sus opciones establecen el flujo de cómo se posicionarán los contenidos dentro o fuera del elemento. La propiedad de CSS display, especifica si un elemento es tratado como “block” o “inline”, es decir, si el flujo es en bloque o en línea. Formalmente la propiedad display establece tipos de visualización “interna” y “externa” de un elemento.

Display Outside

Display-outside, se encarga de establecer el flujo de los elementos hijos, es decir, de que manera son ubicados al generarse. Como anteriormente habíamos nombrado estos pueden estar en bloque o en línea (ver conceptos de '12.layout css').

Valores y sintaxis

.display-outside {
display: block;
display: inline;
display: run-in;
}

El valor por defecto que reciben todos los elementos al generarse es 'block'.

Valores de Outside

Display: block

Block es el valor por defecto del navegador a declarar un elemento. Este valor genera un cuadro de elemento de bloque, es decir, el elemento hace que se ocupe todo el espacio disponible hasta el final donde sea posible, aunque sus contenidos no ocupen todo el espacio.

Soy un div con display block, ocupo todo el espacio disponible de mi elemento. Mi elemento padre, también tiene su display en block.

Display: inline

Este valor modifica el elemento a uno o más contenedores del elemento en la línea del ancho del viewport o del contenedor padre. Es decir, solo ocupa el espacio necesario para mostrar los contenidos.

Display Inside

Los elementos que tienen establecidos algún valor que su comportamiento refiere a display-inside son para establecer el diseño de los elementos hijos que se generen en ese elemento padre.

Valores y sintaxis

.display-inside {
display: flex;
display: grid;
display: table;
display: flow;
}

Los valores de inside como las propiedades de flex box la veremos más adelante.
