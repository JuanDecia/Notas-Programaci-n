Propiedad Border

Introducción

Esta propiedad establece el estilo de línea para los cuatro lados del borde de un elemento. Debemos saber que cuando estamos haciendo referencia a la propiedad border estamos hablando de una “propiedad shorthand”. Esto se debe a que border puede aplicar muchos estilos en una sola declaración. Pero, en este caso puntual vamos a hacer referencia a “border-style” que es enmarcado que nosotros les damos a los contenedores para visualizarlos.


Esta propiedad puede recibir hasta 4 valores comenzando siempre desde la parte top siguiendo el sentido del reloj hacia la derecha, es decir: right, bottom y por último left. Esto debemos saberlo para manipular los bordes cuando no es necesario que todos tengan el mismo valor.

Sintaxis

En el siguiente ejemplo visualizamos la sintaxis de border, recordemos que border es una shorthand en la misma podemos darle diferentes tipos de argumentos como “tamaño” “tipo” y “color”, puede recibir una de estas o como máximo las 3.


Primero hacemos el llamado del uso de la propiedad “border”. Luego es recomendable determinar el “tamaño” del borde en pixeles (px) para visualizar el contenedor (el valor por defecto de este estilo es “none”) recordemos que si no lo hacemos no podemos visualizar el contorno del contenedor.


Seguido, debemos especificar el tipo de “border” que vamos a utilizar para visualizar el contorno, recordemos que para este estilo tenemos diferentes valores. Usualmente se suele trabajar para visualizar el contorno es “solid”.


El último valor que podemos utilizar en esta propiedad es “color”, que al aplicar este estilo es el color que visualizamos el contorno. Su valor por defecto es “black”, pero, si estamos trabajando con varios contornos al mismo tiempo es recomendable usar otros colores para manipular mejor el moldeado de las cajas.

Sintaxis Ejemplo

.border-ejemplo {
border: 1px solid black;
}

Es lo mismo que declarar:

.border-ejemplo {
border: 1px;
border-style: solid;
border-color: black;
}

Estilos de Bordes

Anteriormente nombramos que los bordes podían tener diferentes estilos y estos podían ser declarados y manipulados con la propiedad de "style". A continuación, vamos a visualizar algunos de los estilos que podemos usar con esta propiedad:

Border: None

Es el valor que viene por defecto en el navegador en el cual no podemos visualizar los bordes de los contenedores.

Border: Dotted

Este valor aplica al borde una línea punteada.

.dotted-ejemplo {
border: 1px dotted blue;
}

Border: Inset

Este valor crea un sombreado en la parte superior e izquierda del contenedor ancho.

.inset-ejemplo {
border: 2px inset blue;
}

Border: Outset

Crea un sombreado ancho en la parte derecha e inferior del contenedor.

.outset-ejemplo {
border: 2px outset blue;
}

Border: Dashed

Este valor crea un borde entre líneas.

.dashed-ejemplo {
border: 2px dashed blue;
}

Border: Double

Este valor crea una doble línea sólida en el contorno del contenedor. No lleva valor en pixeles.

.double-ejemplo {
border: double blue;
}

Border: Solid

Este valor crea una línea sólida en el contorno del contenedor.

.solid-ejemplo {
border: 1px solid blue;
}

Border: Grove

Este valor crea un efecto como “inset” pero en todo el contorno del contenedor.

.groove-ejemplo {
border: 3px groove blue;
}

Border: Ridge

Crea un contorno sombreado en todo el contenedor.

.ridge-ejemplo {
border: 3px ridge blue;
}

A tener en cuenta a la hora de utilizar bordes

  • Cuando se especifica un valor, se aplica el mismo estilo a los cuatro lados.
  • Cuando se especifican dos valores, el primer estilo se aplica a la parte superior e inferior, el segundo a la izquierda y a la derecha.
  • Cuando se especifican tres valores, el primer estilo se aplica a la parte superior, el segundo a la izquierda y derecha y el tercero a la parte inferior.
  • Cuando se especifican cuatro valores, los estilos se aplican a la parte superior, derecha, inferior e izquierda en ese orden.

Border Radius

Vamos a comenzar con el efecto de esquinas redondas para las cajas, este efecto se logra con la propiedad border-radius.

Sintaxis

.border-ejemplo {
border-radius: 5px;
-moz-border-radius: 5px;
-webkit-border-radius: 5px;
}

Como esta propiedad no es accesible a los diferentes navegadores, debemos agregar manualmente a nuestra hoja CSS las propiedades para los diferentes navegadores ‘-moz-’ y ‘-webkit-’.


La propiedad border-radius puede especificarse en 4 formas:

Un valor

Todas las esquinas tienen el mismo radio.

.un-valor {
border-radius: 5px;
}

Dos valores

El primero se utiliza para el radio de las esquinas superior izquierda e inferior derecha. El segundo valor se aplica al radio de las esquinas.

.dos-valores {
border-radius: 15px 5px;
}

Tres valores

El primero se aplica al radio de la esquina superior izquierda, el segundo valor a la esquina superior derecha y la esquina inferior izquierda, el tercero se aplica al radio inferior de la esquina derecha.

.tres-valores {
border-radius: 15px 5px 7px;
}

Cuatro valores

El primer valor se aplica al radio de la esquina superior izquierda, el segundo al borde superior derecho, el tercero al borde inferior derecho y, el cuarto al borde inferior izquierdo.

.cuatro-valores {
border-radius: 15px 5px 7px 10px;
}
