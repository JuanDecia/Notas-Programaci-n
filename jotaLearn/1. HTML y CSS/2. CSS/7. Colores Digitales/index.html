Colores Digitales

Introducción

El color digital son números en computadores ordenados de tal forma que al ser representados en dispositivos como monitores (RGB) o dispositivos de salida como impresoras (CMYK) y son percibidos con un color. CSS nos ofrece diferentes propiedades para agregar colores a los elementos que existen en el DOM de nuestro documento. En CSS, hay varias formas de especificar un color, entre ellas son:

  • Nombre de colores: CSS tiene una lista de nombres de colores predefinidos que se pueden usar para especificar un color. Por ejemplo: "red", "blue", "green", etc.
  • Hexadecimal: Es un sistema de codificación de colores que usa una combinación de números y letras para representar un color. Por ejemplo: "#FF0000" representa el rojo.
  • RGB: Este sistema especifica un color en términos de valores de intensidad de los componentes rojo, verde y azul. Por ejemplo: "rgb(255, 0, 0)" representa el rojo.
  • RGBA: Es similar a RGB, pero también incluye un valor de opacidad, que controla la transparencia del color. Por ejemplo: "rgba(255, 0, 0, 0.5)" representa un rojo semitransparente.
  • HSL: Este sistema especifica un color en términos de tono, saturación y luminosidad. Por ejemplo: "hsl(0, 100%, 50%)" representa el rojo.
  • HSLA: Es similar a HSL, pero también incluye un valor de opacidad.

RGB

Cuando aplicamos un color en RGB estamos aplicando una combinación de colores que estos colores son (en inglés) red, green, blue. La medición de estos colores es numérica y dependiendo el valor numérico que le demos a estos valores es la graduación que aplica a la combinación.


Para aplicar esta propiedad debemos colocar la palabra clave 'rgb()' y dentro los paréntesis colocaremos los valores numéricos separados por coma (","). Debemos tener en cuenta que los colores respetan el orden del nombre de la propiedad (red, green, blue).


En el caso que tengamos el valor (0, 0, 0) estamos diciendo que no hay combinaciones de colores y por ende nos retornaría el color “negro”, es decir, estamos aplicando una ausencia de color. Debemos saber que estos colores van de 0 hasta el 255 siendo el 255 la tonalidad más fuerte, es decir, que si colocamos este valor a uno de RGB y los demás los dejamos en 0 nos retornaría el color natural que lo representa. Ejemplo: 0, 0, 255 (retorna blue).

.caja {
background-color: rgb(0, 0, 255);
}

RGBA

Esta propiedad de color que puede ser implementada en la hoja de estilos CSS permite añadirle transparencia (representa la letra a “Alpha”) al texto en el último valor agregado con “.”. Por ejemplo: rgba(68, 241, 15, 0.5) el último valor puede ser de 0 a 1. Teniendo en cuenta que 0 es totalmente transparente (el texto no se va a ver) y 1 no hay transparencia (se ve igual). Los valores que deberíamos manejar son decimales como por ejemplo 0.5 que es una trasparencia media.

.caja {
background-color: rgba(70, 70, 194, 0.404);
}

Color para Fuentes

Propiedad: Color

Es la propiedad que se utiliza para cambiar el color de la fuente de un elemento. Podemos usar todos los formatos de colores que vimos anteriormente. Para ejemplificar esta teoría, crearemos un contenedor con una estructura cuadrada y le aplicaremos un color al texto que se encuentra dentro.

.parrafo-ejemplo-color {
color: #4c26f3;
}

Soy un párrafo de ejemplo, por defecto soy de color negro. En este ejemplo me cambiaron de color gracias a la propiedad 'color' de CSS.

Importante propiedad Color

Es común, salvo en casos muy puntuales que al poner la propiedad color en un elemento que contiene otros, los elementos internos tomen el color de letra que se le colocó al elemento padre, es decir, los elementos hijos van a heredar la propiedad color del elemento que los contiene. Es decir, le hemos aplicado directamente al contenido del elemento 'p' la propiedad de 'color', hubiera sido el mismo resultado si le colocamos la propiedad al contenedor padre 'div' con clase 'ejemplo-fuente'.

.contenedor-color {
color: #4c26f3;
}

Colores para elementos del DOM

Propiedad Background

Es la propiedad que se utiliza para definir el fondo de un elemento. Con ella, podemos establecer el color de fondo, agregar una imagen de fondo, y controlar cómo se repetirá o se ajustará esa imagen. En esta ocasión, como solo estamos especificando como colocar colores a los elementos mediante esta propiedad, solo veremos cómo utilizar background para darle color a un elemento.


Para ejemplificar esta teoría, crearemos un contenedor con una estructura cuadrada y le aplicaremos un color base de fondo en RGB.

.ejemplo-color-rgb {
background-color: rgb(70, 70, 194);
}

Gradientes

Los gradientes en CSS permiten crear efectos de transición de color suaves y personalizados en un elemento HTML. Esta propiedad es útil para crear fondos, efectos de texto o bordes decorativos en un diseño web.


Se definen utilizando la propiedad background-image o el shorthand background seguidos del tipo de gradiente y luego entre paréntesis '()' colocamos el valor que queremos que tome nuestro gradiente. Veamos algunos ejemplos:

Sintaxis

.gradiente-grado {
background: linear-gradient(70deg, black, white);
}

Gradientes Lineal: Ejemplos

En este ejemplo, lo primero que vemos son los gradientes lineales, que se extienden en una dirección específica (desde arriba hacia abajo o de izquierda a derecha, por ejemplo). Además, podemos especificar que se vaya extendiendo con una inclinación de “x” grados y tener más de un color.

Gradiente por grado

.gradiente-grado {
background: linear-gradient(70deg, black, white);
}

Gradiente en una dirección

.una-direccion {
background: linear-gradient(to right, #d7d2cc, #304352);
}

Gradiente en una dirección, con varios colores

.varios-colores {
background: linear-gradient(to top, #0f0c29, #302b63, #24243e);
}

Gradiente radial

Por otro lado, tenemos los gradientes radiales, que permiten crear efectos de transición de color suave y personalizados en un elemento HTML en forma circular.

.radial {
background: radial-gradient(#f69d3c, #3f87a6);
}

Gradiente con repetición

Finalmente, Tenemos los gradientes repetitivos, que hará que la extensión de nuestro gradiente ocupe , en el caso de ejemplo, 50px y luego comenzará de nuevo y se repetirá las veces que sea necesario hasta cubrir todo nuestro elemento HTML.

.repeticion-gradiente {
background: repeating-linear-gradient(#f69d3c, #3f87a6 50px);
}
