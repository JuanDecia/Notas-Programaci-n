Selectores Avanzados

Volver a inicio

Introducción

Los selectores avanzados son una regla especial que nos ayudan a identificar elementos o un elemento específico sin la necesidad de crear id o clases a las etiquetas. Es decir, vamos a poder seleccionar los elementos según su herencia o su atributo.

Selector de elementos descendientes

Un selector de elementos descendientes en CSS se utiliza para seleccionar y aplicar estilos a elementos que están contenidos dentro de otro elemento en la estructura de la página HTML. Se pueden seleccionar elementos descendientes utilizando la sintaxis de selector de espacio.

section>
<div class='div-hijo'>Soy un div con contenido</div>
<div class='div-hijo-dos'>
<p>Soy un párrafo hijo de div que a su vez es hijo de section</p>
</div>
</section>

Ahora nuestro objetivo es identificar y seleccionar cada uno de los elementos que se encuentra dentro del elemento 'section'. Para ello vamos a realizar la práctica nombrada anteriormente:

section .div-hijo {
color: red;
}

section .div-hijo-dos {
color: red;
}

section .div-hijo-dos p {
color: blue;
}

Primer Selector

Como podemos observar en el primer caso para seleccionar al primer elemento 'div' primero hemos hecho de la selección del elemento padre llamado 'section' y seguido, separado por espacio el selector al que vamos a apuntar, es decir, al elemento div. Si prestamos atención en esta selección, hemos llamado por su nombre de la clase, esto es para que solo a ese elemento dentro de la estructura de section se le apliquen los cambios en los estilos.

Segundo Selector

En el segundo caso hemos aplicado la misma regla que en el primer selector, a diferencia que esta vez, hemos cambiado el selector al que apuntamos. Es decir, el selector en esta ocasión es la clase .div-hijo-dos. Notemos que este elemento también tiene un hijo, entonces, los cambios que le apliquemos a este elemento le van a afectar también a su elemento hijo.

Tercer Selector

En el último caso de los selectores de este ejemplo, hemos seleccionado el elemento que es hijo del elemento section y a su vez, también es hijo de un elemento div. La estructura para apuntar a este elemento no ha cambiado, solo que hemos agregado un paso más al selector, es decir, luego de llegar hasta el selector .div-hijo-dos dejamos un espacio y agregamos el selector p que es al que deseamos cambiar los estilos. Para tener en cuenta un detalle, con los cambios realizados, el elemento 'p' que se encuentra dentro del elemento 'div' con clase 'div-hijo-dos' va a contener el texto azul, pero, este va a ser el único con este estilo. Ya que, los demás elementos de '.div-hijo-dos van a tener el texto rojo porque así esta declarado en su selector.

Selector de múltiples elementos

Css nos permite aplicar estilos a más de un elemento a la vez, separando cada uno de los elementos a los que queremos aplicar estilos mediante el uso de una coma (",").

Ejemplo: Estructura en HTML

<section>
<p class='parrafo'>Soy el párrafo #1</p>
<h1>Hola Soy el título</h1>
<p class='parrafo'>Soy el párrafo #2</p>
<p class='parrafo'>Soy el párrafo #3</p>
</section>

Ejemplo: Selector Múltiple en CSS

h1 ~ .parrafo,
section h2,
div {
background-color: red;
}

En este caso de ejemplo, aplicará el mismo estilo a los párrafos que sean hermanos consecutivos del h1 con la clase “parrafo”, a todos los h2 que se encuentren dentro de nuestra sección hero y a todos los divs de nuestra estructura HTML.

Selector de elementos hijos directos

Un selector de elementos hijo en CSS se utiliza para seleccionar y aplicar estilos a elementos que son hijos directos de otro elemento en la estructura de la página HTML. Se pueden seleccionar elementos hijo utilizando la sintaxis de selector de mayor que (">"). Por ejemplo:

<section>
<div class='contenido'>Soy un div con contenido</div>
<div>
<p class='contenido'>Soy un párrafo hijo de div que a su vez es hijo de section</p>
</div>
</section>
section > .contenido {
color: red;
}

En este caso, los elementos que son hijos directos del elemento section y tengan la clase 'contenido' se verán afectados por los estilos que pongamos entre las llaves. El elemento 'p' no se verá afectado por los cambios porque no es un hijo directo de 'section'.

Selector de Hermanos Adyacentes

Un selector de hermanos adyacentes en CSS se utiliza para seleccionar y aplicar estilos a elementos que son hermanos directos en la estructura de la página HTML. Se pueden seleccionar elementos hermanos adyacentes utilizando la sintaxis de selector de adyacencia (+).


Este selector se puede utilizar, por ejemplo, para darle funcionalidad al menú de barras que suelen tener las páginas web en su versión mobile y/o tablet.


Cuando hacemos referencia a hermanos directos o adyacentes nos referimos a aquellos elementos que son elementos hijo del mismo elemento padre y que se encuentran colocados en la estructura del código de manera sucesiva.


Por otro lado, es importante destacar que solo se podrá seleccionar a un hermano adyacente que se encuentre con posterioridad dentro del código, y no uno que se encuentre antes del elemento mediante el cual seleccionaremos un hermano directo. Veamos un ejemplo de cómo utilizar el selector de hermanos adyacentes:

Ejemplo: Estructura en HTML

<section>
<p class='parrafo'>Soy el párrafo #1</p>
<h1>Hola Soy el título</h1>
<p class='parrafo'>Soy el párrafo #2</p>
<p class='parrafo'>Soy el párrafo #3</p>
</section>

Ejemplo: Selector hermano adyacente en CSS

h1 + .parrafo {
background-color: red;
}

En este caso, vemos que todos los párrafos y nuestro título son elementos hermanos, ya que son todos hijos directos. Ahora bien como mencionamos anteriormente, el selector de hermanos adyacentes selecciona elementos posteriores a, en este caso, nuestro h1, por lo cual los estilos no van a afectar al párrafo 1.


Por otro lado, mencionamos también que selecciona al hermano que se encuentra de manera sucesiva en el código, lo cual hace que los estilos no se apliquen al párrafo 3.


Por ende, los estilos aplicarán al párrafo 2 , ya que cumple con ambas características necesarias: Estar colocado luego de nuestro h1 en el código y además está colocado de manera sucesiva.

Selector general de hermanos

Un selector de hermanos generales en CSS se utiliza para seleccionar y aplicar estilos a elementos que son hermanos consecutivos en la estructura de la página HTML. Se pueden seleccionar elementos hermanos consecutivos utilizando la sintaxis de virgulilla (~).

Ejemplo: Selector hermanos generales en CSS

h1 ~ .parrafo {
background-color: red;
}

Tomando como base el ejemplo de los hermanos adyacentes, se mantendrá que no se aplicarán los estilos al primer párrafo, ya que si bien son hermanos no es consecutivo en el código, pero si aplicará los estilos a los otros 3 párrafos que están luego de nuestro h1, ya que son elementos hermanos que se encuentran posteriormente en el código que nuestro título.

Selectores de Atributos

Los selectores de atributos se especializan en capturar los elementos según los atributos que tengan en su estructura en el DOM. Esta herramienta puede ser muy útil a la hora de hacer un selector único o realizar un selector múltiple sin la necesidad de crear clases o Id's específicos para capturar un elemento. Estos selectores ofrecen una funcionalidad mucho más avanzada, podemos realizar búsquedas personalizadas y con mejores filtros. También, tenemos la posibilidad de buscar fácilmente un valor al principio, un valor al final o en todo el contenido del valor del atributo.

Selector de atributo General

El atributo de selector general consiste en seleccionar los elementos que tengan un atributo específico, independientemente del valor que tenga ese atributo que estamos por seleccionar. Por ejemplo, este selector es útil para seleccionar todas las etiquetas "a" que tengan el atributo href en su elemento.


Para realizar dicha búsqueda debemos realizar la siguiente sintaxis de este tipo de selector, primero necesitamos referenciar el elemento del DOM que deseamos seleccionar (en nuestro caso de ejemplo es la etiqueta <a>) y luego, entre corchetes ("[]") colocamos el valor del atributo que debe tener el elemento al que estamos seleccionando (en nuestro caso de ejemplo href).

Ejemplo: Sintaxis

a[href] {
background-color: red;
}

Selector del valor de un atributo

Si queremos hacer una selección más específica por medio de un atributo, también, podemos filtrarlos por el valor que tenga el atributo del elemento al que queremos modificar. Es decir, además de seleccionar el atributo vamos a colocar el valor que este tenga.


Tomando el ejemplo anterior, podemos seleccionar elementos <a> que a su vez tengan el atributo href y también, que los valores específicos del atributo 'href' sean http://www.google.com.

Ejemplo: Sintaxis

a[href='http://www.google.com'] {
background-color: red;
}

Selector general del valor de un atributo

Este selector tiene un comportamiento muy parecido al ejemplo anterior, pero es más general. En el ejemplo anterior, debía coincidir por completo la ruta absoluta del valor para tomar el selector, es decir, generalmente es usado para enlaces.


Este selector es más precisamente para usar elementos que tengan atributos y seleccionar los valores generales de esos atributos. Para hacer uso de este selector debemos aplicar el asterisco ("*"). Primero escogemos el elemento, luego seleccionamos el atributo seguido del asterisco y finalizando el valor que nosotros vamos a apuntar para capturar el elemento.

Ejemplo: Sintaxis

div[class*='cont-codigo'] {
background-color: red;
}

Tomamos los div que solo tengan un atributo class con un valor 'cont-codigo' y cambiamos su fondo a rojo.

Selector inicio de valor de atributo

Existe un selector para capturar con que caracter inicia dicho atributo. Es decir, el atributo que comienza con (agregar descripción de inicio). Para aplicar dicho selector debemos utilizar el símbolo mayor que apuntando hacia arriba ("^").

Ejemplo: Sintaxis

a[href^='#'] {
background-color: red;
}

En el ejemplo hemos capturado los elementos a que a su vez, solo en su atributo 'href' comiencen con '#'.

Selector finalización del valor de un atributo

Existe un selector para capturar un elemento según como finaliza su valor, es decir, hace que el selector siempre busque al final de la palabra especificada al final del valor del atributo. Para aplicar este selector debemos hacer uso del símbolo peso ("$").

Ejemplo: Sintaxis

a[href$='.com'] {
background-color: red;
}

En ejemplo de la sintaxis estamos capturando los elementos 'a' que a su vez su valor finalice en .com.

Selector para múltiples valores

Si nuestro símbolo es ~ se usa cuando el atributo tiene múltiples valores separados por espacio entonces se buscará la palabra completa que se especifica en el selector entre todos los valores que tiene el atributo.

Ejemplo: Sintaxis

a[class~='logo'] {
background-color: red;
}

En este caso el selector buscará los elementos a que contengan un atributo class y a su vez, que en todos los valores posibles que tenga ese atributo el valor específico 'logo'.

Selector busqueda de valor

Si nuestro símbolo del selector en valor de atributo es '|' buscará el valor al principio o en todo el contenido. Pero, deberá estar seguido por un guion "|-". Por ejemplo, si tenemos un grupo de imágenes o íconos y el nombre de cada uno comienza con ícono, es decir: icono twitter, icono Facebook, icono etc, para referenciar a todos usamos este selector que es el más indicado.

Ejemplo: Sintaxis

img[src|='icono'] {
background-color: red;
}
