Formas de aplicar CSS

CSS como etiqueta <style>

Para hacer uso de CSS podemos utilizar una etiqueta llamada <style> la cual debe estar ubicada dentro de la etiqueta <head> y luego de la etiqueta <title> dentro de nuestro archivo HTML. Es decir, Podemos agregar estilos a todas las páginas de un sitio web agregando una sección de estilos en el encabezado de cada página, utilizando la etiqueta <style> en el mismo.

<head>
<title>Título del Navegador</title>
<style>
p {
color: red;
}
</style>
</head>

Como podemos observar en el ejemplo práctico, estamos aplicando estilos CSS desde el encabezado de la página, dentro de las etiquetas 'style'. En este ejemplo específico le damos a las etiquetas <p> un color de texto rojo.

CSS como atributo

CSS puede ser aplicado como un atributo desde el elemento (etiqueta) en el mismo archivo HTML donde estamos trabajando. Este debe ser aplicado dentro de la apertura de la etiqueta usando el atributo llamado ‘style’.


Para conceptuar la descripción anterior, cambiaremos el color del texto que se encuentra como contenido en la etiqueta “<p>”, lo que tendremos que hacer en esta ocasión es lo siguiente:

<p style='color: red'>Hola mundo en un párrafo HTML</p>

A la etiqueta de apertura <p> le debemos agregar el atributo “style”, luego aplicaremos los estilos que deseamos para ese elemento, en este caso son la propiedad “color” y el valor “red”. Guardamos los cambios para ver el resultado en el navegador. Este tipo de estilos son llamados estilos en línea (in line), porque estamos agregando los contenidos dentro de la misma etiqueta.

Diferencias entre estilos en encabezado y estilo en linea.

Cuando aplicamos los estilos de CSS en línea, es decir dentro de la etiqueta, solo afectamos al elemento que hemos aplicado el estilo. Por ejemplo, si tenemos varios elementos <p>, sólo se modificará el elemento <p> que hemos seleccionado con el con el atributo style. Si usamos este método en todo el documento y necesitamos que todos los párrafos tengan las mismas características deberemos agregar a cada uno el atributo style, lo cual hace el procedimiento largo y poco eficiente. Como podemos notar este procedimiento no es una buena práctica en el desarrollo web.


En el caso de la etiqueta <style> en el encabezado, las reglas que se escriban en su interior afectarán a todos los elementos seleccionados del archivo HTML. De todas maneras, esta forma de aplicar los estilos de CSS tampoco es la correcta. A continuación, veremos cual es el método que se eligió para aplicar correctamente los estilos de CSS a nuestro documento HTML.

CSS en un archivo externo

Nos queda saber el último paso de como conectar los estilos de CSS a nuestro documento de HTML. Este método es el más recomendado, ya que, nos genera una estructura de trabajo mucho más limpia y ordenada. En este archivo externo de tipo CSS estarán escritas las reglas necesarias para mejorar el aspecto de nuestro sitio Web.


Para tener un directorio ordenado y mantenible, primero, agregaremos una carpeta llamada “CSS”, que será el lugar donde guardaremos las hojas de estilo que sean necesarias para nuestra página. Luego, en nuestro editor en la carpeta nueva creada anteriormente crearemos un archivo con nombre “styles.css” y lo guardaremos. Este archivo, representará la estructura principal de estilos de la página web donde estemos trabajando.

Ahora es necesario conectar ambos archivos, es decir, que estén comunicados entre sí para que los cambios que apliquemos en el archivo de CSS sean impactados en el archivo HTML y ver los cambios que vamos realizando en el navegador. Para efectuar esta comunicación de ambos archivos, debemos utilizar una etiqueta llamada <link /> desde el código HTML que se ubicará dentro de la etiqueta <head> luego de la etiqueta <title>.


La etiqueta <link> debe tener un atributo llamado “rel” con el valor 'stylesheet' y otro atributo llamado “href” con el valor de la ruta relativa de nuestro directorio de trabajo en donde se encuentra el archivo que vamos a conectar. El atributo “rel” especifica el tipo de código que se encontrará en el archivo externo. En esta ocasión son las reglas CSS, y el atributo href se refiere a la ruta donde se encuentra dicho archivo que es una hoja de estilos nombrada “styles.css”.

<head>
<title>Aplicando CSS</title>
<link rel='stylesheet' href='./css/styles.css' />
</head>

Debemos saber también que esta etiqueta no tiene una etiqueta de cierre, porque, al estar en la etiqueta head estamos referenciando que es solo una etiqueta de información. Además, es una etiqueta que no tiene contenido que se muestre en el viewport del navegador (ej: párrafo).


Un archivo de HTML puede tener varias etiquetas “link” con diferentes enlaces de archivos de tipo “.css”. Esto puede ocurrir ya que tal vez cada uno de ellos nos proporcionarán diferentes estilos y funcionalidades de estilos para nuestra web. Estos links nuevos que podemos conectar externos pueden ser por ejemplo una fuente de “Google fonts”, una página de íconos, etc..

<head>
<title>Aplicando CSS</title>
<link rel='stylesheet' href='./css/styles.css' />
<!-- Fonts -->
<link rel='preconnect' href='https://fonts.googleapis.com' />
<link rel='preconnect' href='https://fonts.gstatic.com' crossorigin />
</head>
