Link & Script

Link

La etiqueta link es importante reconocerla ya que es la que nos permite conectar la hoja de estilos (CSS) o todo lo que esté relacionado con CSS en nuestro código de HTML.

Es una etiqueta que genera información al navegador y se ubica en la sección de la etiqueta "head" , con esto se quiere decir que es una etiqueta que no contiene contenido para mostrar en el navegador por lo que es una etiqueta simple y todo su contenido se pasa por las propiedades y atributos cada uno con sus respectivos valores.

La propiedad que necesitamos saber utilizar que contiene esta etiqueta es la llamada "rel" , la que recibe como valor "stylesheet" (hoja de estilo). De esta manera estamos indicando que nuestro documento se conectará (link) a una hoja de estilos (stylesheet).

rel='stylesheet'

El atributo "rel" no es el único que vamos a utilizar dentro de la etiqueta link, si no, que también vamos a utilizar el atributo "href" que es donde colocaremos la ruta donde se encuentra dicho documento. Generalmente usamos documentos que se encuentran en nuestro directorio y para acceder a ellos utilizamos el siguiente comando:

href="./css/styles.css"

Donde "./ se refiere a la ubicación donde estamos navegando en el directorio. Luego, entontramos el directorio de css, que es una carpeta para los archivos de tipo css que nuestro documento tiene, es decir, ahí colocaremos todos los archivos que vayamos a utilizar. Por último, tenemos el nombre del archivo que en este caso es styles.css. Se utiliza este nombre genérico o alguno parecido para los estilos de la página principal.

<link rel="stylesheet" href="./css/styles.css" />.

Estilos en barra de herramienta

Un estilo adicional que debemos saber y es muy util a la hora de crear sitios webs, es el estilo que podemos agregar una imagen al lado de nuestro título en la barra de herramientas. Esto lo podemos lograr gracias al valor en el atributo de "rel".

En este caso, en vez de colocar stylesheet debemos colocar, "icon" seguido del atributo href que es donde se encuentra el archivo que vamos a conectar a nuestro sitio.

<link rel="icon" href="./img/nombrearchivo.png" />

Script

La etiqueta "script" cumple un objetivo muy similar al de la etiqueta link, los aspectos que la diferencia es que la función de "script" es para insertar un código construído en JavaScript. Además, es una etiqueta que puede contener contenido, esto quiere decir que necesitamos cerrar la declaración con una etiqueta de cierre.

<script></script>

Ahora, para enlazar la página donde construímos el JavaScript debemos colocarla dentro de un atributo llamado "src". Seguido de la ubicación del archivo dentro del directorio.

<script src="./js/scripts.js"></script>

Donde ./ es la ubicación del directorio, js es la carpeta donde se encuentra el archivo y todos los archivos que construiremos con el formato "js" y por último, encontramos el nombre del archivo "scripts.js".
