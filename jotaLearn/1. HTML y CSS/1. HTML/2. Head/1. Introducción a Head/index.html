Sección Head

Etiqueta Head

La etiqueta head es la que se encarga de contener las etiquetas meta, link y title, esta última es la única que contiene una etiqueta de cierre, pero sí debemos especificar que "head" tiene un cierre "</head>".


Esta etiqueta se encarga de contener las etiquetas (metadatos) que le dan información específica al navegador para nuestra Web. Esta información contiene el tipo de carácter que puede mostrarse y como es la visualización en el viewport.


Cuando nos referimos a los metadatos, especificamente nos estamos refiriendo a la etiqueta meta, la cual contiene una serie de propiedades y valores que ya contienen palabras reservadas para que el navegador las identifique y aplique los cambios relacionados de esta etiqueta.


En el ejemplo podemos ver su utilización en las lineas 4, 5 y 6. También, debemos tener en cuenta que meta es una etiqueta simple la cual no contiene contenido, sino, que solo debemos pasarle información a través de atributos o propiedades con sus respectivos valores.

Etiqueta Title

La etiqueta de title es compuesta, esto quiere decir que va a contener un contenido que conforma el elemento, en el ejemplo tenemos el contenido de: mi primer título. Al ser una etiqueta con contenido debe tener su propia etiqueta de cierre, osea, "</title>". Dentro de esta etiqueta le damos el valor al título de la página que aparece en el en la barra de herramientas del navegador.
