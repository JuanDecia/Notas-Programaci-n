Introducción

La etiqueta label cuenta con un atributo que es fundamental para asociarla a la etiqueta input. El atributo “for” se utiliza para vincular una etiqueta <label> con el elemento al que está describiendo, permitiendo que el usuario haga clic en la etiqueta para seleccionar el elemento correspondiente (como una caja de texto o una casilla de verificación). Es importante para hacer accesible el formulario a personas con discapacidades, ya que esto les permite seleccionar elementos de un formulario mediante el teclado.

Sintaxis para usar Label

<form>
<label for='nombre'>Ingrese su Nombre: </label>
<input type='text' name='nombre' id='nombre' placeholder='Ej: Juan'
</form>

Ejemplo Label

A continuación crearemos un ejemplo con un label con valor "Ingrese su nombre" el cual estará conectado con la etiqueta de "input". Donde Ingrese su nombre estará dentro de la etiqueta label y se conectará con el campo de input gracias al atributo for..

Si presionamos o hacemos click sobre el texto que dice 'Ingrese su nombre, automáticamente el cursor nos pondrá sobre el campo del input para dejar preparado el ingreso del texto.
