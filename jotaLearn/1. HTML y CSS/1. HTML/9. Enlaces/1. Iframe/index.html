Introducción etiqueta Iframe

Ir a Ejemplo práctico

En esta ocasión vamos a aprender a incrustar paginas externas o internas a nuestra propia página, el método para hacerlo es usando una etiqueta que crea un “marco” dentro de nuestra página y hacer que, en el interior de ese marco, muestre otro sitio Web que puede ser externo.

Insertamos la etiqueta "Iframe" con el atributo src, que contendrá la ruta absoluta o relativa donde se encuentre el sitio que queremos cargar. En este caso, vamos a escribir la ruta relativa de la página "anime.flv".

Para dimensionar el marco en general, dispondremos de los atributos width y height para ponerlo en el tamaño que deseamos. Antes de cerrar la etiqueta iframe, escribiremos una frase que se muestra si el navegador no renderiza el iframe (No se puede cargar el iframe).

Cuando usamos iframe debemos tener cuidado que el sitio no tenga pop-up, que son las páginas que se abren automáticamente en el navegador y ofrecen interacciones con el cliente, por medio de javascript o solicitan datos a través de formularios. Para evitar esto, debemos agregar el atributo sandbox el cual nos permite generar restricciones para los contenidos que se cargan en el iframe.

Atributo Sandbox

El atributo sandbox tiene los siguientes valores posibles:

  • Allow-forms: permite que los formularios en el interior del iframe, envíen datos si el cliente envia la información que allí le solicite. En caso de no indicar este atributo, el cliente puede diligenciar el formulario que se encuentra en el iframe, pero cuando envie los datos el formulario no enviará ninguna información ingresada.
  • allow-popups: permite que se ejecuten ventanas emergentes en caso que el sitio que se carga las contenga. De esta manera si el sitio tiene ventanas emergentes, el navegador las ejecuta sin inconvenientes. Si no indicamos este atributo y la página alojada en el iframe tiene pop-ups, el navegador las bloqueará y las ejecutará.
  • allow-scripts: Permite que el sitio Web ejecute códigos Javascripts del lado del cliente y la posibilidad de acceder a la web cam, nuestros archivos y demás dispositivos que tengamos conectados al pc. En caso de no indicar este el atributo, el sitio alojado no ejecuta los códigos de este tipo.
  • allow-top-navigation: se usa para permitir que el sitio alojado en el iframe a través de uno de sus enlaces puede reemplazar nuestro contenido por otro propuesto por el mismo, es decir, contenido diferente al nuestro. Si omitimos esta línea, evitamos este inconveniente.

Finalmente recordemos que el atributo src especifica la ruta de lo que deseamos mostrar. Junto a este atributo tenemos el "srcdoc" que permite agregar directamente el iframe código HTML.

Una de las formas más comunes de usar el iframe es usando los contenidos de youtube, Google maps o Facebook entre otras. Por ejemplo si queremos insertar un iframe para Google maps, debemos buscar en Google maps iframe para agregar Google maps y a partir de esta búsqueda, nos salen las instrucciones para generar el iframe y agregarlo a nuestro sitio web.

Para finalizar debemos saber que la etiqueta iframe puede contener el atributo allow=”fullscreen” que en los casos que el contenido lo permita, activa la opción “pantalla completa”.
