Sobreescritura

Introducción

El término sobreescritura en POO refiere a sobreescribir una propiedad o método de un elemento padre. Para ejemplificar este caso vamos a crear una clase padre llamada Empleado, la cual va a tener unas propiedades y un método, el cual luego va a ser sobreescrito por la clase hija llamada "gerente".

Debemos tener en cuenta el concepto de herencia, ya que, en la clase hija que va a ser llamada "Gerente" vamos a tener todas las propiedades de la clase padre (Empleado).

Sintaxis Ejemplo

class Empleado { constructor (nombre, sueldo) { this._nombre = nombre; this._sueldo = sueldo; } obtenerDetalles() { return `Empleado nombre: ${this._nombre}, sueldo: ${this.sueldo}.`; } }
Ahora crearemos la herencia a Gerente y luego le realizaremos la sobreescritura al method de la clase padre.
class Gerente extends Empleado { constructor(nombre, sueldo, departamento) { super(nombre, sueldo); this._departamento = departamento; }

Con esta construcción, ya podemos crear nuestro primer objeto "Gerente". Esto lo realizamos de la siguiente manera:

Construcción de objeto Gerente

let gerente = new Gerente('Juan', 'USD 5000', 'Desarrollo');

A continuación, vamos a llamar al método "obtenerDetalles()" el cual se heredó de la clase padre (Empleado).

console.log(gerente.obtenerDetalles());

Resultado

Empleado: nombre: 'Juan', sueldo: USD 5000.

Como podemos observar, hasta el momento no tenemos el resultado del departamento donde pertenece el gerente. Porque, al llamar el método de la clase padre no se encuentra especificado.
Para poder visualizar el resultado, debemos aplicar en este caso el concepto de sobreescritura y para ello vamos a realizar la siguiente sintaxis:

class Gerente extends Empleado { constructor(nombre, sueldo, departamento) { super(nombre, sueldo); this._departamento = departamento; } obtenerDetalles() { return `Gerente: ${super.obtenerDetalles()}, Depto: ${this._departamento}.`; }

Ahora, veamos como quedo el nuevo resultado:

console.log(gerente.obtenerDetalles());

Ahora sí, podemos observar que se está imprimiendo correctamente el mensaje haciendo uso de sobreescritura.
