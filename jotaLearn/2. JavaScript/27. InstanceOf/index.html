InstanceOf

Introducción

Hay ocasiones que debemos saber al trabajar con polimorfismo a que tipo estamos referenciando y este proceso lo podemos lograr gracias a la palabra reservada instanceof. Lo que podemos hacer con este proceso es preguntar si el tipo que estamos recibiendo pertenece a cierta clase

Para poder ejemplificar este concepto vamos a trabajar con las clases de empleado y gerente que hemos creado anteriormente. Recordemos, que tenemos una función que habíamos creado la cual imprimía el resultado de dicha clase. Ahora, a esta función le agregaremos un condicional if para preguntar si pertenece a cierta clase y modificar el resultado que se muestra en el DOM y la consola. La construcción de la función quedaría de la siguiente forma:

function imprimir(tipoClase) { console.log(tipo.obtenerDetalles()); if (tipoClase instanceof Empleado) { console.log('Es un objeto de tipo Empleado'); }; };

El resultado de este proceso nos retorna: Es un objeto de tipo Empleado en la clase Empleado y recordemos que tenemos la clase Gerente y este también va a retornarnos el mismo mensaje. Esto se debe a la relación de clases que tenemos entre Empleado y Gerente. La clase Padre es Empleado y la clase hija es Gerente.

Entonces al utilizar la palabra reservada instanceof en la clase Empleado nos retorna True ya que tiene una relación con la clase padre. Si realizamos la misma práctica, pero, le colocamos el if a la clase hija Gerente, éste nos va a retornar solo el mensaje de la clase Gerente. Esto sucede porque instanceof toma las clases del mismo tipo o las clases de tipo padre.

Si queremos realizar una práctica para preguntar por ambos debemos realizar la siguiente estructura, donde la primer condición debe ser la clase de menor jerarquía:

function determinarTipo(tipoClase) { console.log(tipo.obtenerDetalles()); if (tipoClase instanceof Gerente) { console.log('Es un objeto de tipo Gerente'); }; else if (tipoClase instanceof Empleado) { console.log('Es un objeto de tipo Empleado'); }; };
