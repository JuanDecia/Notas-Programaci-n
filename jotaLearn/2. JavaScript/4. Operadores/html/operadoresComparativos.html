Ejemplos: Operadores Condicionales

Ejemplo es igual (==)

El operador "==" compara si los dos elementos son del mismo valor o si son iguales sin importar el tipo de dato.

let edadJuan = 28;
let edadSol = 38;

if ( edadJuan == edadSol ) {
console.log(`Juan y Sol tienen la misma edad, la condición retorna 'true'.`);
} else {
console.log(`Juan y Sol tienen diferentes edades, por lo que la condición retorna 'false'.`);
}

Si el valor de “Juan” es igual al valor de “Sol” la condición nos retorna true lo que entraría en la primera sentencia donde el resultado nos daría “ambos tienen la misma edad”. Como la edad es diferente la condición nos retorna false y la condición entra en la sentencia de else donde el cuerpo dice que las edades son “diferentes”.

Igualdad Estricta (===)

El operador "===" no solo compara si es del mismo tipo, sino que también compara si es del mismo valor. Es muy recomendado usar esta expresión, ya que, si tenemos que comparar entre un valor “4” de tipo number y un valor “4” de tipo string.

let edadJuan = 28;
let edadString = '28';

if ( edadJuan === edadString ) {
console.log(`Los valores de las edades son iguales.`);
} else {
console.log(`Los valores de las edades son diferentes.`);
}

Operador Desigualdad (!=)

Compara dos valores, retorna true si son diferentes.

let edadJuan = 28;
let edadSol = 38;

if ( edadJuan != edadSol ) {
console.log(`Las edades son diferentes.`);
} else {
console.log(`Las edades son iguales.`);
}

Operador Desigualdad Estricta (!==)

Compara dos valores, retorna true si son diferentes. También, discrimina el tipo de dato

let edadJuan = 28;
let edadString = '28';

console.log ( 28 !== edadJuan ); // False
console.log ( '28' !== edadJuan ); // True

Operador ">" (Mayor que)

Se utiliza para determinar si el valor que antecede a la condición es mayor al que precede. Si la condición se cumple retorna true.

console.log ( edadJuan > edadSol ); // False

Operador "<" (Menor que)

Se utiliza para determinar si el valor que antecede a la condición es menor a la que precede. Si la condición se cumple retorna true.

console.log ( edadJuan < edadSol ); // True

Operador ">=" (Mayor o igual que)

Se utiliza para determinar si un valor es 'mayor' o 'igual' a otro. Si la condición se cumple, retorna true.

console.log ( edadJuan >= 28 ); // True

Operador "<=" (Menor o igual que)

Se utiliza para determionar si un valor es 'menor' o 'igual' a otro. Si la condición se cumple, retorna true.

console.log ( edadJuan <= 28 ); // True
