Ejercicio Promesas

Instrucciones

Construir una promesa y lograr que imprima por consola o por pantalla si la promesa se cumple o no.
La promesa debe ejecutarse luego de 1 segundo, esto lo logramos con setTimeOut. Debemos crear un numero random y si este es par la promesa se cumple de lo contrario ejecutar un error.

Indicaciones

  1. Crea una página HTML básica que importe un script con el código de la promesa.
  2. Agrega el procesador Babel ES6
  3. Invoca a la promesa y realiza lo siguiente:
    • Si la promesa se cumple, imprime el número aleatorio generado.
    • Si la promesa no se cumple, imprime el mensaje de 'Rechazado, no es un número par'.
