Introducción

En esta práctica vamos a crear clases en JavaScript y el objetivo final es mostrar una orden de pedido de una casa de computación con los siguientes datos descriptivos:

  • Mostrar computadoras armadas
  • Mostrar orden de pedido con las siguientes características:
    • Número de orden.
    • Computadora (ID)
    • Monitor (ID, Marca, Tamaño)
    • Raton (ID, Tipo de Entrada, Marca)
    • Teclado (ID, Tipo de Entrada, Marca)
