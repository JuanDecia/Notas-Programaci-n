Plantilla Ejercicio

A continuación llenaremos una lista de eventos, pero lo haremos progresivamente en la medida que el usuario lo solicite. Inicialmente se cargan 5 eventos y se tiene un boton de "cargar mas". Cada vez que se presione el botón se cargarán los próximos 5 eventos.

Tareas a Realizar:

  1. Utiliza el arreglo inicial con 200 eventos. Fíjate que cada evento debe tener un nombre y una descripción, como se muestra seguidamente:
      Evento 1
      1. Nombre: Evento 1
      2. Descripción Esta es la descripción del Evento 1
  2. Define una variable global que indique cuántos eventos se han mostrado hasta el momento.
  3. Define una función que dada la cantidad de eventos cargue 5 más y los agregue al DOM.

Recuerda que inicialmente solo se deben desplegar los primeros 5 eventos.
