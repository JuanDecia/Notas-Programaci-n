Callbacks

Introducción

Este tipo de funciones son conocidas porque son funciones que son pasadas como parámetros a otra función. Es decir, que dentro de una función podemos realizar el llamado a otra función.

Ejemplos

Comenzaremos creando una función de "imprimirMensaje" que trabajará como tipo callback:

function imprimirMensaje(mensaje) { console.log(mensaje); }

A continuación, crearemos la función a la que llamaremos como callback a la función creada anteriormente:

function operacionSumar(op1, op2, funcionCallback) { let res = op1 + op2; funcionCallback(res); // Función del callback }

Como podemos observar es una función donde se le van a pasar dos parámetros para que realice una operación matemática y luego, por la funcion callback llamada 'imprimirMensaje' podemos imprimir el mensaje del resultado. A continuación, realizaremos los llamados necesarios para visualizar los resultados:

sumar(5, 3, imprimir);
