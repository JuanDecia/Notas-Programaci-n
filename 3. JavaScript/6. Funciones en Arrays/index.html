Funciones en Arrays

Introducción

Los arreglos son colecciones de datos que pueden ser extensas y es importante saber una básica manipulación de los mismos. Para ello, tenemos una serie de funciones que nos ofrece JavaScript para realizar filtros y obtener los datos que necesitemos de estos arreglos. A continuación, comenzaremos a ver algunas funciones con ejemplos prácticos para comprender mejor el funcionamiento.

Concat

El método concat () se usa para unir dos o más arrays. Este método no cambia los arrays existentes, sino que devuelve un nuevo array.

Sintaxis

let nuevo_array = viejo_array.concat(valor1[, valor2[, ...[, valorN]]]);

Lenght

Nos permite saber los elementos en valor numérico que tiene un array. Recordemos que tenemos que restarle -1 al valor ya que los elementos comienzan desde 0.

Para aplicar este método debemos primero colocar el nombre del arreglo al que vamos a acceder seguido de un “.” Y luego el nombre de este método. Como resultado, nos retorna la longitud de elementos que tiene el arreglo.

Sintaxis

personas.lenght;

Pop

El método pop elimina el último elemento de un array.

Primero debemos colocar el nombre de la variable que contiene el array seguido de un punto (“.”) Y luego el método pop().

Sintaxis

personas.pop();

Como podemos observar tenemos la lista de nombres y con el método pop() eliminamos el último elemento (Joan). También, lo podemos ver en la consola (línea 56). En el caso que volvamos a repetir el metodo, vamos a continuar eliminando el último elemento de la lista.

Push

El método push agrega un elemento declarado como parámetro en la lista en la última posición.

Primero colocaremos el nombre de la variable que contiene el arreglo seguido de “.” Y luego el método push (). Dentro de los parámetros colocaremos el valor que agregaremos al array, debemos recordar que si es un string debe estar entre comillas, si es otro valor no.

Sintaxis

Tenemos el siguiente arreglo de "personas": const personas = ["Juan", "Joan"];

personas.push("Sol");

Resultado en consola: [“Juan”, “Sol”, “Joan”];

Splice

El método Splice nos permite agregar o eliminar elementos en las posiciones que deseemos de un array.

Para aplicar este método primero debemos llamar al nombre de la variable que contiene el arreglo a modificar, seguido de “.” Para acceder a los objetos de la variable por último debemos colocar el método splice ().

Sintaxis

let personas = ["Juan", "Sol", "Joan", "Abril"];
personas.splice

Dentro de los parámetros de splice colocaremos los argumentos de los valores que estamos modificando.

Debemos saber que dentro de los argumentos existen posiciones separadas por coma “,” cuando estamos construyendo que cada posición tiene una característica. Supongamos que tenemos la siguiente construcción dentro de los parámetros:

personas.splice(3, 1, 27, 27, 2);

La primera posición del parámetro que la ocupa “3” significa a partir de qué índice comienzan los cambios, en nuestro ejemplo comenzaría desde “Abril”.

La segunda posición del parámetro que la ocupa “1” es la cantidad de valores que se eliminan comenzando del índice 0 seleccionado por “3”. En nuestro ejemplo la posición “0” es “Abril” , ósea que el único elemento (está declarado con 1 en la segunda posición del parámetro de splice) que se eliminará es “Abril”.

Desde la tercera posición en adelante se ubicarán los elementos que se agregarán en el arreglo a partir del índice seleccionado en la primera posición (3). Esta declaración nos retornara:

['Juan', 'Sol', 'Joan', 27, 27, 2];

Sort

El método sort () ordena los elementos de un arreglo (array) localmente y devuelve el arreglo ordenado. La ordenación no es necesariamente estable. El modo de ordenación por defecto responde a la posición del valor del string de acuerdo a su valor Unicode.

Debemos tener en cuenta que las mayúsculas tienen prioridad ante las minúsculas. Por ejemplo, "Cherry" viene antes que "banana". También, en Unicode los números vienen antes que las mayúsculas.

Sintaxis

arr.sort([compareFunction]);

Parámetros

Opcional. Especifica una función que define el modo de ordenamiento. Si se omite, el array es ordenado atendiendo a la posición del valor Unicode de cada carácter, según la conversión a string de cada elemento.

Parámetros del orden de elementos

Debemos tener en cuenta que el orden estándar de sort es el de Unicode. Esta aclaración es para tener en cuenta que sort no ordena automáticamente de manera ascendente los números. Para ello debemos usar el parámetro pasándole unas condiciones específicas según el orden que estemos necesitando.

Este es un ejemplo con un orden numérico standard:

let puntos = [1, 10, 2, 21];

puntos.sort(); // [1, 10, 2, 21]

Como podemos observar, 10 se ubica antes que 2.

Para modificar este orden debemos utilizar el parámetro de sort, donde, si se provee compareFunction dentro del parámetro, los elementos del array son ordenados de acuerdo al valor que retorna dicha función de comparación. Siendo a y b dos elementos comparados, entonces:

  • Si, compareFunction (a, b) es menor que 0, se sitúa a en un índice menor que b. Es decir, “a” viene primero.
  • Si, compareFunction(a, b) retorna 0, se deja a y b sin cambios entre ellos, pero ordenados con respecto a todos los elementos diferentes. Nota: el estándar ECMAscript no garantiza este comportamiento, por esto no todos los navegadores (p.ej. Mozilla en versiones que datan hasta el 2003) respetan esto.
  • Si, compareFunction(a, b) es mayor que 0, se sitúa b en un indice menor que a.
  • compareFunction(a, b) siempre debe retornar el mismo valor dado un par especifico de elementos a y b como sus argumentos. Si se retornan resultados inconsistentes entonces el orden de ordenamiento es indefinido.

Entonces, la función de comparación tiene la siguiente forma:

Para comparar números en lugar de strings, la función de comparación puede simplemente restar b de a. La siguiente función ordena el array de modo ascendente:

function compareNumbers(a, b) {
return a - b;
}

Ejemplo sort en un array de strings:

Array de elementos: let elementos = ["Fuego", "Agua", "Tierra", "Aire"];
elementosOrdenado = elementos.sort()
console.log(`Elementos ordenados: ${elementosOrdenado.join(", ")}.`);

Ejemplo sort en un array de números sin parámetro:

Array de números: let arrayNumeros = [2, 222, 543, 345, 15];
numerosOrdenado = numeros.sort();
console.log(`Numeros ordenado estandar: ${numerosOrdenado.join(", ")}.`);

Ejemplo en un array de números con parámetro dentro de sort():

Array de números: let arrayNumeros = [2, 222, 543, 345, 15];
numerosOrdenado.sort((a, b) => a - b)
console.log(numerosOrdenado.join(", "));

Ejemplo en un array de números con función fuera de sort():

Array de números: let arrayNumeros = [2, 222, 543, 345, 15];
function ordenarDecreciente(a, b) {
return b - a;
}
console.log(numerosOrdenado.sort(ordenarDecreciente));

Every

Este método revisa que todos los valores de un arreglo cumplan con lo que indica lo que pasemos dentro del ámbito de la función, teniendo en cuenta este aspecto nos devuelve un valor booleano.

Sintaxis

const ages = [32, 33, 16, 40];
console.log(ages.every((age) => {return age > 10}));

Some

El método some() nos devuelve un Boolean por cada elemento del array según la condición que vamos a pasar dentro del ámbito de la función.

Primero debemos hacer la llamada del arreglo a cual le vamos a aplicar el method some(). Luego, debemos hacer la llamada de la función some(), seguido de la construcción que tendrá el ámbito de la función que será la condición que filtró el arreglo.

Sintaxis

let elementos = ["Agua", "Aire", "Fuego", "Tierra"];
elementos.some((elemento) => elemento === "Aire");
Resultado esperado: true

En el caso del DOM nos retornará el booleando que estamos comparando, pero, en consola nos retornará todos los resultados que comparó el method. Es decir, el resultado en consola es el siguiente:

Resultado esperado: false, true, 2 false

Como podemos observar, nos retornó un valor booleano para cada uno de los elementos del arreglo.

Filter

El method filter() nos recorre un array y nos devuelve un array nuevo con los elementos que cumplan lo establecido en el ámbito de la función del respectivo método.

Para usar este method, primero realizamos la llamada del array que recorreremos, seguido del method filter(). Dentro de los parámetros construiremos la condición que recorrerá el array llamado, para devolvernos un nuevo array con las condiciones que hemos indicado anteriormente.

Sintaxis

elementos.filter(elemento => {construcción de la condición})

Como podemos observar, dentro de los parámetros de filter() primero colocamos una variable en el ámbito del parámetro del método, con nombre que se adapte al proyecto y a la condición que estamos por realizar (en nuestro caso "elemento").

Luego, colocamos la nueva sintaxis para ES6 que es conocida como función flecha que se identifica como "=>", seguido de llaves "{}" en las cuales colocaremos la condición de filter.

A continuación, realizaremos la condición para que filter nos retorne los elementos que tengan más de 5 caracteres. Recordemos que para recorrer los caracteres de un elemento debemos utilizar otro method conocido como length().

let elementosNuevos = ["Fuego", "Agua", "Tierra", "Aire", "Sonido", "Hielo", "Sombra", "Luz"];
elementos.filter(elemento => {
if (elemento.length > 5) {
console.log(`"${elemento}" tiene más de 5 caracteres.`);
else {
console.log(`"${elemento}" tiene menos de 5 caracteres.`);
}
});

En la condición estamos diciendo que si la variable del ámbito nombrada "elemento" es mayor que el largo, de caracteres a 5, al resultado lo muestre en consola con el nombre de ese elemento. Si, el elemento tiene menos de 5 caracteres que muestre ese elemento que corresponde en la consola también.

Para nuestro ejemplo vamos a trabajar con un arreglo con tipos de dato objetos. Los cuales tienen nombre y edad como claves y sus respectivos valores como contenido. La sintáxis para esta práctica es la siguiente:

La condición nos dice que nos realize un filtro de las personas que tengan una edad menor de 19. Donde recorremos el objeto con la variable local "x" y especificamos que recorra la clave "edad". Ahora, tenemos un arreglo nuevo de tipo objeto con los elementos que hayan cumplido el filtro especificado anteriormente, es decir, solo estarán los datos de "Luis" y "Marcos" que son las personas que cumplieron el requisito del filtro.

Ahora, se mostrará en consola y en pantalla las personas que han sido filtradas. Find

El método find() nos retorna el primer elemento encontrado que coincida con la condición en el callback. Si no existe una condición, el método nos retorna undefined. Para ejemplificar este concepto, volvemos a trabajar con la lista de elementos:

let elementos = ["Fuego", "Tierra", "Agua", "Aire", "Metal", "Roca", "Lava", "Hielo"];

Sintaxis

Primero llamaremos al arreglo que vamos a filtrar, seguido del method "find()", dentro de sus parámetros construiremos el ámbito de la función y dentro de esta la condición que filtrará el array.

elementos.find(elemento => {
if (elemento != 'Aire') {
console.log(elemento)
}
})

La condición dice: que nos muestre en consola todos los elementos en el arreglo que no sean "Aire", en este caso nos salen todos los elementos menos aire.

Como podemos observar el method "find()" no nos retorna todos los elementos como "filter", si no, que solo específicamente los que cumplan con la condición.

En el caso contrario qué "elemento" sea igual que "aire" nos lo muestre en consola y esta condición sólo nos retornará "Aire" en la consola.

elementos.find(elemento => {
if (elemento === 'Aire') {
console.log(elemento)
}
})

Sintaxis

Comenzamos haciendo la llamada del arreglo que deseamos verificar si existe cierto elemento en el seguido del method y dentro de sus parámetros colocamos el elemento a verificar. En este caso realizaremos la salida del resultado en la consola y en el DOM.

let elementos = ["Fuego", "Tierra", "Agua", "Aire", "Metal", "Roca", "Lava", "Hielo"];

A continuación, realizaremos un ejemplo para que nos retorne un número negativo y luego vamos a detectar la posición de "Metal":

contResultado[14].innerHTML += `La posición de 'Rayo' es: ${elementosIndexOf.indexOf('Rayo')}.`;

Este method nos busca dentro de un array el último elemento indicado. Es muy útil utilizarlo cuando nos encontramos con elementos duplicados ya que siempre nos buscará el último que se encuentre en el parámetro de "lastIndexOf".

Sintaxis

contResultado[16].innerHTML += `La primera posición del elemento duplicado ('Tierra') es: ${elementosLast.indexOf('Tierra')}. La posición del elemento duplicado es: ${elementosLast.lastIndexOf('Tierra')}.`;
