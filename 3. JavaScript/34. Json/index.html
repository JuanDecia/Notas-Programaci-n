Json

Introducción

Cuando construimos nuestro código a veces es necesario garantizar la portabilidad de nuestros objetos a otros sistemas, que son parte de la solución global al problema que tenemos planteado.

JSON son las siglas JavaScript Object Notation y se considera hoy en día como el formato líder de intercambio de datos en la Web. JS utiliza este formato para representar los objetos, los cuales son seriados en Strings.

Los JSON son Arrays que contienen Objetos. No son declarados como variables, sencillamente se abren los corchetes y todo su contenido se encuentra dentro. Debemos saber también que no permite escritura externa fuera del JSON.

Un tipo de dato JSON tiene su contenido entre corchetes y los objetos entre llaves pero estos no pueden ser instanciados en una variable. En cambio, cuando utilizamos un array si debemos hacerlo.

Otra de sus diferencias los JSON cuando declaramos los objetos sus claves deben estar también en comillas dobles. En su diferencia, los objetos las claves son declaradas como valores por ende, no llevan las comillas dobles. Un ejemplo a continuación:

{ "nombre": "Luis", "edad": 20, "hobbies": [ { "nombre": "ajedrez", "nivel": "alto" }, { "nombre": "pintura", "nivel": "intermedio" } ], "estudiaActualmente": true, "altura": 1.68, }
