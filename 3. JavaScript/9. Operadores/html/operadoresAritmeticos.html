Introducción

A continuación, se realizaran los ejemplos de cada uno de los operadores aritméticos en JS para poder aplicar y visualizar los resultados.

Operador Suma (+)

Tenemos la siguiente variable let edadJuan = 28;. A continuación, crearemos una operación que sume la edad y nos retorne un nuevo resultado.

Ahora le sumamos con el operador '+' un valor de '10'. Entonces Juan tendrá: ${edadJuan + 10}.

Operador Resta (-)

Tenemos la siguiente variable let edadJuan = 28;. A continuación, crearemos una operación que reste la edad y nos retorne un nuevo resultado.

Ahora le restamos con el operador '-' un valor de '10'. Entonces Juan tendrá: ${edadJuan - 10}.

Operador Producto o Multiplicar

Tenemos la siguiente variable let edadJuan = 28;. A continuación, crearemos una operación que multiplique la edad y nos retorne un nuevo resultado.

Ahora multiplicamos con el operador '*' un valor de '10'. Entonces Juan tendrá: ${edadJuan * 10}.

Operador División

Tenemos la siguiente variable let edadJuan = 28;. A continuación, crearemos una operación que divida la edad y nos retorne un nuevo resultado.

Dividimos con el operador '/' un valor de '2'. Entonces Juan tendrá: ${edadJuan / 2}.

Operador Módulo

Tenemos la siguiente variable let edadJuan = 28;. A continuación, crearemos una operación que solo nos retorne el resto de una división entre la edad y el valor a dividir.

Calculamos con el operador '%' el módulo entre '28' y '2'. Entonces, el modulo es: ${edadJuan % 2}.

Operador de Incremento

Tenemos la siguiente variable let edadJuan = 28;. A continuación, crearemos una operación de incremento que solo nos retornará una adición al valor de la variable con valor 28.

edadJuan++;
Incrementamos la variable "edadJuan" con el operador '++' el valor de '28' a '29'. Entonces, el incremento es: ${edadJuan}.

Esto también es como declarar: edadJuan = edadJuan + 1;

Operador de Decremento

Tenemos la siguiente variable let edadJuan = 28;. A continuación, crearemos una operación de decremento que solo nos retornará undecremento al valor de la variable con valor 28.

edadJuan--;
Realizamos un decremento a la variable "edadJuan" con el operador '--' el valor de '28' a '27'. Entonces, el decremento es: ${edadJuan}.

Esto también es como declarar: edadJuan = edadJuan - 1;
