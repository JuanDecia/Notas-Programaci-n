Ejemplo es igual (==)

El operador "==" compara si los dos elementos son del mismo valor o si son iguales.

Sintaxis

if (lucas == abril) {
console.log(`Lucas tiene ${lucas}, Abril tiene ${abril}. El operador == dice que tienen la misma edad.`);
} else {
console.log(`Lucas tiene ${lucas}, Abril tiene ${abril}. El operador == dice que tienen diferente edad.`);
}

Resultado

Si el valor de “lucas” es igual al valor de “abril” la condición nos retorna true lo que entraría en la primera sentencia donde el resultado nos daría “ambos tienen la misma edad”. Como la edad es diferente la condición nos retorna false y la condición entra en la sentencia de else donde el cuerpo dice que las edades son “diferentes”.

Ejemplo es igual (===)

El operador "===" no solo compara si es del mismo tipo, sino que también compara si es del mismo valor. Es muy recomendado usar esta expresión ya que, si tenemos que comparar entre un “4 int” y un “4 str” esta condición nos retornará “false”, ya que, si utilizamos la condición anterior nos retornará true porque “==” solo compara el valor.

Sintaxis

if (lucas === edadLucas) {
console.log("La edad es del mismo tipo");
} else {
console.log("La edad es de distinto tipo (false), porque, 'lucas' es de tipo int y 'edadLucas' es de tipo str.");
};

Operador ">" (Mayor que)

Compara variables numéricas, si la primera es mayor que la segunda retorna true.

if (lucas > abril) { console.log(`Lucas es mayor que abril.`); } else { console.log(`Lucas no es mayor que abril.`); };

Operador "<" (Menor que)

Compara variables numéricas, si la primera es menor que la segunda retorna true. Si no, retorna false.

if (lucas < abril) { console.log(`Lucas es menor que abril.`); } else { console.log(`Abril es menor que Lucas.`); };

Operador ">=" (Mayor o igual que)

Compara variables numéricas, si la primera es mayor o igual que la segunda retorna true.

if (lucas >= abril) { console.log(`Lucas es mayor que abril.`); } else { console.log(`Lucas no es mayor que abril.`); };

Operador "<=" (Menor o igual que)

Compara variables numéricas, si la primera es menor o igual que la segunda retorna true.

if (lucas <= abril) { console.log(`Lucas es menor que abril.`); } else { console.log(`Abril es menor que Lucas.`); };
