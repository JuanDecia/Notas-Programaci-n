Ejemplo Char At

Volver a Inicio Volver a propiedades

Para este ejemplo práctico vamos a crear una variable con nombre "nombre" y le vamos a asignar el valor de "juan decia" (sin mayúsculas). El objetivo de esta práctica es que luego en la consola, nos retorne el nombre y apellido con mayúsculas respecivamente.

A continuación, realizaremos una serie de pasos para que este string tenga las mayúsculas correspondientes. Uno de estos pasos es el charAt() el cual nos dejará cambiar la letra gracias a que nos podemos ubicar en la posición que deseemos.
