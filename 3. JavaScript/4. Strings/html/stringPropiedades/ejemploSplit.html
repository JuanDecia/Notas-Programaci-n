Introducción

Para ver los ejemplos de Split ver la consola del navegador (F12). Se podran ver los resultados en la consola "console" y el código fuente en "source".

En el primer ejemplo (línea 9), podemos ver el resultado del ejemplo de la teoría. Luego, se realiza la misma práctica en una función para que este proceso pueda ser reutilizado cuando se lo necesite. Para este ejemplo usamos otra cadena (Juan Marcelo Decia).

En los resultados de la consola podemos observar como se crean los nuevos arreglos para que estas palabras puedan ser manipuladas como un valor único.
