Strings: Propiedades

Length

Length es una propiedad de todos los objetos de tipo String, que nos retorna un valor numérico del tamaño de la cadena de caracteres que estamos almacenando en la variable de tipo string. Como es una propiedad no se requiere accederla finalizando con paréntesis; para acceder a esta propiedad ponemos el nombre de la variable o el valor del string y finalizamos con un punto y a continuación sin ningún espacio extra la palabra length.

"Primer nombre".length;

var nombre = "Primer nombre";

nombre.length;

Resultado= 13

toLowerCase

El método toLowerCase de los objetos tipo string, nos permite obtener la cadena de caracteres convertida a caracteres en minúscula; los números, símbolos y otros caracteres no se ven afectados. Recordemos que la manera de llamar un método o una función es el nombre de la función finalizado con los parámetros que recibe el método, separados por comas y encerrados entre paréntesis.

Para nuestro caso, ponemos el valor del string, continuamos con un punto, seguido de apertura y cierre de los paréntesis (esta función no recibe ningún parámetro, con lo cual dejamos este espacio vacío) y terminamos con un punto y coma. A continuación, puedes ver un ejemplo:

"CURSO DE JAVASCRIPT" .toLowerCase ();

Resultado= "curso de javascript"

toUpperCase

El método toUpperCase nos devuelve el contenido del string en mayúsculas.

"estoy aprendiendo javascript". ToUpperCase ();

Resultado= "ESTOY APRENDIENDO JAVASCRIPT"

Split

Vamos a utilizar el procesamiento de strings llamado Split el cual crea un arreglo nuevo y nos permite dividir un string en función de un carácter divisor. Entonces, por cada división que crea este “caracter divisor” se crea una posición en el arreglo.

Parámetros

split.(separador)

Especifica el carácter a usar para la separación de la cadena. El separador es tratado como una cadena o como una expresión regular. Si se omite el separador, el array devuelto contendrá un sólo elemento con la cadena completa.

Cuando se encuentra, el separador es eliminado de la cadena y las subcadenas obtenidas se devuelven en un array. Si el separador no es encontrado o se omite, el array contendrá un único elemento con la cadena original completa. Si el separador es una cadena vacía la cadena es convertida en un array de carácteres.

Es recomendable crear una variable para usar con el separador, es decir:

const separadorEspacio = (‘ ‘);

const separadorComa = (‘,’);

Trim

El método trim devuelve el valor del string, eliminando los caracteres en blanco que estén al inicio y al final de la cadena, sin importar la cantidad de caracteres en blanco que contenga. Por ejemplo:

let frase = ' eliminando espacios en blanco ';

console.log(frase.trim());

Resultado= "eliminando espacios en blanco"

Como podemos observar no necesitamos especificar un argumento en los parámetros de este método para que funcione.

También, es importante destacar que, de esta manera sólo estamos “mostrando” el resultado con la función de trim() pero, no lo estamos guardando. Para guardar el resultado, debemos instanciarlo en una nueva variable, es decir, en vez de utilizar console.log() lo que debemos hacer es crear una variable con nombre “resultado” y en su valor aplicar el método trim.

let resultado = frase.trim();

console.log(resultado);

charAt

El método CharAt selecciona una posición en una cadena de caracteres para devolver un nuevo string.

Parámetros

Índice: charAt(indice)

Un entero entre 0 y 1 menos que la longitud de la cadena. Si no se proporciona ningún índice charAt() utilizará 0.

Los caracteres de una cadena se indexan de izquierda a derecha. El índice del primer carácter es 0, y el índice del último carácter en una cadena es - 1. Si el índice que usted proporciona está fuera del rango, JavaScript devuelve una cadena vacía.

La función CharAt es útil cuando estamos trabajando con un procesamiento de líneas y debemos crear una función para cambiar algún carácter específico de alguna palabra ingresada por un usuario.

Slice

El método slice() devuelve una copia de una parte del array dentro de un nuevo array empezando por inicio hasta fin (fin no incluido). El array original no se modificará.

Sintaxis

arr.slice([inicio [, fin]]);

Parámetros

Inicio

Índice donde empieza la extracción. El primer elemento corresponde con el índice 0. Si inicio es omitido el valor por defecto es 0. Si inicio es mayor a la longitud del array, se devuelve un array vacío.

Fin

Índice que marca el final de la extracción. Slice() extrae hasta el índice indicado, pero sin incluir el final. Osea que, si slice indica (1,4), extrae desde el segundo elemento hasta el cuarto (los elementos con índices 1, 2, y 3) recordemos que el último no entra.
