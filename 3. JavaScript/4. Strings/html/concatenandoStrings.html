Concatenando Simple

En este ejemplo tenemos que:

  1. Definir 3 variables string. Las variables deben estar con nombre: "nombre", "apellido" y "curso".
  2. Definir un mensaje de saludo con la siguiente estructura: Hola Juan Decia, bienvenido al curso de JavaScript.
  3. Darle los valores correspondientes a las variables para que se puedan mostrar correctamente la frase al concatenerlas.

Las variables deben quedar de la siguiente manera:

let nombre = 'Juan';

let apellido = 'Decia';

let curso = 'JavaScript';

Para concatenar las variables usamos la función avanzada de JS colocando los símbolos `` (alt + 96) y luego, debemos utilizar el signo $ seguido de llaves ({}) que es donde vamos a colocar el nombre de la variable que mostrará el valor que corresponda, es decir:

console.log(`Hola ${nombre} ${apellido}, bienvenido al curso de ${curso}`)

Concatenando con función

Opcionalmente podemos realizar este proceso construyendo una función, que luego tome los tres parámetros que contienen los valores que necesitamos, es decir: nombre, apellido y curso. Los pasos a seguir son similares, en esta ocasión, debemos hacer lo siguiente:

  1. Definir una función con nombre "saludar" que reciba 3 parametros (nombre, apellido y curso).
  2. Invocar la función para que retorne el saludo.

La función debería quedar de la siguiente manera:

function saludar(nombre, apellido, curso) {

let saludo = `Hola ${nombre} ${apellido}, bienvenido al curso de ${curso}.`;

return saludo;

}

Para mostrar los resultados obtenidos debemos realiar la siguiente declaración:

console.log(saludar("Marcelo", "Decia", "Python"));

Donde primero llamamos a la función para mostrar en consola console.log. Luego, realizamos la llamada de la función saludar() y dentro de los parámetros (parentesis) utilizamos directamete los valores que necesitamos.

Como podemos notar en esta ocasión no hicimos uso de las variables, sino, que directamente colocamos los valores dentro de la función que hemos creado con anterioridad.
