Tipo de dato Objeto

Introducción

Este lenguaje está orientado a prototipos que es un tipo específico de programación orientada a objetos. La POO es una abstracción de los objetos de la realidad plasmado en nuestro código y es aquella programación donde el sistema que estamos implementando se representa como un conjunto de objetos interactuando entre sí. Para ello, necesitamos identificar los mismos y que hacen dichos objetos.

Para crear un dato de tipo objeto debemos crear una variable y asignarles una “clave” y “valor”. El objeto debe estar encerrado entre llaves "{}" y separados por coma entre ellos. Ejemplo:

let objeto = {clave1: "valor1", clave2: 4444, clave3: "valor3"}

Nos puede quedar en una misma línea o, podemos presionar “enter” para que el código sea más legible:

let objeto = { clave1: "valor1", clave2: 4444, clave3: "valor3", };

Esto nos permite crear una variable que tenga distintos tipos de datos, que es equivalente a lo que en otros lenguajes se conoce como registros o una estructura de datos, que puede representarse en una clase objeto.

Una clase objeto es un elemento de la realidad representado abstractamente en nuestro código. Este, va a tener todas las características que sean necesarias representar.

Agregar atributos a un objeto

Nuestro siguiente paso es saber agregar atributos a nuestro objeto. Para ello debemos tener en cuenta que la sintaxis es muy sencilla:

objeto.clave4 = ‘valor4’;

Como podemos observar, tenemos que realizar el llamado de dicho objeto y luego agregar el nombre del atributo que se agregará al objeto, seguido del operador de asignación “=” y finalizando el valor correspondiente a la nueva propiedad, en este caso, “valor4”. Ahora, el objeto nos quedaría de la siguiente manera:

let objeto = { clave1: "valor1", clave2: 4444, clave3: "valor3", clave4: "valor4", };

Eliminar un atributo de un objeto

Para eliminar un atributo de un objeto debemos usar la palabra reservada delete y la sintaxis sería la siguiente:

delete objeto.clave4;

Como podemos observar luego de la palabra clave debemos seleccionar el objeto y luego acceder al atributo que deseamos borrar. Con esta declaración, el objeto que habíamos agregado un atributo, ahora vuelve a estar como en el principio.

Accediendo a los valores de un objeto

Podemos acceder a los valores de los objetos llamándolos por los nombres de su clave. Primero, debemos colocar el nombre de la variable objeto y luego, debemos colcar entre corchetes ([]) el nombre de la clave a la cual queremos acceder a su respectivo valor. Este nombre debe estar colocado entre comillas ("" o '').

console.log(objeto['clave1']); // Retorna: valor1

Cambiando las claves y valores de un objeto

Cambiando Clave

Si llegamos a cambiar en algún momento el nombre de la clave de “nombre” o el de “apellido” nos desplegará los mismos valores aún así habiendo reemplazado los nombres de las claves anteriores.

Para ejemplificar el concepto podemos colocar el valor de la clave “nombre” del objeto persona en una variable nueva (a esta la vamos a llamar “propiedad”) y luego, realizar el llamado del objeto y la variable nueva, la cual nos debe retornar una copia de la “clave-valor” que le hemos declarado a la nueva variable.

Para realizar el llamado y que nos retorne el resultado que esperamos debemos escribir la siguiente declaración:

let propiedad = ‘nombre’; console.log(persona[propiedad]); // Resultado: 'Juan'

Resultado desde JS:

Donde debemos hacer la llamada del objeto (en este caso “persona”) y luego entre corchetes ([]), la llamada de la variable la cual realizamos una copia de la clave-valor del objeto. Se aclara este paso porque, si realizamos nuevamente la llamada de “persona[‘nombre´]” nos retorna el resultado esperado, osea, Luisa. Otro detalle importante para tener en cuenta es que si llamamos por consola el valor de propiedad nos retornará la cadena de string declarada ‘nombre’.

let propiedad = ‘nombre’; console.log(objetoPersona[propiedad]); // Resultado: Juan. console.log(propiedad); // Resultado: nombre. console.log(objetoPersona[‘nombre’]); // Resultado: Juan.

Cambiando valores

Podemos cambiar los valores dentro del objeto nombrando su respectiva ”clave”. Primero, debemos colocar el nombre de la variable objeto; seguido del nombre de la clave a cambiar que debe estar encerrada entre corchetes y entre comillas. Luego, debemos colocar un igual "=" (porque estamos declarando un nuevo valor) y por último, el nuevo valor que le daremos a la clave que anteriormente escogimos.

objeto['clave3'] = "nuevoValor3"; console.log(objeto['clave3']); // nuevoValor3

Creando un objeto con la palabra reservada new

En los ejemplos anteriores vimos cómo podíamos crear un objeto y luego manipular los atributos que tiene el mismo. En esta ocasión, vamos a realizar otra forma de crear un objeto. Esta manera es avisando al sistema que nos guarde en memoria un espacio para el objeto creado y la sintaxis es la siguiente:

let objetoPersona = new Object();

Como podemos observar, en este caso estamos utilizando la palabra clave “new Object()”, donde el sistema nos guarda en memoria espacio para este dicho objeto. Ahora, para agregar los atributos correspondientes el proceso es muy similar a uno de los que hemos visto anteriormente, la sintaxis sería la siguiente:

objetoPersona.nombre = 'Juan'; objetoPersona.apellido = 'Decia'; objetoPersona.edad = 28;

Recorriendo objetos

Con for in

Como los objetos son una colección de datos similar a los arreglos, podemos aplicar bucles para recorrer el contenido que estos tienen en pocas líneas de código. En este ejemplo vamos a utilizar para recorrer un objeto un bucle "for in".

En el siguiente ejemplo vamos a construir el bucle donde utilizaremos un contador llamado ‘clave’, seguido de la palabra clave ‘in’ y finalizando con el nombre del objeto del que vamos a recorrer.

Imprimiendo solo claves

for (clave in objeto) { console.log(clave); };

Imprimiendo los valores

Como lo explica la sintaxis, en este caso solo podremos imprimir el nombre de las claves del objeto. Para imprimir los valores debemos acceder a su valor y esto lo realizamos de la siguiente manera:

for (clave in objeto) { console.log(objeto[clave]); };

Donde al imprimir los resultados seleccionamos primero el objeto a recorrer y luego, la posición. Esto debe estar entre corchetes, el resultado de esta sintaxis nos retornará el valor de cada una de las claves.

Con object.values

Otra forma de recorrer un objeto es utilizando el method “Object.values()”. Este method convierte el objeto en un arreglo, así que para utilizar esta práctica es recomendable guardar una copia del objeto en una variable nueva. Para realizar esta práctica la sintaxis sería la siguiente:

let personaArreglo = Object.values(objetoPersona);

Con JSON

En este caso utilizaremos un tipo de archivo JSON con el method stringify para convertir nuestro objeto en un string de tipo JSON. Más adelante veremos como es el uso correcto de JSON. Pero, ahora simplemente vamos a conocer que es una alternativa más para el uso de nuestros objetos. ya que, este tipo de archivo (JSON) es el más útil para trabajar con objetos.

El primer paso que debemos realizar es guardar el objeto dentro de una variable usando JSON y stringify. La sintaxis sería la siguiente:

let personaJson = JSON.stringify(objetoPersona);

Si imprimimos esta variable de tipo JSON nos retornará las claves y los valores como una cadena string de tipo JSON.
