Ejemplo While

Resultado de la condición del bucle: 'n <= 10':

Guardamos el resultado del contador en un arreglo para mostrar en una linea en el DOM:

let n = 0; let arregloNumeros = []; while (n <= 10) { arregloNumeros.push(n); n++; };

Ejemplo While 2

Resutado de la condición del bucle "x += n".

Creamos el acomulador "x" y luego mostramos el resultado:

let numero = 0; let x = 0; let arregloAcomulador = [];
while (numero < 3) { numero++; x += numero arregloAcomulador.push(x) };

Ejemplo ciclo While con Break

El ciclo se termina luego de que nos retorne un numero impar

let contador = 0;
while (contador < 10) { if (contador % 2 == 0) { console.log("Se encontró el primer número impar y se corta el bucle."); break; } else { console.log("El contador está en: " + contador + "."); } contador++; };
