Arrays

Introducción

Los tipos de datos de listas o arreglos son llamados estructuras de control, ya que, dentro de estos pueden almacenar varias variables de diferentes tipos de datos. Específicamente las listas son declaradas como una variable y sus valores están encerrados entre corchetes (“[]”). Los valores son de tipo string deben estar encerrados entre comillas. Además, cada uno de los valores sean string o de otro tipo deben estar separados por comas.

Índice

Para acceder a las ubicaciones de los elementos dentro del arreglo debemos indicar su respectivo índice. En el caso que deseemos acceder solo a la primera posición y nos muestre en pantalla “Juan” debemos colocar la variable del arreglo seguido de [] y la posición “0”. Esto, nos retorna “Juan”. En los arreglos las posiciones comienzan a contarse desde 0 hasta el último índice de forma creciente.

Podemos acceder desde la última posición si colocamos el índice [-1]. Haciendo este paso recorremos el arreglo de derecha a izquierda, el conteo siempre debe ser creciente, es decir: -2, -3, etc.

nombres[0] // Juan //

En el caso que deseemos visualizar la cantidad elementos de un array y al mismo tiempo visualizar cuales son los elementos tenemos que simplemente hacer la llamada de la variable que lo contiene, luego en la consola nos quedaría un ejemplo parecido a este:

console.log(nombres);

(5) ['Juan', 'Sol', 'Joan', 'Abril', 'Virginia']

Arrays Integrados

A las listas en JavaScript también podemos reconocerlas como matriz. Un detalle que no destacamos y es muy importante es que, cada elemento de la matriz es una cadena, pero ten en cuenta que puedes almacenar cualquier elemento en una matriz — cadena, número, objeto, otra variable, incluso otra matriz. También, puedes mezclar y combinar tipos de elementos — no todos tienen que ser números, cadenas, etc.

let random = ['tree', 795, [0, 1, 2]];

Como vemos en el ejemplo anterior una lista dentro de otra lista también debe estar encerrada entre los corchetes (“[]”) y en el caso de que integre a la lista padre debe estar separada por coma con los demás elementos hermanos.

Para acceder a la lista integrada debemos escoger su posición índice, en el caso del ejemplo anterior es: [2]. Esto nos retornaría [0, 1, 2].

En el caso que deseemos acceder a la posición donde se encuentra estos números luego del primer índice donde seleccionamos esta posición, debemos seleccionar la posición específica que queremos acceder, supongamos que elegimos 1, el ejemplo sería el siguiente:

console.log(random[2][1]); // 1 //

Agregando elementos al array

Uno de los conocimientos básicos que debemos saber para manejar arrays es el de agregar elementos a un array existente. Tenemos diferentes formas de realizar este proceso y vamos a conocer cada uno de ellos. Para los ejemplos prácticos los vamos a poder encontrar en la Página de ejemplos.

1. Push

Una de las formas que podemos usar para agregar un elemento a un arreglo es usando el método push() , donde vamos a agregar el elemento que se encuentre dentro de sus parámetros al final de la posición del arreglo. Su sintaxis sería la siguiente:

arregloExistente.push(elementoNuevo);

2. Lenght

Otra manera que tenemos para agregar un elemento al arreglo es utilizando el largo del mismo. Esto sucede porque el metodo lenght toma siempre el largo del arreglo. Para poder aplicar este concepto debemos saber cual es el último índice de nuestro arreglo porque al elemento que vamos a agregar le debemos pasar el que sería el siguiente.

La sintaxis para este método sería la siguiente:

arreglo[arreglo.lenght] = 'Lucas';

3. Índice

Otra forma que podemos agregar elementos a un arreglo es utilizando la posición del índice que a este le correspondería. Para esta manera, debemos conocer cual es la última posición del arreglo, en el caso de que la ultima posición sea "3", nosotros debemos pasar como parámetro "4" y este ocuparía esa respectiva posición.

Primero seleccionaremos el arreglo que vamos a agregar el elemento, seguido de corchetes ([]), colocaremos el número del indice correspondiente. La sintaxis quedaría como el siguiente ejemplo:

let personas = ['Juan','Joan','Sol','Abril'];
personas[4] = 'Lucas';

Recorriendo arrays con ciclos

Más adelante vamos a ver que son los ciclos iteradores con más especificidad. Pero, ahora lo vamos a utilizar para recorrer el arreglo y nos retorne por consola cada uno de los elementos que este contiene. A este proceso lo podemos lograr con un iterador llamado “for” y la sintaxis es la siguiente:

let personas = ['Juan','Joan','Sol','Abril'];

for (let i = 0 i "menor que" personas.lenght; i++) {

console.log(personas.[i])

};

Donde estamos declarando que por cada posición, es decir, por cada [i] en la longitud (lenght) del arreglo, nos imprima dicha posición.

Existe otra manera similar en JavaScript que es utilizando el mismo proceso pero con una nueva función llamada “for of”. Cambia la sintaxis pero el resultado es el mismo.

for (nombre of personas) {

console.log(nombre);

};

En este caso estamos diciendo al sistema que nos imprima la variable nombre del arreglo de personas.

Array destructuring

A continuación veremos como destructurar un array esto es funcional para poder acceder a los elementos del array de manera más sencilla.

Para hacer el destructuring debemos pasarle los valores (sin las comillas) porque estos tienen que estar como valores al nombre del array, este último como instancia.

Cuando trabajamos con destructuring debemos prestar mucha atencion de poner los elementos en el mismo orden para no alterar sus índices.

Sintaxis y ejemplo

Array de nombres:
const nombres = ['Juan', 'Sol', 'Joan', 'Abril'];

Destructuring:
const[Juan, Sol, Joan, Abril] = nombres;

Al realizar esto aplicamos un console.log del elemento, nos tendria que devolver su mismo valor.

console.log(Juan); // Juan

Realizando este ejercicio no debemos llamar al array ni colocar la posición del elemento que deseamos seleccionar.
