Audio Digital

Volver a Inicio

El audio digital es la codificación digital de una señal eléctrica que representa una onda sonora. Consiste en una secuencia de valores enteros y se obtienen de dos procesos: el muestreo y la cuantificación digital de la señal eléctrica.

Etiqueta Audio

Comenzaremos por la etiqueta "audio", esta etiqueta debe definirse junto con el atributo "src" que representa el archivo a reproducir en el navegador. Para que sea posible reproducir el audio y que sea visible en nuestra web necesitamos ciertos controles, como un botón para pausar e iniciar la reproducción, un control de volumen, y una barra de progreso. Existe un atributo que hace posible que nos representa esta barra de reproducción, su nombre es: controls. Este se define dentro de la etiqueta "audio". Al incluir este atributo podemos usar este elemento en cualquiera de nuestros proyectos.

Existen otros atributos que pueden cambiar el comportamiento de esta etiqueta. Estos son:

  • Autoplay: reproduce el video automáticamente se abre la Web, sin necesidad de dar click en el botón del reproductor
  • Loop: Este atributo se encarga automáticamente de reproducir el audio siempre que termine.
  • preload: Este atributo tiene un comportamiento especial. Cuando estamos trabajando con archivos multimedia que son muy pesados, nuestro sitio Web tiende a ponerse lento y la carga puede ser un proceso demorado. Este atributo se encarga de almacenar nuestros archivos de audio en una memoria temporal después de ser cargado la primera vez, dependiendo del valor que especifiquemos.
  • none: No almacenar nuestros archivos.
  • auto: Almacena temporalmente el archivo multimedia.
