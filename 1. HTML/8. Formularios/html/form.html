Etiqueta form y sus atributos:

Cuando diseñamos páginas o aplicaciones en algún momento necesitaremos crear formularios para capturar y recibir cualquier información de parte de nuestros clientes y usuarios.

La etiqueta "form" es la que debemos usar para crear el contenedor de la estructura de un formulario. Las etiquetas de los formularios se pueden dividir en dos tipos: captura de información y enunciado.

El contenedor del formulario debe encontrarse dentro de la etiqueta "body". Si debemos darle una estructura a nuestro formulario para eso usaremos la etiqueta "fieldset" dentro de la estructura de "form", esta etiqueta generará un encuadre con un borde de un 1px para nuestro formulario. Dentro de "fieldset" debemos agregar la etiqueta "legend" con su respectivo cierre que esta llevará el título del formulario en medio del encuadre generado.
Solo se recomienda utilizar "fieldset" cuando necesitemos específicamente encuadrar el título dentro del formulario.

Título del Formulario

La etiqueta "form" ofrece varios atributos que complementan su funcionamiento, en el ejemplo visualizamos autocomplete y name.

  • Autocomplete: sirve para autocompletar campos dentro de un formulario que en alguna ocasión ya hemos diligenciado. Esto con el fin de evitar repeticiones con el llenado del formulario. Este atributo puede recibir dos valores: “on” y “off”, sino es especificado en la etiqueta "form" el navegador asumirá por defecto el modo “on”, y auto completará los campos que correspondan a información previamente llenada.
  • Name: es un atributo informativo que no desempeña ninguna función dentro de nuestro formulario, es utilizado cuando se tienen muchos formularios y se hace necesario utilizar un nombre para distinguirlos entre sí.
  • Action: este atributo envia los datos del mismo a donde se especifique. Dentro de los valores estará enlazada la ruta de la base de datos que guardará los datos del usuario. Este atributo está asociado al atributo submit que es para el envio de los datos del formulario de la etiqueta input.
  • Method: este atributo recibe dos valores con dos funciones diferentes. El valor “get” el cual nos permite ver el resultado de los datos del formulario en la web, cuando el usuario envie la infomación aparecerá los datos ingresados en la url. El valor “post” envia los resultados del formularios pero estos quedan encriptados respetando la privacidad del usuario que ingreso los datos.
