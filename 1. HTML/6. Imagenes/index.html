Generalmente cuando estemos construyendo nuestro sitio web, es muy usual que tengamos que aplicar el uso de imágenes, estas pueden estar alojadas en nuestro ordenador o podemos buscarla a través de internet y conseguir la ruta donde se encuentra alojada.

Imagen Absoluta

Para comenzar a utilizar las imágenes en nuestro sitio, debemos realizar algunos pasos para generar buenas prácticas en nuestro código. Primero, debemos crear una subcarpeta donde almacenaremos todas las imágenes que vamos a usar en nuestro sitio web. Esta carpeta generalmente se recomienda nombrarla como “img”. También, podemos cargar imágenes a nuestra página siempre y cuando éstas estén almacenadas en algún servidor de la web.

Para incluir las imágenes que hemos seleccionado en nuestro sitio web usaremos una serie de etiquetas. Una cumple una función semántica y la otra es la que va a renderizar la imagen.

La etiqueta que cumple la función semántica es llamada figure, la cual nos prepara un campo en forma de conenedor (caja) para que podamos colocar la imagen. La etiqueta que vamos a utilizar para renderizar la imagen es "img", la cual se le pasan dos atributos. Estos atributos son src y alt.

El atributo principal de esta etiqueta (img) es src ya que como valor, le vamos a pasar la ruta donde estará alojada la imagen. Si la imagen esta en nuestro directorio , la ruta es relativa y, si la ruta donde vamos a buscar la imagen está en la web, es denominada absoluta.

Generalmente para usar las imagenes que esten en nuestro directorio, debemos saber movernos a través de las carpetas como si estuvieramos utilizando la consola de nuestro ordenador. En el caso de que nuestra imagen no está en el directorio en donde nos encontramos trabajando debemos comenzar la ruta con “../” el cual nos permite salir de nuestro directorio y luego seleccionar la ruta relativa donde se encuentra la imagen que estamos necesitando para nuestra página web.

Podemos agregar a la etiqueta img información complementaria para cumplir su contenido en los buscadores a partir de los atributos “alt” y “title”. Se suele aplicar obligatoriamente el atributo alt porque es el respaldo para el usuario si la imagen no llegase a cargar correctamente.

Atributo alt

Este atributo recibe como valor la descripción que dejaremos al usuario si accidentalmente la imagen cargada en la página no pueda llegar a visualizarse.

Atributo title

Este atributo recibe como valor la descripción que le dejamos al usuario si pasa el mouse sobre la imagen. En el ejemplo de la imagen, cuando el usuario realice esa acción le aparecerá una pequeña viñeta con “River Plate”.

Como podemos ver también en nuestro ejemplo encontramos el atributo width que este determina el ancho de la imagen. Es recomendable que al usar este atributo de tamaño no usemos otro atributo ya que la imagen puede quedar distorsionada. Sus medidas pueden variar según el método de medida que estemos usando en nuestro caso son pixeles (px).

Imagen en Servidor

Si vamos a usar una imagen guardada en un servidor muy probablemente deberemos usar la ruta absoluta de la imagen para poder incluirla en nuestra página. Esta práctica es adecuada siempre y cuando nos aseguremos que la imagen no cambiará de ubicación.

Recordemos que una referencia absoluta contiene la ruta completa donde se encuentra el archivo, es decir, protocolo/ servidor y ruta. Esto se puede conseguir fácilmente si hacemos click derecho donde se encuentra la imagen que deseamos utilizar y seleccionamos en la barra de herramientas la opción copiar dirección de la imagen.

Cuando renderizamos el archivo el navegador carga la imagen desde el servidor a partir de la ruta absoluta.

Tamaño en las imágenes

Existen varios procedimientos para cambiar el tamaño de las imágenes en el caso de la etiqueta "img", existen dos atributos que podemos usar para cambiar el ancho y el alto de la imagen que son: width y height respectivamente.

Si usamos medidas arbitrarias la imagen se verá deformada, tanto para el ancho como para el alto, sin tener en cuenta ninguna proporción.

Para cambiar el tamaño de una imagen sin que se deforme asignamos SOLO UN VALOR al ancho o alto de la imagen. Esto hace que la imagen se redimensione proporcionalmente a partir de uno del valor cambiado.

Si al cargar nuestra página en el navegador aparece un símbolo de enlace roto, debemos verificar si el enlace que pusimos en el atributo src=”” está escrito correctamente. De lo contrario, la imagen no se verá en la pantalla.
