Tablas: Etiquetas y Atributos

Las tablas son muy útiles cuando necesitamos mostrar datos bajo un orden específico. Son un conjunto de celdas organizadas en filas (rows) y columnas (cols). Las filas son horizontales y las columnas son verticales. La intersección entre una fila y columna se llama celda.

Para hacer su llamado en un sitio web, es necesario hacer uso de sus etiquetas: "talbe" es la etiqueta principal para declarar la construcción de una tabla, dentro de ella debemos construir las filas y las columnas. Las etiquetas de estas últimas nombradas son: "tr" y "td", todas con sus respectivos cierres.

Para indicar al navegador que haremos uso de una tabla, usaremos la etiqueta "table" y en el caso de ser necesario agregaremos al atributo border=”1” que, este atributo define el grosor de las celdas de la tabla. Cuando no se indica este valor el navegador toma como valor predeterminado “0”, es decir, que no será visible.

Para comenzar con la creación de una tabla luego de usar la etiqueta "table" seguimos con las etiquetas "tr" que harán creación de las filas y para formas las celdas usamos las etiquetas "td" que representan las columnas siempre deben estar en el interior de las etiquetas "tr".

Semántica en una tabla

Para dar un mejor orden a las tablas, existen etiquetas que encuadran el contenido, para un código más ordenado. Estas etiquetas son:

  • thead: en esta etiqueta se colocará solo el encabezado de la tabla. En nuestro caso del ejemplo: Código, Producto y Valor.
  • tbody: en esta etiqueta colocaremos todo el contenido o cuerpo de la tabla, en nuestro ejemplo: los componentes del pc, su código y su valor.
  • tfoot: en esta etiqueta ubicaremos el resultado del contenido de la tabla en caso de ser necesario. En nuestro ejemplo: Total y valor total.

Cabe destacar que no notaremos cambios en la tabla que se renderiza en el navegador, pero son útiles cuando trabajamos en el proceso del diseño de la página. Esto permite manipular mejor la tabla y su estructura dentro del código.

Estilos en las tablas

Para que los textos de las tablas aparezcan resaltados debemos intercambiar la etiqueta "td" por "th", esta última cumple la función del formato de texto "strong". Pero, la etiqueta "th" corresponde a las tablas y "strong" a los párrafos.

Podemos realizar una unificación de celdas con un atributo llamado colspan=”” que sirve para combinar columnas y estas deben estar dentro del mismo "td" o "th".

Dentro de las comillas (“ ”) ubicaremos el número de la combinación de columnas que realizaremos (Caso del ejemplo = 2).

También, existe el atributo rowspan=” ” que sirve para hacer combinación de las filas. Podemos hacer tantas combinaciones posibles sin superar el número de la cantidad de filas o columnas existentes.

Tablas anidadas

En ocasiones posiblemente tendremos que hacer uso de una tabla dentro del contenedor de otra tabla, esto es posible maquetando una tabla como contenedora principal sin el uso de rows y cols. Dentro de ella, podremos construir sub-contenedores de tablas que ocupen el espacio del contenedor principal.

En la actualidad este proceso se consigue construyendo un contenedor principal (para este caso "div") y gracias a los estilos de flexbox podremos disponer de este contenedor como si fuese una tabla esto lo logramos declarando el contenedor principal “div” que tenga un “display” con valor “table”.

Fondo en la tabla

En el caso que quisiéramos agregar una imagen de fondo a nuestra tabla podemos llegar a ese objetivo de dos maneras.

La primera manera es directamente desde nuestro HTML usando la propiedad background dentro de la etiqueta principal del contenedor de la tabla (table) y haciendo uso de una ruta relativa o absoluta de donde se encuentre la imagen que estamos deseando colocar.

La segunda manera es trabajando desde nuestra hoja de estilos CSS. Debemos hacer uso del selector del contenedor principal donde esta alojada la tabla y, usando la misma propiedad en CSS "background" agregamos como valor la imagen que estamos deseando colocarle.

Debemos tener en cuenta que también podemos colocar un fondo a rows (filas) aplicando el background a "td" o a las columnas cols aplicando el background a "tr".
