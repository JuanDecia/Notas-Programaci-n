Selectores del DOM

Cuando trabajamos con CSS es necesario identificar el elemento del DOM al que se le aplicaran los estilos o propiedades necesarias para nuestra página.

Los selectores son un conjunto de expresiones que sirven para referirnos a los elementos que mencionábamos. Existen muchos tipos de selectores que son una poderosa característica que ofrece CSS

A continuación, se muestra una lista donde están las notas de cada uno de los selectores que podemos utilizar en el DOM:

  • Selector de Id y Clase
  • Selector de Atributos
  • Selectores Avanzados
