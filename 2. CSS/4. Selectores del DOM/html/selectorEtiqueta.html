Selector de etiqueta

Volver a Inicio

Como su nombre lo indica estos selectores son aquellos que buscan directamente un elemento DOM según el nombre de la etiqueta seleccionada.

Si estamos dentro de nuestra hoja de estilo y colocamos "body", estaremos seleccionando dicha etiqueta, es decir, seleccionamos todo el elemento. Esto significa que los cambios que se le apliquen en los estilos a esta etiqueta afectarán a todos los elementos que se encuentren dentro de ella también.

Así como podemos seleccionar la etiqueta "body", podemos seleccionar cualquier otra etiqueta.

Para aplicar más estilos en nuestro documento de ejemplo vamos a realizar los siguientes pasos:

Para tener una mejor visualización de los cambios que vamos a realizar, vamos a aplicar en los selectores un atributo llamado background, este aplicara un color de fondo dentro del contenedor seleccionado. Aplicaremos esto para "header", "section" y "footer". Respectivamente aplicaremos como valor al background: azul, verde y negro.

A continuación, Aplicaremos estilos a la etiqueta “footer” para que nuestro pie de página se separe un poco del resto del contenido. Llamaremos a la etiqueta por su nombre y en el contenido agregaremos: margin-top: 20px.

Una breve explicación sobre margin es el espacio que existe entre los contenedores de la página web.

Este tipo de selector tiene una desventaja importante y es que cuando tenemos una Web Page con mucho contenido y nuestro documento html tiene muchas veces la misma etiqueta donde las etiquetas pueden repetirse muchas veces, no podremos aplicarle los estilos a una etiqueta en particular ya que estos harán efectos en todas las etiquetas con ese mismo nombre. Procederemos a ver como se afectan a todas las etiquetas.

Supongamos que agregamos el estilo margin-top: 20px a body, esto afectara a todos los elementos que esten dentro de esta etiqueta y hará que todos los contenedores tengan una separación superior de 20px.
