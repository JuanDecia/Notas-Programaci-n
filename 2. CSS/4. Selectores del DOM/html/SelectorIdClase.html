Selectores de Id y Clase

Volver a inicio

Este tipo de selectores ayudan a filtrar los elementos y son más específicos con los elementos que queremos estilizar por medio de CSS.

Selector ID

Una forma fácil de agregar una "identificación única" o ID a un elemento del DOM es a través del atributo "ID".

Como ejemplo le agregaremos un ID a este mismo párrafo. Para realizar buenas prácticas se recomienda usar nombres acordes con el contenido, en este caso pondremos como ejemplo "parrafo". Una vez hecho asignamos los estilos para el atributo, colocaremos El color de este párrafo en azul.

El selector de "ID" se representa con el signo numeral (#) en la hoja de estilos seguido del nombre que le hemos dado al "ID", en nuestro caso es "#parrafo".

Es muy importante saber que no debemos aplicar un mismo id a diferentes elementos, porque, se considera una mala práctica y además, puede generar un conflicto en los estilos de nuestro sitio.

Selector Class

Cuando queremos darles estilos a múltiples elementos, pero no queremos llenar nuestro código de id porque podría llegar a ser inmanejable. En la construcción de un proyecto grande podemos utilizar otro tipo de selector llamado selector de clase, a diferencia del id, este selector puede localizar un número de elementos en el DOM y aplicarles las reglas o estilos que definamos. Es útil en muchos casos en los que tenemos el mismo componente y en todos debe aparecer el mismo aspecto y presentación dentro de la página. Para hacer uso del selector de clase debemos incluir el atributo class a nuestros elementos HTML y darles un nombre para identificarlos.

Si quisiéramos hacer esto con el selector id, tendríamos que generar un id diferente para cada elemento y luego referenciar cada uno de ellos en la hoja de estilos ocasionando pérdida de mucho tiempo y escritura de código que no necesitamos.

Para este ejemplo colocaremos este selector a las etiquetas h2 de esta página y cambiaremos el estilo del color de las letras a naranja.

Luego, en la sección de estilos se selecciona con un punto (.) seguido del nombre que le hemos dado a la clase, en nuestro caso va a ser .subtitle, y le aplicamos los estilos que la página requiera.
