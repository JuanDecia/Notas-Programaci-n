Display

Introducción

Comenzaremos a ver la propiedad Display para los contenedore de CSS. Esta propiedad es una de las más importantes a la hora de maquetar una página web, ya que, sus opciones establecen el flujo de como se posicionarán los contenidos dentro del contenedor.

La propiedad CSS display, especifica si un elemento es tratado como “block” o “inline”, es decir, si el flujo es en bloque o en línea. El diseño usado por sus hijos (elementos dentro del contenedor) son: “Flow layout” (diseño de flujo), “grid” (cuadrícula), “flex” (flexible).

Formalmente la propiedad display establece tipos de visualización interna y externa de un elemento.

La tipo “externa”, establece la participación de un elemento en Flow layout

La tipo interna establece el layout (diseño) de los hijos.

Sintaxis

La propiedad display se especifica mediante valores de palabras clave. En la actualidad, es mejor especificar el display utilizando una sola palabra clave, las combinaciones de palabras claves aún no están bien soportadas por los navegadores.
