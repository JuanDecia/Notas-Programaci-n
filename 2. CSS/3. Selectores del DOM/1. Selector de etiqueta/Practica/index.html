Selector de Etiqueta

Cuando trabajamos con CSS es necesario identificar el elemento del DOM al que se le aplicaran los estilos o propiedades necesarias para nuestra página.

Los selectores son un conjunto de expresiones que sirven para referirnos a los elementos que mencionábamos. Existen muchos tipos de selectores que son una poderosa característica que ofrece CSS.

Los selectores de etiqueta son aquellos que buscan directamente un elemento en el DOM con el nombre de la etiqueta. Para ver como funciona el selector de etiqueta cambiaremos el "background" (fondo) de la página web. Para ello, aplicaremos un selector de etiqueta "body, lo cual basta con escribir el nombre de la etiqueta en nuestra hoja o sección de estilos de CSS.
Al tener la etiqueta seleccionada aplicar el siguiente estilo: background: rgb(194, 189, 173).

Al igual que con body podemos hacer referencia a cualquier etiqueta con su nombre y de esta manera se le aplicarán los estilos correctamente.

Aplicaremos estilos a la etiqueta “footer” para que nuestro pie de página se separe un poco del resto del contenido. Llamaremos a la etiqueta por su nombre y en el contenido agregaremos: margin-top: 20px.
Una breve explicación sobre margin es el espacio que existe entre los contenedores de la página web.

Este tipo de selector tiene una desventaja importante y es que cuando tenemos una Web Page con mucho contenido y nuestro documento HTML tiene muchas veces la misma etiqueta donde las etiquetas pueden repetirse muchas veces, no podremos aplicarle los estilos a una etiqueta en específico, ya que estos harán efectos en todas las etiquetas con ese mismo selector.
