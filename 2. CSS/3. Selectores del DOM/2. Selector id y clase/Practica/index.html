Selector de ID y Clase

Selector ID

Una forma fácil de agregar una "identificación única" o ID a un elemento del DOM es a través del atributo "ID".

Como ejemplo le agregaremos un ID al título principal de esta página. Para realizar buenas prácticas se recomienda usar nombres acordes con el contenido. Una vez hecho asignamos los estilos para el atributo.

El selector de "ID" se representa con el signo numeral (#) en la hoja de estilos seguido del nombre que le hemos dado al "ID", en nuestro caso es "title".

Sintaxis: (apertura etiqueta)"etiqueta" id="title"(cierre etiqueta)

Sintaxis en estilos: #title

Ahora le aplicaremos los siguientes estilos una vez seleccionado por su ID: centramos, color verde y subrayado.

Selector Class

Cuando queremos darles estilos a múltiples elementos, pero no queremos llenar nuestro código de id porque podría llegar a ser inmanejable, en la construcción de un proyecto grande podemos utilizar otro tipo de selector llamado selector de clase a diferencia del id, este selector puede localizar un número de elementos en el DOM y aplicarles las reglas o estilos que definamos. Es útil en muchos casos en los que tenemos el mismo componente y en todos debe aparecer el mismo aspecto y presentación dentro de la página. Para hacer uso del selector de clase debemos incluir el atributo class a nuestros elementos HTML y darles un nombre para identificarlos.

Si quisiéramos hacer esto con el selector id, tendríamos que generar un id diferente para cada elemento y luego referenciar cada uno de ellos en la hoja de estilos ocasionando pérdida de mucho tiempo y escritura de código que no necesitamos.

Para este ejemplo colocaremos este selector a las etiquetas h2 de esta página. La sintaxis para este selector es la siguiente:

(apertura de etiqueta h2)H2 class="subtitulo"(cierre de etiqueta h2)

Luego, en la sección de estilos se selecciona con un punto (.). Y le aplicamos los estilos que la página requiera.
