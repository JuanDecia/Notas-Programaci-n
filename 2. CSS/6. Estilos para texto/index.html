Estilos para textos

Cuando estamos trabajando en CSS tenemos la posibilidad de cambiar el formato estándar del texto que nos ofrece el navegador, estos estilos pueden ser la posición, diseño y propiedades.

Es importante saber que las fuentes varían mucho en los diferentes sistemas operativos y que puede ser posible que una fuente no esté disponible de un sistema operativo a otro. Hay un conjunto de tipografías mínimas existentes (Arial, Georgia y Verdana) son los estándares de internet y están soportadas para todos los S.O del mundo.

Para acceder a estos cambios debemos utilizar las palabras claves dentro de los selectores de los elementos que nos ofrece CSS. Para estos cambios podemos aplicar:

Text Align

Esta propiedad de texto nos permite alinear el texto en el contenedor donde se encuentra. La propiedad cuenta con los siguientes valores a aplicar:

  • Left: Alinea el texto a la izquierda del contenedor.
  • Right: Alinea el texto a la derecha del contenedor.
  • Center: Alinea el texto al centro del contenedor.
  • Justify: Justifica el texto en el contenedor.

Por ejemplo, si tenemos un contenedor de h3 que ocupa el 100% del su contenedor padre, el text align desplaza el texto que contiene el h3 a lo largo de su "width". En el siguiente ejemplo centraremos el texto segun el ancho del contenedor que lo contiene.

Text decoration

Esta propiedad de texto nos permite decorar el texto al que estamos seleccionando, estas decoraciones son las siguientes y se describen con estos valores:

  • Underline: Subraya el texto del contenedor seleccionado con una línea recta.
  • Overline: Subraya la parte superior del texto seleccionado con una línea recta.
  • Line-through: Tacha el texto del contenedor seleccionado.

Para demostrar un ejemplo de la teoría anterior vamos a subrayar un texto corto con una propiedad underline. Adenás, lo centraremos con text align.

Text transform

Esta propiedad especifica el cambio entre mayúsculas y minúsculas del texto de un elemento. Puede ser usada para que un texto aparezca completamente en mayúsculas, en minúsculas o con la primera letra de cada en mayúscula. Los valores para estos cambios son los siguientes:

  • Capitalize: Fuerza a que la primera letra de cada palabra sea convertida a mayúscula.
  • Uppercase: Fuerza a que todos los caracteres sean convertidos a mayúscula.
  • Lowercase: Fuerza a todos los caracteres a convertirse en minúscula.
  • None: previene a que sea cambiado el tamaño de los caracteres.

Para el ejemplo anterior modificaremos todas los comienzo de palabras en mayúsculas utilizando la propiedad text transform. Además, centraremos el texto con text align.

Font Properties

Las propiedades de las fuentes son las que permiten controlar el tamaño, tipo, grosor o estilo de las letras entre otras cosas.

Font Family

Comenzamos por la propiedad font-family que define una fuente o familia de fuentes, con un orden de prioridad, para utilizar en un elemento seleccionado. En el caso de aplicar más de un valor se separa por “,” (comas).

Para ejemplificar la teoría anterior aplicaremos otra fuente de letras a este mismo párrafo utilizando esta respectiva propiedad. Vamos a colocar la fuente arial o verdana que son las más utilizadas en la web.

Cuando creamos el ejemplo podemos observar que automáticamente se colocan las siguientes fuentes Verdana, Geneva, Tahoma, sans-serif. Esto sucede por si algún navegador no lee la que está declarada en primer lugar, pasa a la segunda y así sucesivamente hasta poder renderizar una fuente disponible.

Font Style

Esta propiedad permite definir el aspecto de una familia tipográfica entre los valores: normal, italic, oblique, entre otros.

Font Weight

Esta propiedad agrega una anchura a los caracteres, por defecto, el valor de esta propiedad es de 400 en el caso que deseemos convertir las letras a “negrita” con esta propiedad podemos usar el valor recomendado de 600 o 700. También, podemos usar el valor de “bold”.

Font size

Esta propiedad modifica el tamaño de la fuente, los valores que puede recibir son en pixeles, em, tallas o porcentaje . Debemos tener en cuenta que en MAC la tipografía se renderiza más pequeña que en Windows. Es importante cuando trabajamos en diseños adaptativos colocar los valores de las letras en em ya que siempre se adaptarán a su viewport.

Letter Spacing

Para controlar el espacio existente entre las letras se usa la propiedad letter-spacing y su medida debe entregarse en píxeles (5px).

Si queremos controlar el espacio que existe entre las palabras debemos utilizar Word-spacing de manera similar a la propiedad anterior.

Sombras y Efectos

La sombra de texto en CSS es una repetición o copia del texto, pero con un leve desplazamiento para generar el efecto usando preferiblemente un tono de color distinto del original. Este efecto se crea a través de la propiedad CSS llamada text-shadow y viene acompañada de varios parámetros:

Text-shadow: 1px 2px 3px red;

Para que esta copia generada sea visible debemos desplazarla hacia algún sentido y lo haremos con los dos primeros argumentos que tiene esta propiedad. Si escribimos 1px en primer parámetro la copia generada se desplazará hacia la derecha. En 2px (segundo) se generará un desplazamiento vertical hacia abajo. El 3px hace referencia al blur o desenfoque que en este caso será de 3px. El argumento final (red) especifica un argumento para la copia. Cabe destacar que si queremos más copias del argumento principal (ej: 5.), no será necesario agregar la propiedad text-shadow 5 veces. Solamente colocaremos una coma después de los valores de la primera sombra y repetiremos los argumentos tantas veces como se requiera.
