Colores Digitales

El color digital son números en computadores ordenados de tal forma que al ser representados en dispositivos como monitores (RGB) o dispositivos de saluda como impresoras (CMYK) los percibimos como color.

Color

Cuando trabajamos con los colores en CSS tenemos diferentes propiedades las cuales le aplican estilo de colores a estas mismas. Generalmente la propiedad color se encuentra con las propiedades avanzadas en las que no hay que dar muchos detalles para aplicarlos.

En el caso que utilicemos colores sencillos debemos llamarlo por su nombre (en inglés). También podemos buscar páginas online en donde encontraremos el nombre de todos los colores que podamos aplicar, son 140 colores que podemos aplicar con nombre.

Para ejemplificar esta teoría, crearemos un contenedor con una estructura cuadrada y le aplicaremos un color base de fondo.

RGB

Cuando aplicamos un color en RGB estamos aplicando una combinación de colores que estos colores son (en inglés) red, green, blue. La medición de estos colores es numérica y dependiendo el valor numérico que le demos a estos valores es la graduación que aplica a la combinación.

Para aplicar esta propiedad debemos colocar la palabra clave rgb() y dentro los paréntesis colocaremos los valores numéricos separados por “,” (coma). Debemos tener en cuenta que los colores respetan el orden del nombre de la propiedad (red, green, blue).

En el caso que tengamos el valor (0, 0, 0) estamos diciendo que no hay combinaciones de colores y por ende nos retornaría el color “negro”, es decir, estamos aplicando una ausencia de color. Debemos saber que estos colores van de 0 hasta el 255 siendo el 255 la tonalidad más fuerte, es decir, que si colocamos este valor a uno de RGB y los demás los dejamos en 0 nos retornaría el color natural que lo representa. Ejemplo: 0, 0, 255 (retorna blue).

Para ejemplificar esta teoría, crearemos un contenedor con una estructura cuadrada y le aplicaremos un color base de fondo en RGB.

RGBA

Esta propiedad de color que puede ser implementada en la hoja de estilos CSS permite añadirle trasparencia (representa la letra a “Alpha”) al texto en el último valor agregado con “.”.

Ejemplo: rgba(68, 241, 15, 0.5) el último valor puede ser de 0 a 1. Teniendo en cuenta que 0 es totalmente trasparente (el texto no se va a ver) y 1 no hay trasparencia (se ve igual). Los valores que deberíamos manejar son decimales como por ejemplo 0.5 que es una trasparencia media.

Para ejemplificar esta teoría, aplicaremos el mismo color que el ejemplo anterior, pero en este caso colocaremos una trasparencia.
