Etiqueta Label

Volver a Inicio

La etiqueta label cuenta con un atributo que es fundamental para asociarla a la etiqueta input, esta se llama for y el valor que le debemos especificar es el correspondiente al nombre del dato que se ingresa o al id que le pondremos al input.

Su funcionalidad está relacionada al trabajo con JavaScript ya que nos otorga la posibilidad de crear un contenedor a parte y mejora la manipulación del mismo, gracias a la propiedad que anteriormente hemos nombrado como for.

A continuación crearemos un ejemplo con un label con valor "Ingrese su nombre" el cual estará conectado con la etiqueta de "input". Donde Ingrese su nombre estará dentro de la etiqueta label y se conectará con el campo de input gracias al atributo for..
