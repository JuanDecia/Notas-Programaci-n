Formulario con controles

Select: Nos sirve para crear una lista desplegable de opciones, por ejemplo, cuando el navegador te pide que ingreses la cuidad donde vivís y el buscador despliega una lista de las ciudades disponibles.
Para agregar opciones a esa lista, debemos definir dentro de la etiqueta "select" la etiqueta option, osea, select > option.

Textarea: Es un área de texto que tiene un tamaño mucho más grande que un input tipo texto y sirve para ingresar párrafos, comentarios u observaciones largas. Este tiene 2 atributos especiales, que sirven para definir el tamaño (cols; rows) ambos reciben valores numéricos y representan el número de filas y columnas que puede tener el textarea.
