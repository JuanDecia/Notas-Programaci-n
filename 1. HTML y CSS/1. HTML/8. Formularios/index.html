HTML dispone de una amplia gama de opciones de entrada de datos que van desde las más básicas, como los tipos de datos de texto hasta otras más sofisticadas, como los checkboxes y radio buttons.

Tan importante como capturar datos lo es validar que ellos han sido correctamente introducidos por el usuario. HTML proporciona soporte a estas validaciones y verificaciones, entre las cuales se encuentra chequear que un campo obligatorio no está vacío.

Las etiquetas semánticas cuya finalidad es ofrecer información relacionada al sentido y significado del contenido de nuestras páginas. Ellas son usadas por los buscadores para poder ubicar y posicionar nuestras páginas en las listas resultantes de las búsquedas de los usuarios.

Objetivos de aprendizaje

  1. Describir las etiquetas HTML empleadas para crear formularios y capturar datos con la finalidad de soportar la interacción del usuario con una página Web.
  2. Usar las etiquetas de enlace para enganchar diferentes contenidos en una página Web.
  3. Identificar las etiquetas usadas para otorgar estilos a los códigos de computación, con la finalidad de facilitar el diseño de contenidos que involucran el despliegue de pedazos de programas computacionales.
  4. Crear páginas web que integren el uso de etiquetas y atributos HTML en la presentación del contenido.
