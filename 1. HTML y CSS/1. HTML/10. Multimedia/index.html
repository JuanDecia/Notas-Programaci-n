Introducción a archivos multimedia en HTML

  • Ir a "Media en la web"
  • Ir a "Audio"
  • Ir a "Video"
  • Ir a "Etiqueta Video"

Las etiquetas de HTML para integrar elementos multimedia, son conocidas como las etiquetas "audio" y video. Dada la amplia gama de formatos de archivos multimedia y la falta de compatibilidad de estos formatos con los navegadores existentes en el mercado, exploramos la etiqueta "source" que complementa perfectamente la integración de ítems multimedia en nuestro sitio Web.

Una de las características más importantes de HTML es permitir integrar componentes creados con otras tecnologías. Es por ello que nos detenemos en los componentes embebidos para incorporar archivos CSS y Javascript en HTML y la etiqueta "iframe" para enmarcar nuestras páginas.
