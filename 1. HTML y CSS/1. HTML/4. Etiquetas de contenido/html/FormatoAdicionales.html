Existen etiquetas especiales cuando queremos representar código fuente en nuestro HTML. En esta ocasión realizaremos un mini tutorial de python usando las etiquetas de formato de html y las etiquetas para códigos fuentes. Para ello vamos a necesitar utilizar algunas etiquetas que cumplen una función de formato de texto específico.

Etiqueta var

Esta etiqueta crea un contenedor en línea, en los que puede contener cero o más elementos. Al ser una etiqueta que puede contener elementos debemos cerrarla con su respectiva etiqueta de cierre. Solo cumple con la función de formatear el texto a itálica en representación de la instancia de una variable o el argumento de un programa.

Para esta ocasión la vamos a utilizar cuando representemos la variable "x".

Etiqueta code

El elemento HTML code inserta texto que representa código de computadora. Puede ser útil para mostrar funciones, código de programación o variables. En nuestro elemplo utilizaremos el formato de la etiqueta code para representar el código que escribiremos en python.

Etiqueta pre

El texto en este elemento típicamente se muestra en una fuente fija, no proporcional, exactamente como es mostrado en el archivo. Los espacios dentro de este elemento también son mostrados como están escritos. Para nuestro ejemplo es muy importante que apliquemos esta etiqueta ya que debemos respetar los espaciados del código como si estuvieramos en la sección de python.

Etiqueta Samp

El elemento HTML Sample (samp) se utiliza para incluir texto en línea que representa una muestra (o cita) de la salida de un programa de ordenador. El contenido de esta etiqueta es renderizado generalmente usando la tipografía monoespaciada por defecto del navegador.

Para nuestro ejemplo utilizaremos esta etiqueta para mostrar el retorno del programa.

Ejemplo:

Definimos la variable x y usamos la instrucción print() 

        
    x = "Hola Mundo"
    print(x)

Salida de consola

Hola Mundo

Más etiquetas de formato

Así como existen las etiquetas de formato común, tales como "h1", "h2", "p". . Existen muchas otras que indican un formato específico. Estas son las siguientes:

Etiqueta Descripción
<abbr> Para abreviaciones
<b> En negrita
<blockquote> Referencia de un texto
<cite> Cita
<del> Texto eliminado (tachado)
<em> Énfasis
<ins> Texto insertado
<meter> Medida
