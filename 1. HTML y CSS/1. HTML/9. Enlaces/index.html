Para crear una estructura de enlaces básicos tendremos que usar la etiqueta <a>. Entre la apertura y el cierre de la estructura de este elemento (cierra con </a>), agregaremos el contenido que se anclará en el viewport y que se verá como un enlace donde el usuario puede hacer 'click' y luego, esta ancla (enlace) lo redirigirá. La etiqueta <a> (anchor en inglés) recibe como atributo 'href' el cual debemos pasar como valor la dirección relativa o absoluta que tendrá el enlace.


Si en nuestra página estamos creando un proyecto donde debemos abrir otro archivo alojado en nuestro directorio la ruta será absoluta en el caso de que el enlace deba abrir una página externa a nuestro proyecto debemos pasar como valor en el 'href' la ruta relativa de esa nueva página a abrir.


Cuando renderizamos la página web podremos observar que estarán los links creados con la descripción que se han colocado entre las etiquetas <a> como contenido del elemento. Si pasamos el mouse sobre el link, en la parte inferior izquierda del navegador podemos visualizar una pequeña pestaña informativa de la ruta donde nos redireccionará ese link.


Para evitar que el contenido de nuestro sitio web, sea reemplazado por el de la página vinculada agregamos el atributo target=”” con el valor _blank. Esto hace que se abra una nueva pestaña con el contenido vinculado.

<a href='https://www.google.com' target='_blank'>Enlace a Buscador Google</a>

Enlaza secciones internas

Si queremos crear hipervínculos desde la página principal a paginas internas, primero, debemos crear las páginas interiores. En este caso, la página principal debe estar ubicada en la raíz de la carpeta del proyecto con el nombre de index.html. En esta página que es la principal, agregaremos los enlaces que nos llevarán a otras páginas dentro del mismo sitio web.


Para representar el ejemplo de esta practica, debemos crear una carpera llamada html que es donde se encontrarán alojados todos los archivos de tipo '.html' que van a estar anclados para que el usuario pueda navegar por el directorio de páginas de nuestro sitio. Una vez que tengamos el directorio llamado html, creamos los archivos '.html' que sean necesarios.

Como necesitamos referenciar las páginas ubicadas en una estructura de directorios que se encuentra en nuestro computador local, debemos usar rutas absolutas.

Ejemplo

<a href= './html/estructuraEnlaces.html'>Estructura Enlaces</a>

Anclas

Al generar enlaces hacia páginas internas de nuestro sitio web y luego volver a la página principal, estamos creando una estructura de navegación dentro de nuestro sitio web. A veces, necesitamos generar una estructura de navegación que funcione dentro de la misma página (misma hoja). En el caso de tener un gran volumen de contenido en una misma hoja, podemos usar anclas para crear la navegación dentro de la misma página para enriquecer la experiencia del usuario.


Para aplicar el concepto de ancla podemos agregar un grupo de enlaces en la parte superior de la página, que permita a los visitantes saltar rápidamente a los títulos o subtítulos del contenido. Al finalizar la lectura o visualización de esa sección es muy recomendable volver a crear un enlace de tipo ancla para que nos vuelva a posicionar en el título principal o hacia el tope de la página.


Sintaxis

En la sintaxis de esta construcción lo primero que debemos tener en cuenta, es que debemos crear una propiedad ID en las etiquetas que se van a comportar como anclas. Por ejemplo, al contenedor de esta sección le colocaremos un ID con valor ancla para que desde el header con un solo click nos traiga hasta esta posición. También, crearemos un ancla en el título principal para que nos lleve hacia arriba cuando finalizamos de visualizar el artículo.

Ejemplo ID y Ancla

<article id='ancla' >
<h1 id='titulo' >

Para hacer uso de las anclas construiremos la etiqueta <a> y en el atributo href colocaremos “#” seguido del valor que le hemos dado al id a la etiqueta que deseamos crear como ancla. Como ejemplo vamos a dirigirnos desde esta posición hacia el título.

<a href='#titulo'>Volver a Inicio</a>
Volver a Inicio

Las etiquetas anchor podemos aplicarla a cualquier elemento que deseemos que tenga una referencia a cualquier página externa, interna o alguna parte de la página que estamos visualizando en el momento. Por ejemplo, podemos aplicarla a imágenes, párrafos, títulos, entre otros elementos más.
