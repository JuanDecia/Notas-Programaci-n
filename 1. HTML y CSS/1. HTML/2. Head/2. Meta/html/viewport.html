¿Qué es el Viewport?

Volver a inicio

El viewport es un concepto que aparece junto con el diseño web adaptativo y el diseño web, fue pensado para que los sitios web pudieran adaptarse a los dispositivos móviles.

Este concepto hace referencia al área disponible que se va a renderizar una página web , es decir, el área visible del navegador excluyendo la barra de búsqueda, marcadores, pestañas y demás herramientas del navegador.

El "viewport" se configura dentro de la etiqueta meta a través de propiedades y valores. Entre los valores de configuración que podemos especificar se encuentran:

  • Nivel de escalado que puede realizar el usuario "user-scalable".
  • Zoom Inicial que debe realizar el navegador"initial-scale".
  • Área disponible para renderizar "width-device".
