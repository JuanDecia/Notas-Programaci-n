Introducción

La etiqueta meta son las que nos permite definir el comportamiento de nuestra página y que también ayudan a posicionar mejor a la página dentro de las búsquedas del navegador. Estas etiquetas no tienen etiqueta de cierre por lo que solo se colocan propiedades o atributos con sus respectivos valores. Como hemos visto anteriormente, estas etiquetas son colocadas dentro de la estructura de la etiqueta head.

Sintaxis de meta

Generalmente la etiqueta meta se contruyen con una propiedad llamada "name", la cual contiene un valor que representará el comportamiento de dicha etiqueta. Al valor lo debemos colocar dentro de una propiedad llamada "content" la cual se le asigna el valor que realmente modificará la etiqueta.

En el siguiente ejemplo podremos ver una sintaxis de como construir la etiqueta meta:

(meta name="viewport" content="width=device-width" /)

Debemos tener en cuenta que existen diferentes tipos de valores para las propiedades de name y content.

Propiedades y valores de la etiqueta Meta

A continuación, se describen algunos atributos o propiedades con sus respectivas funciones para tener en cuenta en los proyectos.

Propiedad Charset

La propiedad charset indica el formato de caracteres que va a ser renderizado por nuestro sitio web. Como valor debemos especificar los caracteres como nuestro formato de caracter es el latín, debemos usar el siguiente valor: "UTF-8".

La sintaxis de esta propiedad en la etiqueta sería la siguiente:

(meta charset = "UTF-8" /)

Propiedad Name

La propiedad name recibe el valor que modificará la etiqueta y el comportamiento de nuestra página y el navegador. Name puede recibir los siguientes valores:

  • Viewport: Este valor declarado en la propiedad de name trabaja sobre el viewport de nuestra página, es decir, el espacio visible del navegador.
  • Author: Este valor especifica quién es el propietario del sitio web.
  • Keywords: Este valor identifica las palabras claves que corresponden al sitio web. Su funcionalidad es ayudar a la página a que sea encontrada más facilmente.
  • Description: Este valor nos permite crear una descripción breve del sitio web

Propiedad Content

Esta propiedad se coloca conjuntamente con name, ya que, content va a recibir el valor que realmente va a modificar el comportamiento de la etiqueta. Esta propiedad puede recibir más de un valor declarado.

A continuación, se especificarán la propiedad content de cada una de las propiedades de name respectivamente:

La propiedad name que recibe el valor viewport, en su propiedad content rebibe varios valores que modifican su comportamiento. Entre esos valores podemos encontrar los siguientes en la lista:

  • width
  • initial-scale
  • user-scalable

Cada uno de estos valores se comportan como un atributo, porque, ellos también deben recibir un valor específico.

Width

Width va a recibir como valor device-width, esto se traduce a que el viewport del dispositivo (device) tendrá el ancho (width) de la pantalla donde se esté visualizando.

Initial-scale

Este valor-atributo, refiere a el zoom que tendrá el sitio web al iniciarse en su pantalla principal. El valor que generalmente se coloca es de 1.0.

user-scalable

Este valor-atributo, refiere a que si el usuario puede o no hacer zoom en el sitio. Generalmente, el valor que recibe es no, ya que, si está activado podría ser negativa la visita del usuario en el sitio que se está visitando.

A continuación se colocará una sintaxis que contendrá a todos estos valores dentro de la misma etiqueta de meta.

(meta name = "viewport" content = "width = device - width, initial-scale = 1.0, user-scalable = no" /)

Author

La propiedad de name que recibe el valor de author debemos especificar su verdadero comportamiento en el valor de la propiedad de content, este debe ser justamente, el dueño del propietario de la página. Por ejemplo, Juan Decia.

(meta name = "author" content = "Juan Decia" /)

Keywords

En el valor de content de esta construcción meta debemos colocar palabras clave que estén relacionadas con el sitio.

(meta name = "keywords" content = "etiqueta meta, notas, aprendizaje, html" /)

Description

El valor que le debemos pasar a content de esta construcción de la etiqueta meta debe ser un breve titulo a lo que se refiere la página.

(meta name = "description" content = "construcción de metadatos en html" /)
