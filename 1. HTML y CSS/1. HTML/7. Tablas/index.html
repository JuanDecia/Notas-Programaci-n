Tablas: Etiquetas y Atributos

Las tablas son muy útiles cuando necesitamos mostrar datos bajo un orden específico, son conocidas como un conjunto de celdas organizadas en filas (rows en el código) y columnas (cols en el código). Las filas son horizontales y las columnas son verticales. La intersección entre una fila y columna se llama celda.


Para renderizar una tabla en un sitio web, es necesario hacer uso de sus etiquetas: <table> es la etiqueta principal para declarar la construcción de una tabla, dentro de ella debemos construir las filas y las columnas. Las etiquetas de estas últimas nombradas son: <tr> y <td>, ambos elementos del DOM con sus respectivos cierres.


Como habíamos especificado anteriormente, la creación de una tabla comienza con la etiqueta <table> donde el navegador prepara el espacio para renderizarla. A continuación, debemos crear las celdas, para realizar esta estructura primero debemos crear una fila las cuales se renderizan con las etiquetas <tr>. Para finalizar con la estructura de una celda usaremos las etiquetas <td> que representarán las columnas, es muy importante destacar que siempre deben estar en el interior de las etiquetas <tr>. Los textos que están dentro de las celdas deben escribirse dentro de las etiquetas <td>.

Construcción semántica de una tabla

Para que el navegador identifique de mejor manera las secciones de la tabla, existen etiquetas que encuadran el contenido según la jerarquía que cumple dentro de esta estructura. Estas etiquetas semánticas son:

  • <thead>: en esta etiqueta se colocará (generalmente) el encabezado de la tabla. Generalmente, se trata de la primera fila de nuestra tabla en donde dentro de esta estructura semántica utilizaremos la etiqueta <th> (table head) en vez de <tr> para completar el contenido de las celdas.
  • <table> <thead> <tr> <th>Nombre</th> <th>Apellido</th> </tr> </thead> </table>
  • <tbody>: en esta etiqueta colocaremos todo el contenido o cuerpo de la tabla. En esta sección, utilizaremos las etiquetas <td> en vez de las <th>.
  • <table> <thead> <tr> <th>Nombre</th> <th>Apellido</th> </tr> </thead> <tbody> <tr> <td>Juan</td> <td>Decia</td> </tr> </tbody> </table>
  • </tfoot>: en esta etiqueta semántica ubicaremos el resultado del contenido de la tabla en caso de ser necesario. Como ejemplo de esta sección podemos colocar un total y valor total.
  • <table> <tfoot> <tr> <th>Total: </th> </tr> </tfoot> </table>

Estilos en las tablas

Negritas en el encabezado

Para que los textos de las tablas aparezcan resaltados debemos intercambiar la etiqueta "td" por "th", esta última cumple la función del formato de texto bold para resaltar el contenido en el encabezado, además, ayudamos a los navegadores a identificar que es la fila que contiene los títulos de nuestra tabla.

Estilos en bordes

Cuando estamos agregando estilos desde nuestra stylesheet vamos a encontrarnos que al especificar la propiedad border se nos crean 2 lineas de bordes que son las correspondientes de los contenedores de las celdas (<td> o <th>). Para corregir este estilo y solo nos quede una línea de borde, debemos especificar el siguiente estilo en los respectivos selectores de los contenedores de las celdas:

table, th, td { border: 1px solid black; border-collapse: collapse; }

Unificar Celdas

Podemos realizar una unificación de celdas con un atributo llamado colspan=”” que sirve para combinar columnas y estas deben estar dentro del mismo "td" o "th". Dentro de las comillas (“ ”) ubicaremos el número de la combinación de columnas que realizaremos (Ejemplo: colspan='2').


También, existe el atributo rowspan=” ” que sirve para hacer combinación de las filas. Podemos hacer tantas combinaciones posibles sin superar el número de la cantidad de filas o columnas existentes.

Tablas anidadas: Display Table

En ocasiones posiblemente tendremos que hacer uso de una tabla dentro del contenedor de otra tabla, esto es posible maquetando una tabla como contenedora principal sin el uso de rows y cols. Dentro de ella, podremos construir sub-contenedores de tablas que ocupen el espacio del contenedor principal.


Este proceso se consigue construyendo un contenedor principal (para este caso "div") y gracias a los estilos de flexbox podremos disponer de este contenedor como si fuese una tabla esto lo logramos declarando el contenedor principal “div” que tenga un “display” con valor “table”.

Fondo en la tabla

Trabajando desde nuestra hoja de estilos CSS. Debemos hacer uso del selector del contenedor principal donde está alojada la tabla y, usando la misma propiedad en CSS "background-image" agregamos como valor la imagen que estamos deseando colocarle. Debemos tener en cuenta que también podemos colocar un fondo a rows (filas) aplicando el background a "tr" o a las columnas.
