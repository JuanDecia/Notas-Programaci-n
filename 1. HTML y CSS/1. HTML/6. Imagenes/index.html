Manejo básico de imágenes en HTML

  • Más acerca de Figure y Figcaption

Generalmente cuando estemos construyendo nuestro sitio web, es muy usual que tengamos que aplicar el uso de imágenes, estas pueden estar alojadas en nuestro ordenador o podemos buscarla a través de internet y conseguir la ruta donde se encuentra alojada. Estas maneras pueden conocerce como ruta absoluta o ruta relativa. A continuación, comenzaremos a detallarar cada una de ellas para profundizar su conocimiento.

Imagen Absoluta

Para comenzar a utilizar las imágenes en nuestro sitio, debemos realizar una serie de pasos para continuar el aprendizaje de las buenas prácticas en nuestro código. Primero, debemos crear una subcarpeta donde almacenaremos todas las imágenes que vamos a usar en nuestro sitio web. Esta carpeta generalmente se recomienda nombrarla como “img”. También, podemos cargar imágenes a nuestra página siempre y cuando éstas estén almacenadas en algún servidor de la web.


Para incluir las imágenes que hemos seleccionado y almacenado en nuestro sitio web usaremos una serie de etiquetas para que nuestro navegador las renderice. Una cumple una función semántica y la otra es la que va a renderizar la imagen.


La etiqueta que cumple la función semántica es llamada figure, la cual nos prepara un campo en forma de conenedor (caja) para que podamos colocar la imagen. La etiqueta que vamos a utilizar para renderizar la imagen es "img", la cual se le pasan tres atributos. Estos atributos son src, alt y title.


El atributo principal de esta etiqueta (img) es src, ya que, como valor le vamos a pasar la ruta donde estará alojada la imagen. Si la imagen esta en nuestro directorio, la ruta es absoluta y, si la ruta donde vamos a buscar la imagen está en la web, es denominada relativa.


Generalmente para usar las imagenes que esten en nuestro directorio, debemos saber movernos a través de las carpetas como si estuvieramos utilizando la consola de nuestro ordenador. En el caso de que nuestra imagen no está en el directorio en donde nos encontramos trabajando debemos comenzar generalmente la ruta con “../” el cual nos permite salir de nuestro directorio y luego seleccionar la ruta relativa donde se encuentra la imagen que estamos necesitando para nuestra página web.


Si la imagen que vamos a agregar, está ubicada en la carpeta img se le denomina ruta absoluta porque, se construye con la dirección exacta de la ubicación del archivo. Es decir, debemos especificar el camino exacto desde la ubicación donde se encuentra el archivo que estamos construyendo, hasta el archivo donde se encuentra la imagen que va a ser renderizada en el navegador. En este proceso debemos indicar el nombre de la carpeta donde está ubicado el archivo (img), más el nombre del archivo. En el caso que se requiera realizar a más accesos a los directorios debemos seguir los pasos de la navegación de la consola hasta llegar al archivo de la imagen que deseamos abrir.

<img src= "img/River.png" alt= "River Plate" title= "River Plate" />

En este ejemplo de imagen absoluta podemos observar que nuestra imagen se encuentra en un directorio llamado img que en este se encuentra la imagen que hemos seleccionado con nombre River.png. Se dice que es absoluta porque realizamos la ruta específica para dar con la imagen.


Podemos agregar a la etiqueta img información complementaria para mejorar el SEO y ayudar a reconocer de manera más efectiva la página en los buscadores. Para realizar esta práctica debemos usar a los atributos “alt” y “title”. El atributo alt es el respaldo para el usuario si la imagen no llegase a cargar correctamente y el atributo title es para ayudar al navegador a reconocer el nombre de la imagen.

Atributo alt

Este atributo recibe como valor la descripción que dejaremos al usuario si accidentalmente la imagen cargada en la página no pueda llegar a visualizarse.

Atributo title

Este atributo recibe como valor la descripción que le dejamos al usuario si pasa el mouse sobre la imagen. En el ejemplo de la imagen, cuando el usuario realice esa acción le aparecerá una pequeña viñeta con “River Plate”.

Imagen Relativa

Si vamos a usar una imagen guardada en un servidor muy probablemente deberemos usar la ruta absoluta de la imagen para poder incluirla en nuestra página. Esta práctica es adecuada siempre y cuando nos aseguremos que la imagen no cambiará de ubicación.

<img src= "https://i.pinimg.com/564..." alt= "Estadio-River" title= "Estadio-River" />

Recordemos que una referencia relativa contiene la ruta completa donde se encuentra el archivo en el servidor, es decir, protocolo/ servidor y ruta. Esto se puede conseguir fácilmente si hacemos click derecho donde se encuentra la imagen que deseamos utilizar y seleccionamos en la barra de herramientas la opción copiar dirección de la imagen. Cuando renderizamos el archivo el navegador carga la imagen desde el servidor a partir de la ruta absoluta.

Tamaño en las imágenes

Existen varios procedimientos para cambiar el tamaño de las imágenes en el caso de la etiqueta "img", existen dos atributos que podemos usar para cambiar el ancho y el alto de la imagen que son: width y height respectivamente. Si usamos medidas arbitrarias la imagen se verá deformada, tanto para el ancho como para el alto, sin tener en cuenta ninguna proporción.


Para cambiar el tamaño de una imagen sin que se deforme asignamos SOLO UN VALOR al ancho (width) o alto (height) de la imagen. Esto hace que la imagen se redimensione proporcionalmente a partir de uno del valor cambiado. También, es importante destacar que estos cambios de estilos es recomendable realizarlos desde las hojas de estilos de tipo .css (tema que veremos al finalizar html), ya que, es una buena práctica cambiar los estilos desde dicha página y no desde el propio archivo HTML.


Si al cargar nuestra página en el navegador aparece un símbolo de enlace roto, debemos verificar si el enlace que pusimos en el atributo src=”esta dirección” está escrito correctamente o que la ruta sea la indicada. De lo contrario, la imagen no se verá en la pantalla.
