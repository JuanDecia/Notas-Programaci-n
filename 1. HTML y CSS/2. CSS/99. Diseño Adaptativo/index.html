Diseño Adaptativo

Introducción

Esta estación es un paso inicial por el concepto y características del Diseño Web Adaptativo o Responsive y la introducción del término Viewport. Esto trata de que las imágenes y videos se vean a una forma proporcional a la dimensión de las pantallas de nuestros equipos.

Hablaremos de la media queries, que se usan para adaptar el formato de los contenidos a las características de los dispositivos. Asociados a la media queries están los media types que describen la categoría de un dispositivo.

Finalizaremos estudiando diversos Patrones de Diseño Web que ayudan a definir las características de una página web, tanto lo que corresponde con la distribución de sus elementos, como su adaptabilidad a diversos dispositivos. Entre los patrones que exploraremos están: Tiny Tweaks, Mostly Fluid, Column Drop, Layout Shifter y Off Canvas.

Objetivos de aprendizaje

  1. Interpretar el concepto y las características del Diseño Web Adaptativo con la finalidad de poder aplicarlo a los contenidos Web Responsive.
  2. Aplicar el concepto de viewport con la finalidad de especificar el área disponible para renderizar una página web.
  3. Usar los media queries y media types para adaptar el formato de los contenidos a las características de los dispositivos.
  4. Aplicar patrones de Diseño Web Adaptativo para la creación de contenidos Web Responsive.

Diseño Adaptativo

Responsive Web Design es una técnica que permite desarrollar paginas web que se adapten a cualquier dispositivo. Consiste en usar HTML y CSS3 con el fin de acomodar los contenidos de una página para que se vean bien en cualquier tipo de plataforma.

Para lograr que un proyecto web sea adaptativo, debemos determinar el viewport, o el área de la página web que será visible para el usuario, y definir un conjunto de condiciones (media queries) y reglas CSS3 de acuerdo con cada tamaño de pantalla. Estableciendo las características de los elementos para que puedan verse correctamente en cada dispositivo.

Existe una amplia gama de frameworks CSS para diseños web de este tipo, cuenta con herramientas muy útiles al momento de desarrollar proyectos que requieran adaptabilidad.

Ventajas del diseño web adaptativo

Cuando creamos una página web es importante pensar en los dispositivos de los usuarios que usan al ingresar a ella. El Diseño Web Adaptativo permite que los proyectos puedan verse bien en los diferentes dispositivos.

¿Por qué es importante?

La primera razón es la compatibilidad visual con los diferentes dispositivos. Si una página web no cuenta con este diseño al acceder a ella desde un dispositivo móvil el navegador mostraría una versión reducida de lo que sería una versión de escritorio.

Otra de las ventajas es la codificación de un solo sitio web para los diferentes dispositivos. Si nuestra página no cuenta con esta característica deberemos hacer un código para cada una de las versiones incrementando la dificultad al momento de realizar cambios o actualizaciones.

Por último, también contamos que nuestras paginas sean encontradas con más facilidad en los portales web porque cuando alguien está buscando información en un navegador (ej: Google) este retornara resultados con prioridad a las páginas que cuentan con diseño adaptativo.

Media adaptativa

Imágenes Adaptativas

Cuando colocamos las imágenes a nuestro sitio web, siempre debemos realizar la revisación de la visualización en los diferentes dispositivos con la herramienta de desarrolladores del explorador. Con este paso podemos notar si la imagen se adapta a la pantalla de cierto dispositivo, el error más común es que el tamaño excede a del dispositivo.

Ya hemos trabajado con la etiqueta img con el atributo width o height aplicando medidas en pixeles. Ahora, vamos a cambiar la medida para que las imágenes realmente sean adaptativas y para ello debemos manejar las medidas en porcentaje. Recordemos de ahora en adelante que el tamaño máximo será de 100% para que la imagen no se salga de la pantalla o del contenedor donde se encuentre. Si mermamos o aumentamos el tamaño del navegador, la imagen se adaptará a esta modificación y siempre ocupará el 100% del espacio donde se encuentra. Recordemos hacer la prueba con la resolución de móvil y Tablet.

Videos Adaptativos

Luego agregamos el video a nuestra hoja html con la etiqueta video. El problema que puede existir en un video que además sea más ancho que la pantalla, solo se puede reproducir a 360px. Para modificar esto vamos a crear unos selectores en nuestro archivo CSS que nos permiten cambiar el tamaño del video. Debemos tener en cuenta que en la etiqueta video debemos sacar la etiqueta width.

En nuestra hoja de estilos, seleccionaremos la etiqueta de nuestro ejemplo video y le agregaremos ahí mismo el width. Para cuando se agrande la pantalla del video y no exceda el máximo de pantalla se recomienda agregar también el estilo max-width con un valor 100%. Con esta regla el video no superara el máximo de ancho del browser cuando se maximice el video.
