Introducción

El valor absoluto se posiciona tomando en cuenta el contenedor que lo contiene, este posicionamiento justamente es absoluto. Es decir, si tenemos un elemento de tipo “div” padre, el elemento “div” hijo es el que contendrá el valor absoluto. Podemos decir que es un valor que trabaja fuera del elemento.

Absolute se maneja con las propiedades de top, right, bottom y left. A diferencia de “fixed” este valor se queda en la posición establecida en el contenedor y no en el viewport.

Cuando estamos trabajando con el valor absolute es muy importante que al elemento padre debemos declararle que su posición es relative.
