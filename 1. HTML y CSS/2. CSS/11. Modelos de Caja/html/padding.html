Introducción

Padding es el espacio generado dentro de la caja entre el contenido y el borde. Esta propiedad tiene el mismo comportamiento que margin en cuanto a que es una propiedad shorthand y a cuanto a la declaración de valores

h2 {
padding: 10px; /* Genera espacio en todas sus coordenadas */
padding: 10px 5px; /* Genera espacio en top - bottom de 10px y en right-left de 5px */
padding: 0 0 0 10px; /* Genera espacio de 10px en left */
}

Ejemplo Padding

Soy un párrafo sin padding.

A continuación, vamos a modificar el elemento 'p' que contiene el texto agregando un padding de 15px en todos sus lados.

.parrafo-padding {
padding: 15px;
background-color: rgba(127, 255, 212, 0.432);
}

Soy un párrafo con un padding de 15px en todos sus lados

Como podemos observar se ha separado el contenido (el texto) de los bordes internos de la caja en una medida de 15px, también, debemos notar que el contenedor principal no ha cambiado su ubicación original en relación (izquierda/ superior) ya que no hemos modificado el margin.
