Introducción

Es el espacio que existe del borde de la caja hacia afuera. Margin tiene un comportamiento especial, al ser una propiedad con característica shorthand puede recibir como mínimo un valor y como máximo cuatro. Los valores de margin pueden estar en px (pixeles), vw/ vh (escala de viewport) o porcentaje (%). La sintaxis sería la siguiente:

h2 {
margin: 20px 0 20px 0;
}

Como podemos observar en la sintaxis están declarados los cuatro valores de margin los cuales comienzan desde top y siguiendo la orientación del reloj hacia la izquierda (top - right- bottom- left).

La caja contenedora tiene lugar desde el borde de “padding” (amarillo) siendo que margin (celeste) es el espacio que este nos ofrece para separar los bordes de los elementos que se encuentren agrupados alrededor.

Margin: Valores

Margin: auto

Además de los valores numéricos que recibe margin también, nos encontramos con el valor “auto”. Toma como referencia las posiciones “right” y “left” del viewport para centrar el contenedor (no utiliza como referencia top y bottom). Es decir, que la orientación de esta propiedad es horizontal.

h2 {
margin: auto;
}

Si deseamos utilizar ambas orientaciones (posición horizontal y posición vertical), debemos declarar dos valores en margin. Es decir, primero debemos colocar el valor que corresponde a las posiciones de top y bottom y segundo el valor que corresponde a los valores de rigth y left. Nos quedaría una sintaxis como la siguiente:

h2 {
margin: 25px auto;
}

Margin: inherit

Este valor aplica una heredación de propiedad, es decir, se debe aplicar esta propiedad a un elemento hijo y este tomará la propiedad del elemento padre. Para ejemplificar este concepto, vamos a crear un elemento "div" con un margin de 20px y vamos a crear dentro de este elemento, un elemento "p" al cual le vamos a declarar margin: inherit para que este tome el mismo valor que el elemento padre.

.contenedor-parrafo {
margin: 20px;
}

.parrafo-inherit {
margin: inherit;
}

Margin: un valor

Si colocamos un único valor en margin este tomará el mismo para todas las coordenadas (top - right - bottom - left). Existen diferentes formas para colocar un valor a una coordenada.

Primer Forma

Una de las maneras que aplicamos un solo valor a la propiedad de margin es declarando un único valor el cual toma como referencia a todas las coordenadas del elemento seleccionado.

h2 {
/* Aplica 25px en todos los bordes externos */
margin: 25px;
}

Segunda Forma

Como anteriormente hemos nombrado, margin es una propiedad shorthand. Esta aclaración es adecuada en este momento porque como hemos visto que si declaramos un solo valor éste toma como referencia a todos los lados del elemento.


Para que esto no suceda, debemos aplicar la siguiente manera de declarar un solo valor para margin. Para aplicar un valor y que solo tome de referencia ese único valor, debemos declarar los demás valores pero en 0. A continuación podrán ver un ejemplo de referencia:

h2 {
/* Aplica 25px solo en bottom */
margin: 0 0 25px 0;
}

Tercera Forma

La tercera forma que vamos a aplicar es no utilizando la shorthand que nos ofrece la propiedad margin. Es decir, vamos a utilizar la propiedad específica que apunta la coordenada que deseamos modificar. A continuación, podremos ver un ejemplo:

h2 {
/* Aplica 25px solo en bottom */
margin-bottom: 25px;
}

Como podemos observar, con margin podemos especificar la coordenada seguido de la palabra margin. En este caso, también estamos aplicando sólo un valor y si tratamos de agregar otro valor en este tipo de declaración ninguno será tomado porque no lo reconocería el navegador, es decir, se genera un error.

Margin: dos valores

Si hacemos el uso de dos valores en margin las coordenadas especificadas son: primero “vertical” (top y bottom) y segundo “horizontal” (right y left). Al declarar un valor en respectivas posiciones, éstas aplican ese dicho valor tanto en la posición vertical como horizontal, es decir, si tenemos margin 20px 40px la posición vertical separa los contenidos (top y bottom) en 20px y la posición horizontal (right y left) separa los contenidos en 40px.

h2 {
margin: 20px 40px;
}

También, tenemos la posibilidad o podemos encontrarnos con el valor “auto” en uno de los dos valores lo cual centraría el contenido en el que especifiquemos.

h2 {
margin: 20px auto;
}
