Estilos para textos

Introducción

Cuando estamos trabajando en CSS tenemos la posibilidad de cambiar el formato estándar del texto que nos ofrece el navegador, estos estilos pueden ser la posición, diseño y propiedades.


Se puede controlar la tipografía de un texto especificando atributos como el tipo de letra, el tamaño, el estilo (negrita, cursiva, etc.), la alineación, el color, entre otros. Las fuentes varían mucho en los diferentes sistemas operativos y puede ser posible que una fuente no esté disponible de un sistema operativo a otro. Hay un conjunto de tipografías mínimas existentes (Arial, Georgia y Verdana) son los estándares de internet y están soportadas para todos los sistemas operativos que existen.

Font Properties

Font-Family

Comenzamos por la propiedad font-family que define una fuente o familia de fuentes, con un orden de prioridad, para utilizar en un elemento seleccionado. En el caso de aplicar más de un valor se separa por “,” (comas).

body {
font-family: 'Open Sans', sans-serif;
}

Font Style

Esta propiedad permite definir el aspecto de una familia tipográfica entre los valores: normal, italic, oblique, entre otros.

p {
font-style: italic;
}

Soy un párrafo en italics.

El estilo "normal" es el estilo de fuente predeterminado y no incluye inclinación. El estilo "italic" incluye una inclinación hacia la derecha y es comúnmente utilizado para destacar texto o dar una sensación más relajada a la tipografía. El estilo "oblique" es similar al "italic", pero es generado automáticamente por el software en lugar de estar disponible como una fuente independiente.

Font Weight

Esta propiedad agrega una anchura a los caracteres, por defecto el valor de esta propiedad es de 400 en el caso que deseemos convertir las letras a “negrita” con esta propiedad podemos usar el valor recomendado de 700 o usar el valor de “bold”.

p {
font-weight: 700;
font-weight: bold;
}

Soy un párrafo con un font weight en 700.

Los números que se utilizan para especificar el grosor de la fuente varían de 100 a 900, donde 100 es muy delgado y 900 es muy grueso. Las palabras clave "normal" y "bold" son sinónimos de 400 y 700, respectivamente. Algunos otros valores comunes incluyen "lighter" y "bolder", que ajustan el grosor de la fuente en relación al grosor predeterminado del elemento.


Es importante tener en cuenta que no todas las fuentes tienen todos los valores de grosor de fuente disponibles. Por lo tanto, es importante elegir una fuente que tenga el grosor de fuente deseado.

Font Size

Esta propiedad modifica el tamaño de la fuente, los valores que puede recibir son en pixeles, em, tallas o porcentaje. Debemos tener en cuenta que en MAC la tipografía se renderiza más pequeña que en Windows. Es importante cuando trabajamos en diseños adaptativos colocar los valores de las letras en em, ya que siempre se adaptarán a su viewport.

p {
font-size: 20px;
font-size: 1.5rem;
font-size: 10%;
}

Soy un parrafo con un tamaño de 1.3em

Es importante tener en cuenta que el tamaño de la fuente también puede ser influenciado por otros factores, como la configuración del usuario o las preferencias de accesibilidad en el navegador. Recomendamos que al elegir la fuente se elija un tamaño de fuente que sea legible y accesible para la mayoría de los usuarios , sin exagerar en el tamaño de la misma.

Text Align

Esta propiedad de texto nos permite alinear el texto en orientación horizontal dentro de un elemento HTML. La propiedad cuenta con los siguientes valores a aplicar:

  • Left: Alinea el texto a la izquierda del contenedor.
  • Right: Alinea el texto a la derecha del contenedor.
  • Center: Alinea el texto al centro del contenedor.
  • Justify: Justifica el texto en el contenedor.
  • Inherit: Hereda la alineación de texto del elemento padre.
div {
text-align: justify;
}

Soy un texto alineado por defecto en izquierda.

Soy un texto alineado en derecha.

Soy un texto alineado al centro.

Soy un texto justificado, ocupo siempre el 100% del renglón.

Soy un texto alineado en relación al elemento padre.

Vertical Align

Esta propiedad se utiliza para establecer la alineación vertical de un elemento en relación a su elemento padre o contenedor. Los valores más comunes de esta propiedad son top, middle y bottom.

.top {
vertical-align: top;
}

.middle {
vertical-align: middle;
}

.bottom {
vertical-align: bottom;
}

En este ejemplo, los elementos con la clase .top tendrán una alineación vertical en la parte superior, los elementos con la clase .middle tendrán una alineación vertical en el medio y los elementos con la clase .bottom tendrán una alineación vertical en la parte inferior.

Text Shadow

La sombra de texto en CSS es una repetición o copia del texto, pero con un leve desplazamiento para generar el efecto usando preferiblemente un tono de color distinto del original. Este efecto se crea a través de la propiedad CSS llamada text-shadow y viene acompañada de varios parámetros.


.texto-sombra {
/* offset-x | offset-y | blur | color */
text-shadow: 1px 2px 3px red;
}

Para que esta copia generada sea visible debemos desplazarla hacia algún sentido y lo haremos con los dos primeros argumentos que tiene esta propiedad. Si escribimos 1px en primer parámetro la copia generada se desplazará hacia la derecha. En 2px (segundo) se generará un desplazamiento vertical hacia abajo. El 3px hace referencia al blur o desenfoque que en este caso será de 3px. El argumento final (red) especifica un color. Cabe destacar que si queremos más copias del argumento principal (ej: 5.), no será necesario agregar la propiedad text-shadow 5 veces. Solamente colocaremos una coma después de los valores de la primera sombra y repetiremos los argumentos tantas veces como se requiera.

Soy un texto sombreado

Text Transform

Esta propiedad especifica el cambio entre mayúsculas y minúsculas del texto de un elemento. Puede ser usada para que un texto aparezca completamente en mayúsculas, en minúsculas o con la primera letra de cada en mayúscula. Los valores para estos cambios son los siguientes:

.texto-transformado {
text-transform: uppercase;
text-transform: lowercase;
text-transform: capitalize;
}

Soy un párrafo convertido a mayúsculas.

soy un párrafo en lowercase por defecto.

soy un párrafo con cada primera letra en mayúscula.

El valor “uppercase” convertirá el texto a mayúsculas, el valor “lowercase” a minúsculas y “capitalize” transformará en mayúscula la primera letra de cada palabra.

Text Decoration

Esta propiedad se utiliza para aplicar o quitar decoraciones al texto de un elemento html, como subrayados, tachados y líneas por encima.

.texto-decorado {
text-decoration: underline;
text-decoration: none;
text-decoration: overline;
text-decoration: line-through;
}
  • Underline: Subraya el texto del contenedor seleccionado con una línea recta.
  • Overline: Subraya la parte superior del texto seleccionado con una línea recta.
  • Line-through: Tacha el texto del contenedor seleccionado.
  • None: Decoración por defecto.

Soy un párrafo subrayado.

Soy un párrafo con overline.

Soy un párrafo con line-through.

Hay varias decoraciones más para los textos pero en los ejemplos vistos son lo más utilizados.

Decoración para Vínculos

Cuando utilizamos vínculos el navegador les establece unos estilos ya definidos para que el usuario sepa identificarlos, por ejemplo, este tipo de texto aparece decorado con un text-decoration en underline y un color blue. Si deseamos dejar este texto como si fuera 'normal, debemos aplicarle los siguientes estilos:

a {
text-decoration: none;
color: black;
}

a:visited {
color: black;
}

Primero, debemos seleccionar el elemento que se encuentra el texto con el vínculo (usar el selector más adecuado) y luego aplicar los estilos que tenemos en el ejemplo anterior. De esta manera el contenido que se encuentra con los estílos del vínculo quedan igual que el texto de un elemento normal.

Letter Spacing

Para controlar el espacio existente entre las letras se usa la propiedad letter-spacing y su medida debe entregarse en píxeles (px).

.p-letter-spacing {
letter-spacing: 5px;
}

Soy un párrafo con letras separadas en 5px.

Word Spacing

Esta propiedad permite controlar el espacio entre las palabras en un elemento HTML. Se utiliza para ajustar el espacio entre las palabras en una línea de texto, de manera que pueda verse más legible o estética. Podemos utilizar como valor las diferentes unidades de medida que vimos anteriormente.

.p-word-spacing {
word-spacing: 20px;
}

Soy un párrafo con las palabras separadas en 20px.

Line Height

La propiedad line-height en CSS permite controlar la altura de línea de un elemento HTML. Esta propiedad se utiliza para ajustar la altura de la línea de texto, de manera que pueda verse más legible o estética.

.p-line-height {
line-height: 1.9;
}

En este ejemplo, se establece la altura de línea de los párrafos en 1.9 veces el alto de la fuente utilizada. Podemos utilizar como valor las diferentes unidades de medida que vimos anteriormente. Es importante tener en cuenta que la altura de línea puede afectar la legibilidad del texto, por lo que es necesario utilizarla con cuidado. Una altura de línea demasiado baja puede hacer que el texto sea difícil de leer, mientras que una altura de línea demasiado alta puede hacer que el texto se vea demasiado espaciado.

Google Fonts

Google Fonts es una colección gratuita de fuentes tipográficas que se pueden usar en proyectos web y de diseño gráfico. Fue lanzado por Google en 2010 y desde entonces se ha convertido en una de las fuentes más populares de fuentes en línea.


Google Fonts incluye más de 900 fuentes de diferentes estilos y categorías, desde fuentes serif y sans-serif hasta fuentes con símbolos y emojis. Las fuentes están disponibles para su uso gratuito en proyectos personales y comerciales, lo que las hace ideales para sitios web, folletos, tarjetas de visita y otros proyectos de diseño.

Importando Google Fonts

Primero debemos ingresar a la página de: google fonts. Una vez que ingresamos, podemos seleccionar las fuentes que deseamos aplicar en nuestra página web. Para este ejemplo vamos a escoger la fuente roboto.

Al seleccionarla, luego, vamos a estar seleccionando cuáles son los pesos (weight) de fuente de la fuente “Roboto” que nos interesa tener en nuestro proyecto. Esto lo hacemos clickeando en el botón de más ('+') que aparece en cada opción. El valor del peso que debemos seleccionar para las fuentes es de 400 y 700 (regular 400 y bold 700).

Ahora debemos visualizar en el borde superior derecho de la página un ícono donde estará toda la información que hemos seleccionado. Debemos clickear en ese ícono para llevarnos la información a nuestro código.

Al desplegarse la barra, tendremos dos opciones de importación.

Link

La primera es la opción de etiquetas <link>, la cual debemos copiar su contenido dentro del elemento header en nuestra hoja HTML.

@import

La segunda es @import, la cual podremos copiar y pegar en nuestro archivo Css para importar la fuente. Debemos colocarla arriba de todo en nuestra hoja de estilos.

Importante: Debemos quitar las etiquetas style que nos da esta opción, ya que están puestas por si queremos pegarlo dentro de nuestro html, lo cual no recomendamos.

Aplicar Fuente en CSS

Finalmente debemos declarar la fuente en el elemento que deseamos utilizarla en nuestro caso va a ser en un elemento párrafo donde su texto tendrá el estilo que seleccionamos.

.parrafo-roboto {
font-family: 'Roboto', sans-serif;
}
