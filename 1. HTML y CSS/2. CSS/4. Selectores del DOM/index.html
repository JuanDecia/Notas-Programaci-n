Selectores del DOM

Introducción

Cuando trabajamos con CSS es necesario identificar el elemento del DOM al que se le aplicarán los estilos necesarios para nuestra página. Para ejecutar esta práctica conoceremos a los selectores que son un conjunto de expresiones que sirven para referirnos a los elementos existentes en el DOM. Existen muchos tipos de selectores, son una poderosa característica que ofrece CSS para darles estilos a las páginas web.

Selector Universal

El primer selector que debemos conocer es el selector universal asterisco (*). Cuando se define una regla usando solamente el selector universal se aplicará a todos los elementos de un documento HTML.

* {
margin: 0;
box-sizing: border-box;
}

En este ejemplo hemos aplicado a todos los elementos del documento html un margen de 0px y que los bordes de los elementos de las cajas sean parte del tamaño del elemento.
