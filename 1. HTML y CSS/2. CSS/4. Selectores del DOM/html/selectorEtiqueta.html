Introducción

Los selectores de etiqueta son aquellos que buscan directamente un elemento en el DOM según el nombre de la etiqueta que deseamos seleccionar. Este selector tiene un comportamiento general, ya que, al aplicar a una etiqueta específica los estilos que le coloquemos a este selector se aplicarán a todas las demás etiquetas del mismo tipo, es decir, si seleccionamos una etiqueta <p> para cambiar el color a rojo, todas las demás etiquetas <p> también cambiarán su comportamiento. Por eso, hay que prestar mucha atención al usar un selector tan general como este.

Ejemplo práctico

Para ejemplificar el concepto anterior, crearemos tres headings de nivel h3 a los cuales les daremos unos valores específicos.

h3 {
margin: 30px 0;
color: rgb(103, 45, 240);
font-size: 1.3em;
}
