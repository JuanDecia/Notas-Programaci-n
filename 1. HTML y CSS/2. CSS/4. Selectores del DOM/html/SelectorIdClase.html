Selectores de Id y Clase

Volver a inicio

Introducción

Este tipo de selectores ayudan a filtrar los elementos y son más específicos con los elementos que queremos estilizar por medio de CSS.

El selector de "ID" se representa con el signo numeral (#) en la hoja de estilos seguido del nombre que le hemos dado al "ID", en nuestro caso es "#parrafo".

Es muy importante saber que no debemos aplicar un mismo id a diferentes elementos, porque, se considera una mala práctica y además, puede generar un conflicto en los estilos de nuestro sitio.

Selector ID

Una forma fácil de agregar estilos solo a un elemento es utilizando una identificación única en un elemento o etiqueta de HTML y se logra a través de la utilización del atributo “id” que nos ofrece el propio HTML, en otras palabras, un ID en HTML es un atributo que se utiliza para identificar de manera única un elemento en una página HTML.


Es muy importante destacar que no puede haber más de un elemento con el mismo ID en una página HTML, por lo tanto, no podemos repetirlos , solo debemos usarlo en un único elemento.

Ejemplo práctico ID

A continuación, realizaremos un ejempo donde crearemos tres elementos 'span' donde los tres tendrán un ID diferente al cual le aplicaremos un estilo específico a cada uno de ellos.


Antes de aplicar los estilos, debemos saber cómo identificar este selector en CSS. A los elementos con atributos ID los identificaremos con las almohadillas '#' seguido del nombre del contenedor que tiene el ID. Es decir, si tenemos un elemento <span id='span-rojo'> lo identificaremos como '#span-rojo'.

#span-rojo {
color: red;
}

#span-blanco {
color: white;
}

#span-azul {
color: blue;
}

Texto a color Rojo Texto a color Blanco Texto a color Azul

Como podemos observar en el resultado, cada uno de los elementos span tienen las propiedades que le hemos aplicado a los elementos en la hoja de estilos. Ahora, volvemos a destacar que no podemos volver a repetir este atributo ID en todo el documento, es decir, no puede volver a tener otro elemento 'span' el mismo id.


Por eso, esta práctica no es muy recomendable en un código extenso, porque deberíamos aplicar nombres diferentes a cada elemento que vamos a escoger con un ID. Para evitar y no generar esta mala práctica vamos a ver el selector de clases a continuación.

Selector Class

Introducción

Cuando queremos darles estilos a múltiples elementos, pero no podemos llenar nuestra estructura de HTML de id porque podría llegar a ser inmanejable y poco mantenible en el tiempo debemos aplicar los selectores de clases. En la construcción de un proyecto grande los selectores de clases son la mejor opción de identificar a los elementos, a diferencia del id, este selector puede localizar un número de elementos en el DOM y aplicarles los estilos que definamos. Para hacer uso del selector de clase debemos incluir el atributo class a nuestros elementos HTML y darles un nombre para identificarlos.

Ejemplo práctico

Para el ejemplo de los selectores de clases vamos a volver a crear tres elementos span pero en esta ocasión todos compartirán la misma clase y en la hoja de estilos de CSS, solo debemos aplicar estilos a la clase que tienen los elementos span. Es decir, vamos a tener tres elementos con la siguiente estructura <span class='span-green'>.


Antes de pasar al ejemplo, vamos a conocer como identificamos un elemento que tiene una clase en HTML. Para identificarlo debemos usar el símbolo punto, es decir, '.' luego el nombre del elemento que contiene la clase. La estructura quedaría de la siguiente forma: '.span-green'.

.span-green {
color: green;
}

Multiples clases en un solo elemento

Como vimos en el apartado anterior, es posible colocar más de una clase a un elemento HTML, y para eso, vamos a estar utilizando los espacios, siendo cada espacio la separación entre una clase y la otra.

<p class='fondo-negro parrafo-grande color-blanco'>
Hola soy un párrafo con fondo negro, letra blanca y más grande.
<p>

Ahora aplicaremos el estilo a ese párrafo:

.fondo-negro {
background-color: black;
}

.parrafo-grande {
font-size: 1.2em;
}

.color-blanco {
color: white;
}

Resultado:

Hola, soy un párrafo con fondo negro, letra blanca y más grande.

Pautas a la hora de usar Id y Clases

  1. Utilizar nombres descriptivos: El nombre de una clase debe ser claro y descriptivo, evitando nombres genéricos como “clase1” o “clase2”. El objetivo de esto es poder indicar claramente el propósito de una clase y a qué elemento hace referencia de nuestra página, facilitando así la comprensión del código.
  2. Mantenibilidad: Las clases deben ser lo suficientemente genéricas como para ser reutilizadas, pero también deben ser lo suficientemente específicas como para no causar problemas en la aplicación de estilos. Por ejemplo, la clase navbar-link es lo suficientemente específica para saber qué se trata de un link del navbar y poder utilizarla para todos los links del navbar, pero no entra en detalle de cómo son esos links.
  3. Utilizar minúsculas y evitar usar tildes: Se recomienda usar minúsculas para definir clases ya que es una convención estándar en la industria hacerlo, ya que ayuda a mantener la consistencia y la legibilidad del código. Además, ayuda a la mejor comprensión del código. Es recomendable utilizar mayúsculas únicamente si decidimos nombrar nuestras clases que tengan más de una palabra con el método camelcase (navbarLink , navbarItem , etc)
  4. No utilizar espacios para definir una clase individual: Los espacios a la hora de nombrar clases son caracteres inválidos. Al colocar uno, el código interpretará que estamos poniendo dos clases diferentes. Por lo tanto, si tenemos que usar dos palabras para definir una única clase, utilizaremos camelcase (navbarLink) o guiones medios o bajos (navbar-link / navbar_link).
