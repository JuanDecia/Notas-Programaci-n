Introducción

Cuando trabajamos con la position relative, sí debemos especificar las propiedades de top, right, bottom y left. Es decir, dependiendo de los valores que agreguemos es como se moverá el elemento y se moverá de manera relativa al flujo normal según al padre que lo contiene. Si el elemento que contiene la propiedad relative no se le agrega ningún valor, éste, continuará con su flujo normal.


Si en el caso que tenemos un div centrado y le agregamos una position relative con un top de 100px y un left de 50px, este se moverá hacia abajo 100px y 50px hacia la derecha. Prestemos atención que los elementos son empujados desde la posición que se está indicando (elemento padre).

En este ejemplo, el contenedor padre es el que tiene bordes negros y hemos aplicado en el elemento hijo (bordes azules) una posición relativa de top: 50px desde su ubicación. Es decir, el elemento con bordes azul esta a 50px de distancia desde la parte superior del elemento con bordes negro.


También, haciendo esta práctica hemos roto el flujo normal de los elementos. El elemento con bordes azules debería estar dentro del elemento con bordes negros, esto va a generar un overflow con el siguiente contenedor.
