Introducción

Elemento Sticky

El valor de Sticky trabaja parecido a fixed, la diferencia entre ellos es que el movimiento de sticky se realiza cuando el scroll llega a un límite donde el elemento podría no visualizarse.

Este valor utiliza las propiedades de top, right, bottom y left, los valores indicados son los que luego tomaran esta posición de tipo fixed en la pantalla. Generalmente es usado en los headers o elementos en el que deseemos que siempre esten en el viewport y para ofrecerle una comodidad al usuario. Por eso, al tener el header con el menú de opciones en sticky, este podrá siempre estar a disposición en el viewport del usuario.

.elemento-sticky {
position: sticky;
top: 0;

Este elemento funciona de manera relativa hasta que sale del scroll. Luego, tomará los valores de la posición que hemos declarado y se posicionará en esa parte del viewport (top: 0). No es una propiedad que funcione en todos los navegadores, debemos utilizar la propiedad de webkit y moz para que esta propiedad tenga funcionamiento en respectivos navegadores.
