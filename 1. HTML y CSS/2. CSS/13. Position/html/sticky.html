Introducción

El valor de Sticky trabaja parecido a fixed, la diferencia entre ellos es que el movimiento de sticky se realiza cuando el scroll llega a un limite donde el elemento podría no visualizarse.

Este valor utiliza las propiedades de top, right, bottom y left, los valores indicados son los que luego tomaran esta posición de fixed en la pantalla. Generalmente es usado en los headers o elementos en el que deseemos que siempre esté y para ofrecerle una comodidad al usuario. Por eso, al tener el header con el menú de opciones en sticky, este podrá siempre estar a disposición en el viewport del usuario.

Es decir, este elemento funciona de manera relativa hasta que sale de pantalla. Luego, tomará los valores y se posicionará según le declaremos a las propiedades de posición (top. . .).
