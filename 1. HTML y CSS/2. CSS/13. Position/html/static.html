Introducción

Esta posición es la que se encuentra por defecto en los elementos que colocamos en el DOM si no especificamos ningún otro valor de posición. Debemos saber que, si el elemento se encuentra en esta posición, no podremos usar los valores: top – right - bottom y left.

Hay que aclarar que no es necesario colocar este valor en la propiedad de “position”, ya que como anteriormente habíamos aclarado, es la que se encuentra por defecto en los estilos del navegador.

Para ver el funcionamiento de esta propiedad se recomienda abrir el inspector (F12) y cambiar los valores de top, right, bottom y left. Notaremos que el contenedor no sufrira cambios al respecto.
