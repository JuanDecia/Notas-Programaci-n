Propiedad Overflow

Introducción

En ocasiones los contenidos que agreguemos a los elementos que actúan como padres pueden ser más grandes que el mismo. Este caso es llamado overflow y en CSS tenemos una propiedad disponible llamada de la misma manera para poder manipular ese sobre exceso de flujo.

El valor standard de esta propiedad es visible la cual nos muestra tal cual se encuentra el elemento al utilizarlo. Generalmente se aplica el valor de hidden que oculta todo lo que sobresalga como resultado del overflow.

En el ejemplo anterior tenemos un caso de overflow, el cual sucede porque tenemos el alto del contenedor con una medida específica y el contenido del mismo ocupa más de lo que la dimensión del contenedor puede soportar. En este caso deberíamos aprender a utilizar la propiedad de overflow para poder manipular mejor los elementos de nuestro sitio.

Para manipular este overflow debemos utilizar dicha propiedad y como ejemplo utilizaremos el valor de hidden.

Como vemos en este ejemplo, pudimos aplicar el valor de hidden y solo se muestra el texto que el contenedor puede abarcar.

En el caso de que necesitemos que el contenido se pueda visualizar completo pero el tamaño del contenedor debe respetarse, debemos usar el valor de scroll. Este efecto también puede aplicarse con el valor de auto el cual automáticamente crea las barras de scroll pero no estarán activas si esto no fuera necesario.
