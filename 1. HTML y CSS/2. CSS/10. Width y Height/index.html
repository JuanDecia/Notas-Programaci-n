Width & Height

Width

La propiedad width (ancho) del box model es ancho del área de contenido de un elemento y también, cuando la aplicamos puede modificar el ancho del elemento seleccionado. Cuando estamos trabajando con elementos, es recomendable aplicar la propiedad “border” para trazar el tamaño del elemento y así visualizar los cambios que estamos realizando con respecto al box model con el objetivo de llegar al estilo deseado.


Esta propiedad es aplicable a todos los elementos que se encuentren en el DOM de HTML. Los valores de medida aplicables a esta propiedad pueden ser en px, vw y porcentaje. Se recomiendan usar medidas relativas, ya que los elementos son adaptativos a las diferentes pantallas de los distintos dispositivos que se puede visualizar nuestro contenido.


Existen propiedades llamadas min-width y max-width. Estas sobrescriben el valor de width y son útiles cuando necesitemos que el elemento siempre respete el contenedor padre. Es decir, si tenemos un valor de un elemento con width 100% y tenemos un elemento con la propiedad de max-width 70%, debemos saber que tiene prioridad jerárquica el elemento con max-width por ende el elemento se renderizará con un ancho del 70% y no del 100%.

.elemento-html {
width: 200px;
width: 40%;
width: 4rem;
width: 5em;
width: 30vw;
}

Diferencia entre width: auto y 100%

Se emplea “width:100%” para que un elemento adopte todo el ancho del elemento padre. Si se agrega espacio adicional al tamaño del elemento (width:100%) sin tener en cuenta el tamaño del elemento padre puede generar un overflow, en otras palabras, si tenemos alguna modificación sobre margin, padding o border en el elemento hijo además del tamaño del ancho, esto puede generar que el contenido se salga del contenedor padre.


Con la propiedad y valor “width: auto” el contenido se puede expandir o contraer para ocupar el espacio horizontal disponible dentro del elemento padre, si el elemento hijo tiene propiedades declaradas como border o rellenos en espacios no se suman al ancho total del elemento por lo que no genera un overflow.

Min-Width

Esta propiedad establece un ancho mínimo para el elemento seleccionado. En el caso que declaremos un 'width-min: 30%' al contraer el viewport este elemento se contraerá solo hasta el 30%.

Max-Width

Esta propiedad establece un ancho máximo al elemento seleccionado. En el caso que declaremos 'max-width: 50%' al extender el viewport este no seguirá aumentando el tamaño del elemento.

Height

La propiedad height (alto) especifica la altura del área de contenido de un elemento. Esta propiedad tiene las propiedades y valores aplicables con las reglas muy similares a “width”. Debemos destacar que su única diferencia al aplicar valores porcentuales al “viewport” en vez de usar “vw” que es para el ancho, debemos usar la propiedad “vh” que es para el alto. Las iniciales que le corresponden a esta propiedad son V y H que están haciendo referencia a “Viewport” y “height”.

.elemento-html {
height: 200px;
height: 40%;
height: 4rem;
height: 5em;
height: 30vh;
}
