Estilos para elementos del DOM

Introducción

En el tema anterior comenzamos a ver cómo aplicar algunos estilos a nuestros elementos, uno de ellos fue el tema de aplicar colores. En esta sección de CSS vamos a ver un poco más en profundidad cuáles son los estilos o propiedades que podemos utilizar en los elementos del DOM.


Propiedades abreviadas

En CSS existen propiedades nos permiten asignar el valor de muchas otras propiedades al mismo tiempo. Usando una sintaxis llamada “Shorthand” nos permite escribir código más conciso, esta práctica es favorable para mantener un trabajo mantenible y mejor legible.


Las shorthands agrupan las propiedades para un mismo tema. Por ejemplo, la propiedad “background” puede definir el valor de “color”, image”, “repeat” y position, gracias a esta sintaxis podemos agrupar todas estas propiedades sólo declarando background.
